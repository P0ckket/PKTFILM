# 世界电影摄影风格速查（World Cinema Reference）

## 用途

速查"某个时代/某种视觉风格"对应哪些代表作，为创作寻找摄影参照（cinematography reference）。

> 本索引是**反查工具**，不是分析文章。详细分析见 [电影分析方法论](../theory/film-analysis-method.md)。

## 快速反查索引

**按风格找参照**：
- 黑色电影（Film Noir）→ 马耳他之鹰、双重赔偿、公民凯恩
- 手持纪录片风 → 精疲力竭、阿尔及尔之战、仇恨、人类之子
- 极致对称 → 大都会、库布里克系列、布达佩斯大饭店
- 诗意长镜头 → 塔可夫斯基三部、花样年华、细细的红线
- 霓虹低调光 → 银翼杀手、驾驶、遁入虚空、月光男孩
- 超宽史诗 → 阿拉伯的劳伦斯、星球大战、血色将至、狂暴之路
- 自然主义/新现实主义 → 偷自行车的人、仇恨、罗马、杀人回忆
