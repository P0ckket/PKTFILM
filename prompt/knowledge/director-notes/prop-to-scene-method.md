# 道具生场景法(命题作文应对)

## The question it answers
**考试给了一个词(雨伞/钥匙/火柴/戒指),怎么在 20 分钟内从这个词构思出一个 2000 字的完整故事?**

## Decision mapping

### Step 1:道具八法快扫(3 分钟)
拿到道具后,快速对着八种功能过一遍:
| 用法 | 这个道具能怎么用? |
|---|---|
| 价值 | 这个道具对谁是"不可或缺"的? |
| 象征 | 这个道具能代表什么抽象情感? |
| 悬念 | 这个道具的来源/去向不明能引发什么问题? |
| 反差 | 这个道具出现在"不该出现"的人/地点,会怎样? |
| 转折 | 这个道具揭露了什么真相? |
| 激励 | 这个道具触发了谁的什么动机? |
| 危机 | 这个道具制造了什么紧急状况? |
| 高潮 | 这个道具在最后时刻发挥了什么作用? |

**挑 2-3 条最好的**,放进后面的五事件大纲。

### Step 2:道具 → 人物(2 分钟)
问自己:**谁最需要/最害怕/最不适合拥有这个道具?**
- "最需要" → 道具是**价值**
- "最害怕" → 道具是**危机**
- "最不适合" → 道具是**反差**

选"最不适合"的人物 = 自动获得 [反差发动机法](./contrast-engine-method.md) 里的"离水之鱼"效果。

### Step 3:人物 → 三维度(2 分钟)
用 [人物设计素材库](../story-elements/character-design-library.md) 快速设定:
- 可怜 = ____
- 可信(缺陷) = ____
- 可爱 = ____

### Step 4:五事件大纲(8 分钟)
用 [五事件大纲法](./five-event-outline.md) 的模板:
| 事件 | 道具功能 | 一句话 |
|---|---|---|
| 开场(400) | 价值 or 反差 | 道具出场,主角出场 |
| 激励(400) | 激励 | 因为道具,主角有了目标 |
| 转折(400) | 转折 or 悬念 | 道具揭露了新信息 |
| 危机(400) | 危机 | 道具制造了紧急状况 |
| 高潮(400) | 象征 | 道具回扣主题 |

### Step 5:写!(5 分钟检查 + 130 分钟写作)
开始写之前,最后检查:
- [ ] 道具在五个事件中**至少出现 3 次**?
- [ ] 道具在开场和结尾的**功能不同**?(变化 = 主题)
- [ ] 主角与道具的关系有**变化**?(一开始忽视/害怕 → 最后珍惜/使用)

## 示例:以"火柴"为例

### Step 1 快扫
- 价值:荒野中的最后一根火柴 = 生存
- 象征:希望、温暖、脆弱
- 悬念:谁点燃了那根火柴?
- 反差:消防员口袋里的火柴
- 转折:火柴盒里藏着一张纸条
- 激励:父亲留给女儿的火柴盒
- 危机:火柴就剩一根,风越来越大
- 高潮:女儿划燃最后一根火柴,照亮了父亲的遗照

### Step 2 选人物
"最不适合"拥有火柴的人 → 消防员(反差!)
或者"最需要"火柴的人 → 流浪者(价值)

选:**消防员的女儿**(结合反差+情感)

### Step 3 三维度
- 可怜:父亲(消防员)因公殉职,她成了孤儿
- 可信:脾气暴躁,不接受心理辅导
- 可爱:偷偷给流浪猫留食物

### Step 4 五事件
| 事件 | 道具功能 | 一句话 |
|---|---|---|
| 开场 | 反差 | 少女小林拒绝参加父亲的追悼会,把火柴盒摔在地上 |
| 激励 | 激励 | 消防队的叔叔告诉她火柴盒里有父亲的遗言,她被迫打开 |
| 转折 | 悬念 | 火柴盒里没有遗言,只有一根火柴——她开始追问原因 |
| 危机 | 危机 | 她去火灾现场寻找答案,差点被废墟困住 |
| 高潮 | 象征 | 她在废墟里划燃那根火柴,在火光中看到父亲刻在墙上的字:"活下去" |

## Exercises

### 练习一:随机词造故事
闭眼翻开词典,指到哪个词就用哪个词,20 分钟内完成五事件大纲。

### 练习二:同一道具 × 三种人物
选一个道具,分别搭配三个完全不同的人物,写三个不同的故事核:
- eg. 钥匙 × 小偷 / 钥匙 × 孤寡老人 / 钥匙 × 宇航员

### 练习三:道具功能翻转
一个道具在开场是"危机",结尾变成"象征"。写出这个翻转的一句话。

## Anti-patterns
| 反模式 | 为什么不行 | 怎么修 |
|---|---|---|
| 道具只出现一次 | 跟命题词的关系太弱 | 至少 3 次,且每次功能不同 |
| 道具只是背景 | 没有参与情节推进 | 让道具成为情节的触发器 |
| 硬凑道具含义 | 象征太牵强 | 先写故事,再回头给道具赋义 |
| 道具太多 | 记忆点分散 | 一个主道具贯穿五事件 |
