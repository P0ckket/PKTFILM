# 反差发动机法(故事核设计)

## The question it answers
**一句话怎么概括你的故事?概括之后,怎么保证这个故事自带吸引力?**

## Decision mapping

### 第一步:写一句话故事(p.40)
**一句话故事 = 做什么 + 结局**
- eg. 楚门想要逃离虚假的世界,回到现实世界中生活——最终成功。

### 第二步:提取故事核(p.40)
**故事核 = 谁 + 做什么**(人物 + 行动)
- **行动 ≠ 动作**,也非遭遇——而是**主动的、有目标的、克服不同方面阻力的连续动作**
- eg. 一个被监视了三十多年的男人决心逃回现实世界。

### 第三步:注入反差(p.40)
**让人物和行动之间必须构成反差——从而引起悬念,激发阅读兴趣**

| 故事核 | 反差点 |
|---|---|
| 一个商人爱上了他雇来假扮女友的妓女(《漂亮女人》) | 商人 × 妓女 |
| 一对新婚夫妇要在圣诞节这天跑遍双方父母的四个再婚家庭(《四个圣诞节》) | 新婚(简单) × 四个家庭(复杂) |
| 一个新入职员工去参加公司周末集体度假,结果发现有个人想杀他(《度假》) | 公司度假(安全) × 被杀(危险) |

### 第四步:使用"离水之鱼"公式(p.40)
**离水之鱼 = 逆人物能力去设置目标和环境**
- 特定类型人物被推进**超出他理解范围**的世界
- **主角越"不适合",故事越好看**

| 离水之鱼 | 例子 |
|---|---|
| 普通人进入危险世界 | 《虎胆龙威》 |
| 理性人进入疯狂世界 | 《飞越疯人院》 |
| 成年人进入儿童世界 | 《放牛班的春天》 |
| 都市人进入原始世界 | 《荒岛求生》 |

### 第五步:检验三层张力(p.40)
见 [[../concepts/narrative/contrast-three-layers]] 的详细分析:

| 张力层 | 反差轴 | 检验问题 |
|---|---|---|
| **人物张力** | 过去与当下 / 自我与非我 | 主角过去是什么人?现在被迫成为什么人? |
| **情节张力** | 事件与人物 / 命运与抉择 | 事件要求主角做的事,与主角的性格是否矛盾? |
| **场面张力** | 整体与局部 / 和谐与突兀 | 画面里有没有一个"不协调"的元素? |

## Exercises

### 练习一:故事核造句(5 分钟)
从以下人物-行动配对中选一个,扩展成一句话故事核:
| 人物 | 行动(反直觉) |
|---|---|
| 盲人 | 开出租车 |
| 和尚 | 组建乐队 |
| 死刑犯 | 申请当志愿者 |
| 消防员 | 放火 |
| 奶奶 | 参加格斗比赛 |

### 练习二:反差强度测试
写出你的故事核,问自己:
- 人物和行动之间的反差有多大?(1-10 打分)
- 如果反差 < 5,能不能换一个人物 or 换一个行动?

### 练习三:一鱼多吃
同一个"鱼"(人物),放进三种不同的"水":
- eg. **一个退伍军人** + 幼儿园(喜剧)/ 精神病院(惊悚)/ 烹饪学校(温情)

## Anti-patterns
| 反模式 | 为什么不行 | 怎么修 |
|---|---|---|
| 人物和行动之间没反差 | 故事核无聊——"一个警察抓坏人" | 逆转:一个警察帮坏人逃跑 |
| 反差太极端以至于不可信 | 观众不买账——"一个婴儿拯救地球" | 找到合理的"桥梁":婴儿身上有某种能力 |
| 只有人物张力,没有情节/场面张力 | 设定很好看,展开很无聊 | 给事件本身也制造反差 |
| 只写"遭遇"不写"行动" | 主角被动,观众同情但不紧张 | 让主角做"连续的、有目标的、克服阻力的动作" |

## Reflection prompts
- 如果你用一句话把故事核讲给妈妈听,妈妈会不会问"然后呢?"——如果会,说明反差够了。
- 你的主角是不是"离水之鱼"?如果他在这个环境里如鱼得水,你需要换一个环境。
- 三层张力里,哪一层最弱?花 5 分钟强化最弱的那一层。
