# 五事件大纲法(艺考编讲故事)

## The question it answers
**一个 2000 字的编讲故事,应该怎么分段、每段写什么、字数怎么分配?**

## Decision mapping

### 总体框架(p.41)
| 维度 | 标准 |
|---|---|
| 考试时长 | 2.5 小时(150 分钟) |
| 时间分配 | 20 分钟构思 + 130 分钟写 |
| 统考合适字数 | **2000 字** |
| 高中生写作最大问题 | **主角虚假没有特点 + 情节虚假不会解决冲突** |

### 故事三要素(p.41)
1. **人物**
2. **事件**
3. **主题**

### 三类人物(p.41)
| 类型 | 功能 | 数量要求 |
|---|---|---|
| **主角** | 主动力 + 行动力 | 1 人(戏份最多) |
| **配角** | 与主角互补 + 产生关系变化 | 1 人 |
| **工具人** | 不需要塑造形象,只用于推动事件 | 0-2 人 |

**人物硬性要求**(p.41):
- **一个主角 + 一个配角 + 二人关系**
- **现代中国人**
- 主角身份:**可怜、可信、可爱**
- 所有人物动作:**拍出来**(必须是可视化的)
- 欲望:**原始欲望**(名利权情) + **高级欲望**(奉献、牺牲、正义)

### 五事件大纲(p.41 + p.43)

#### 事件一:开场事件(400 字)
- [ ] **冲突开场**——不要从平静开始
- [ ] **一个可怜可爱大于缺陷的主角出场**
- [ ] **明确主角原始欲望**
- **可信** = 缺陷或坏习惯
- **可爱** = 值得被观众喜欢(顾家、技能)
- **原始欲望的示例**:赚钱养家糊口

#### 事件二:激励事件(400 字)
- [ ] **配角出场**
- [ ] **主角与配角产生冲突**
- [ ] **明确主角的具体目标**

#### 事件三:转折事件(400 字)
- [ ] **主角遇到阻碍目标的小麻烦**
- [ ] **主角意识到自己的缺陷,展示可爱**
- [ ] **主角开始改善与配角的关系**

#### 事件四:危机事件(400 字)
- [ ] **主角遇到阻碍目标的大麻烦**
- [ ] **主角逐步改善自己的缺陷**
- [ ] **主角逐步改善与配角的关系**

#### 事件五:高潮事件(400 字)
- [ ] **主角没有完美完成目标**(注意:不是完美胜利!)
- [ ] **主角弥补了自身缺陷**
- [ ] **主角和配角的关系达到了顶点**

### 五事件 × 弧光的对应关系
| 事件 | 缺陷 | 关系 | 目标 |
|---|---|---|---|
| 开场 | 缺陷暴露 | 主角独立 | 原始欲望 |
| 激励 | 缺陷与他人冲突 | 对立 | 具体目标确立 |
| 转折 | 意识到缺陷 | 开始改善 | 小障碍 |
| 危机 | 开始改善缺陷 | 深度互助 | 大障碍 |
| 高潮 | 缺陷弥补 | 关系顶点 | 目标未完美达成 |

## Exercises

### 练习一:10 分钟大纲
给定题目后,用 10 分钟写完以下表格:
| 栏 | 内容 |
|---|---|
| 主角身份 | ____（可怜/可信/可爱各一条） |
| 配角身份 | ____ |
| 二人关系 | ____ |
| 原始欲望 | ____ |
| 具体目标 | ____ |
| 缺陷 | ____ |
| 开场一句话 | ____ |
| 激励一句话 | ____ |
| 转折一句话 | ____ |
| 危机一句话 | ____ |
| 高潮一句话 | ____ |

### 练习二:反推已有电影
用五事件模板拆解一部你看过的电影:
- 它的五个事件分别是什么?
- 哪个事件最薄弱?如果你来写,怎么加强?

## Anti-patterns
| 反模式 | 为什么不行 | 怎么修 |
|---|---|---|
| 开场从平静写起 | 浪费 400 字,老师已经无聊了 | 冲突开场——让主角一出场就在"麻烦"里 |
| 高潮完美胜利 | 观众没有回味,主题变浅 | 目标"部分达成"——赢了比赛但失去了朋友(代价) |
| 没有配角 | 没有关系变化 = 没有主题 | 加配角,至少一个 |
| 主角一开始就完美 | 没有弧光 | 给主角一个明确缺陷 |
| 工具人太多 | 分散注意力 | 砍到 1 个主角 + 1 个配角 + 最多 1 个工具人 |
| 台词 > 动作 | 不可视化,没有"画面感" | 动作 > 环境 > 台词(p.42) |

## Reflection prompts
- 这五个事件,每一个都有"画面"吗?如果摄影师在场,每个事件能拍到什么?
- 主角在第五个事件里,跟第一个事件比,哪里变了?这个变化 = 主题吗?
- 你的配角在第三、四事件里有存在感吗?如果配角被删掉故事还成立,说明配角是工具人。

## Seed list
可以用本大纲套路处理的题目类型:
- **命题作文**(给一个词:雨伞/钥匙/手表)→ 把词变成道具,从道具生五事件
- **续讲/改编故事**(给一个开头)→ 把给定开头当作"开场事件",补后四个
- **看图写故事**(给一张图)→ 把图的场景当作"开场场景",从场景生五事件
- **故事接龙**(北电三试)→ 找到前面同学的主角,反转人物关系(p.42)
