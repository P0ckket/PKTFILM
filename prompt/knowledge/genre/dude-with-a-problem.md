# 麻烦家伙型(Dude with a Problem)

## TL;DR
**麻烦家伙型(Dude with a Problem)** 是 Snyder 10 类型里的"无辜卷入"型——**一个普通甚至不如普通人的主角,被莫名卷入一场生死考验**。三元素是:**无辜主人公 + 祸从天降 + 生死考验**。它的冲突底层是"**力量失衡**"(普通人 vs 大事),对应的两难是"**逃避 vs 面对现实**"。核心公式:**主角越普通 + 麻烦越大 + 坏蛋越坏 = 故事越好看**。惊悚片和灾难片的大部分都属于这一类。

## Definitions

- **无辜主人公**:普通、平凡,甚至不如普通人倒霉的主角。
- **祸从天降**:无辜主人公陷入无尽的伤害之中,被莫名卷进麻烦当中。
- **生死考验**:生死边缘的极端情境,过关之后个体/家庭/群体/社会得以继续生存。

## Key terms

### 三元素(【原文】p.21)
| 元素 | 含义 |
|---|---|
| 无辜的主人公 | 普通、平凡、甚至不如普通人、倒霉 |
| 祸从天降 | 无辜卷入麻烦 |
| 生死考验 | 极端处境,过关后有人继续生存 |

### 核心公式(【原文】p.21)
**主角越普通 + 麻烦越大 + 坏蛋越坏 = 故事越好看**

### 两难(【原文】p.28)
**逃避 vs 克服内心恐惧、面对现实**

### 衍生类型(【原文】p.21)
| 衍生 | 特征 | 例子 |
|---|---|---|
| 执法麻烦 | 主人公必须独自一人应付坏人和警察 | 虎胆龙威、神探大战 |
| 家庭麻烦 | 家并没有想象的那么安全,表面的假象与真实情况截然不同 | 房间、陌生来电 |
| 自然麻烦 / 灾难麻烦 | 渺小的人类面对疾病、灾害、野兽时的孤单无助 | 唐山大地震、世界末日、釜山行、泰坦尼克号 |

## Context & problem
**这个类型解决的问题是**:怎么让观众快速共情主角?

答:**让主角不比观众更强**。观众在银幕前是普通人,如果主角一开始就是英雄,观众需要跨越一个"认同障碍"。但如果主角是"跟我一样的普通人",观众立刻进入——"换我也会这样"。然后让事情变糟、变得更糟——观众会跟着主角一起崩溃,一起挣扎,一起寻找出路。

这是所有惊悚片和灾难片最可靠的"观众入口"。

## Claims

1. **主角必须"无辜"**(【原文】p.21)
   - 【原文】"一个无辜主人公——普通、平凡甚至不如普通人,倒霉"
   - 关键是"**不是自作自受**":如果主角是因为自己作恶才遭报应,观众不会同情。

2. **麻烦必须"从天降"**(【原文】p.21)
   - 【原文】"无辜的主人公陷入无尽的伤害之中——被莫名卷进麻烦当中"
   - 主角**没有主动找麻烦**,但麻烦主动找了他。

3. **生死考验的功能是"筛选什么值得保留"**(【原文】p.21)
   - 【原文】"生死考验后——个体、家庭、群体或是社会得以继续生存"
   - 【我的理解】过关不是为了胜利本身,而是为了让某样东西(自己/家人/社会)**继续存在**。

4. **越普通 + 越大 + 越坏 = 越好看**(【原文】p.21)
   - 这是本类型的唯一公式——**三项成正比关系**。
   - 【你的推断】这条公式解释了为什么惊悚片爱用"郊区中产家庭"作为主角——他们是"最普通"的极致。

5. **两难的核心是"逃避 vs 面对"**(【原文】p.28)
   - 主角的弧光通常是"从想逃避到最终面对"——这是本类型的情感引擎。

## Method / How to use

### 设计 4 步
1. **定主角的"普通值"**:越低越好——职业越卑微、能力越弱、处境越倒霉
2. **设计"祸从天降"事件**:必须是主角**无法预见**的外部强制
3. **升级麻烦**:一层接一层,让主角"逃无可逃"
4. **设定生死考验**:最极端的处境,过关才有人/事/物继续存在

### 艺考编讲故事的"三升级"模板
- **初级麻烦**:主角发现事情不对劲(发现尸体 / 收到威胁)
- **中级麻烦**:主角的亲人也卷入(家人被绑架 / 朋友失联)
- **终极麻烦**:主角自己的生命受到直接威胁(被追杀 / 被困绝境)

### 麻烦家伙 checklist
- [ ] 主角够普通吗?(能力/身份/处境)
- [ ] 麻烦是"从天降"的吗?(主角不是自找的)
- [ ] 麻烦有没有持续升级?(不是一个麻烦撑全片)
- [ ] 有没有明确的"生死考验"时刻?
- [ ] 主角最终是选择"面对"而不是"逃避"?
- [ ] 过关后,某样东西(生命/家人/尊严)得以继续存在?

## Debates & limits
- **"越普通越好"的极限**。主角不能普通到"没有任何个性"——他必须有**一个让人记住的标签**(洁癖、结巴、胆小)。纯素人会让观众失去抓手。
- **"祸从天降"的因果性困境**。完全的无因性会让故事缺乏主题——现代惊悚片常在结尾揭露"**其实有因**"(如主角年轻时的错误),这破坏了"无辜"但提供了主题。这是麻烦家伙与 Whydunit 类型的交界地带。
- 【我的理解】艺考编讲故事最稳的麻烦家伙是"**主角发现了不该发现的东西**"——这种设定既"普通"(他不是侦探)又"祸从天降"(不是找的),还能带主题(知情的代价)。
- 【你的推断】p.21 的"越普通 + 越大 + 越坏 = 越好看"是教学归纳,实际作品中常有变体——《冰血暴》的主角杰瑞·伦德加德并不普通(他是汽车销售经理),但因为他**欲望大能力低**,依然触发麻烦家伙机制。

## Directing takeaways
1. **开场要建立"普通"**。用 1-2 场戏展示主角的平凡生活——越平凡,对比越强。
2. **祸从天降的那一下必须"冷"**。不要预警,不要铺垫——让观众和主角同时被击中。
3. **升级麻烦时,要**每一次升级都掐住主角的逃路**。不能让他还能"躲一下"。
4. **生死考验戏要慢**。观众需要时间感受主角的崩溃和重建——剪辑太快会让人觉得"轻松过关"。
5. **结尾保留一个"代价"**。麻烦家伙很少"毫发无伤"地获胜——主角通常失去某样重要的东西(亲人、天真、过去)。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 麻烦家伙 = 动作片 | 麻烦家伙是**无辜卷入**,很多动作片的主角是主动冒险(eg.007 不是麻烦家伙)。 |
| 麻烦家伙 = 小人物 | "普通"是起点,但主角的**成长**是核心。只写"小人物受苦"不是麻烦家伙,是苦情戏。 |
| 祸从天降 = 没有理由 | 祸从天降指"主角没找",不是"完全无因"——可以有深层原因,但主角当时并不知情。 |
| 麻烦家伙 = 灾难片 | 灾难麻烦是衍生类型之一,不是全部。 |
