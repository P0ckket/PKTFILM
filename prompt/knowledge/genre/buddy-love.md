# 伙伴之情型(Buddy Love)

## TL;DR
**伙伴之情(Buddy Love)** 是 Snyder 10 类型里的"关系型"故事——**双主人公在互相补足的动态关系中推进剧情**。它的三元素是:**一个"不完整"的主人公 + 一个"伙伴"+ 一种"复杂情况"**。本质是"**人物反差 + 动态关系 + 催化剂**"。它的通用规律是:**一开始互相厌恶 → 共同经历发现离不开 → 更大冲突分道扬镳 → 高潮回到一起解决问题**。爱情故事、宠物之爱、同事之情、禁忌之爱都是它的衍生类型。

## Definitions

- **伙伴之情**:以双主角的互补关系为核心推动力的叙事类型。
- **不完整的主人公**:在生理、精神、道德方面有所欠缺,需要另一人帮助才能变得完整。
- **伙伴**:身上有主人公所欠缺的品质,帮助主人公"完整"得以实现的角色。
- **复杂情况**:让两人被迫相处的外部情境(误解、个人/道德观点差异、历史灾难、社会反对)。
- **催化剂**:让两人发生微妙转变的事件或互动。

## Key terms

### 三元素(【原文】p.20)
| 元素 | 功能 |
|---|---|
| 不完整的主人公 | 生理/精神/道德有欠缺 |
| 伙伴 | 有主角所缺的品质 |
| 复杂情况 | 让两人无法分开的处境 |

### 四段式规律(【原文】p.20)
1. 一开始互相厌恶反感
2. 共同经历引出"谁也离不开谁"的事实
3. 更大冲突带来"一无所有",两人分道扬镳,但心里都离不开对方
4. 高潮时两人走到一起共同解决困难

### 本质(【原文】p.20)
- 人物反差
- 动态关系
- 发生在两个主人公之间的矛盾
- 催化剂:人物有微妙转变或没有变化

## Context & problem
**这个类型解决的问题是**:怎么让"两个人的故事"比"一个人的故事"更好看?

答:**让两人互补却互不相容**。主角 A 身上的缺陷恰恰是主角 B 身上的优点,反之亦然——这样他们就必须靠近,但又天然排斥。这种"既依赖又排斥"的张力就是伙伴之情的引擎。

它也解决了"二元对立如何产生主题"的问题:两人的碰撞就是两种价值观的碰撞,碰撞的结果就是主题。

## Claims

1. **主角 1 有缺陷或边缘化,主角 2 是催化剂**(【原文】p.21)
   - 【原文】"主角 1 有缺陷或边缘化,是正常人眼中的异类"
   - 【原文】"主角 2 催化剂,则要负责修正和弥补伙伴的问题"
   - 【原文】"主角弱点 = 伙伴优点"——两人的缺点/优点必须是**镜像关系**。

2. **CP 必须分离再结合**(【原文】p.21)
   - 【原文】"CP 会因某些特定情况和原因暂时分离,但会为之后更紧密地结合打下伏笔"
   - 这是伙伴之情的"一无所有"节拍——对应十五节拍表的第 11 个节拍。

3. **伙伴关系的冲突源有多种类型**(【原文】p.21)
   - 阶级
   - 种族矛盾 + 性格矛盾
   - 年龄差距
   - 职业身份所带来的任务
   - 主角要自杀
   - 身体疾病或大脑障碍

4. **人物弱点是为了通过变化展示主题**(【原文】p.21)
   - 【原文】"开局有弱点,结尾变优点——让观众看到明显的人物变化"
   - 【原文】"人物变化 + 事件结果 = 故事主题的揭示"
   - 【我的理解】伙伴之情与弧光机制是**同源**的——两人的关系变化本身就是双重弧光。

5. **目标、动机、弱点的配套设置**(【原文】p.21)
   - 目标 = 观众的期待
   - 目标 = 主角对结果的期待(主观设想)
   - 动机 ≠ 目标,动机催生目标
   - 人物弱点是为了"通过变化展示主题"

### 衍生类型(【原文】p.20)
| 衍生类型 | 特征 | 例子 |
|---|---|---|
| 宠物之爱 | 主人公的情感通过与宠物的互动间接展示给观众;前后反差;共同经历麻烦绑定命运 | — |
| 爱情故事 | 复杂情况 = 两个不开窍的男女无法意识到对方是完美伴侣 | 夏洛特烦恼、傲慢与偏见、花束般的恋爱 |
| "爱在"三部曲 | 同上细分 | 爱在黎明破晓前、爱在日落黄昏时、爱在午夜降临前 |
| 禁忌之爱 | 游离在当代伦理框架之外的爱情,不被社会认可 | 断背山、美女与野兽、洛丽塔、罗密欧与朱丽叶、向阳处的她 |
| 灾难之情 | 患难见真情 | 泰坦尼克号、乱世佳人、史密斯夫妇 |
| 同事之情 | 双方都需要对方的保护 | 尖峰时刻、唐人街探案、雨人 |

## Method / How to use

### 伙伴之情设计 4 步
1. **定主角 1 的缺陷**(弱点)
2. **定主角 2 的对应优点**(正好补足主角 1 的缺陷)
3. **定复杂情况**(强制让他们相遇并无法分开的情境)
4. **走四段式规律**(厌恶 → 互补 → 分离 → 重聚)

### 艺考编讲故事快速套路
选一对反差鲜明的人物身份:
- 富商 + 流浪汉
- 律师 + 罪犯
- 医生 + 黑帮
- 严师 + 差生
- 老人 + 孩子

再放进一个"离不开"的情境:
- 旅行(公路片)
- 共同任务(同事)
- 继承关系(照顾)
- 契约(以命换命)

### 伙伴之情 checklist
- [ ] 主角 1 有明确的缺陷吗?
- [ ] 主角 2 的优点是主角 1 的缺陷的镜像吗?
- [ ] 有一个强制让两人在一起的复杂情况吗?
- [ ] 四段式里,"分离"环节写出来了吗?(容易被省)
- [ ] 高潮时两人是否"重聚解决困难"?
- [ ] 主题是否从两人的碰撞里浮出?

## Debates & limits
- **"必须分离再结合"的规律性**。【学界常见解释】这是好莱坞模板,但独立电影里不一定遵守——《爱在午夜降临前》里两人几乎全程在一起,"分离"只是争吵。
- **"缺陷与优点互为镜像"的教条化风险**。过度公式化会让两人变成"工具人"——一人负责缺陷,一人负责救赎。好的伙伴之情,两人都有缺陷和优点,只是互补的侧面不同。
- 【我的理解】艺考编讲故事里,最保险的伙伴之情配置是"**城市人 + 乡村人 + 公路情境**"——这是中国版的 buddy road movie 固定模板。
- 【你的推断】宠物之爱是伙伴之情的退化形式——宠物无法"反驳"主人,所以动态关系简化为"主人单向变化"。这让它更适合情感戏但缺乏辩论深度。

## Directing takeaways
1. **开场必须让两人同框但对立**——同一个画面里两人的反差最直观。
2. **"离不开"的情境要写硬**——不能只是"刚好遇到",要让他们**没有选择必须一起**。
3. **分离段不能省**。这是伙伴之情最容易塌的地方,很多学生写到中段就让两人一直在一起到底——这样缺少"伪胜利 + 一无所有"的起伏。
4. **主题藏在两人的争论里**,不是在独白里。关键戏应该是两人争执的那场戏。
5. **结尾不必"明确的言语确认"**——一个动作(紧握的手、共同走出的背影)就够了。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 伙伴之情 = 爱情片 | 爱情片只是伙伴之情的一个**子类**。伙伴之情还包含宠物之爱、同事之情、灾难之情、禁忌之爱。 |
| 伙伴之情 = 双主角片 | 不是所有双主角片都是伙伴之情——《教父》有两主角(维托 + 迈克尔)但属于被制度化类型。关键看"**是否以互补关系为核心动力**"。 |
| 伙伴之情需要浪漫 | 不需要。雨人、尖峰时刻都是伙伴之情但没有浪漫。 |
| 四段式必须按顺序 | 一般遵守,但可以压缩或跳过某些阶段——只要保留"厌恶 → 互补 → 解决"的总轨迹。 |
