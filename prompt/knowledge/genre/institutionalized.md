# 被制度化型(Institutionalized)

## TL;DR
**被制度化(Institutionalized)** 是 Snyder 10 类型里的"群体型"故事——**主角个人与"集体"做对抗后被同化或吞并**。核心理念是"**为了多数人牺牲少数人**"。三元素是:**一个有规则的集体 + 一种选择 + 一种牺牲**。冲突本质是**力量悬殊**——个体无论如何都难以战胜集体。Snyder 类型学里最悲观的一类——结局常常是主角被集体同化、毁灭群体或自杀。集体的类型很广:军队、家庭、公司、师徒、监狱、民族。

## Definitions

- **被制度化**:主角与一个有规则的集体对抗,最终被同化、吞并或以某种方式牺牲的叙事类型。
- **有规则的集体**:具有自身行为准则、制度、道德、情感模式的群体——排斥性是其共同特征。
- **外来者视角**:一定会有一个"新人"进入集体,成为观众的观看视角。
- **关键人物**:完全赞同这个群体的捍卫者,是外来者的最大劲敌。

## Key terms

### 核心理念(【原文】p.23)
**"为了多数人牺牲少数人"**
**"主人公个人与'集体'做对抗后被同化或吞并"**

### 三元素(【原文】p.23)
| 元素 | 功能 |
|---|---|
| 有规则的集体 | 这个集体下的人有自身的行为准则、制度、道德、情感模式;共同特点:排斥性 |
| 一种选择 | 三种选项:反叛型 / 天真型 / 公司型 |
| 一种牺牲 | 三种结局:加入组织 / 毁灭群体 / 自杀 |

### 集体的载体(【原文】p.23)
- 监狱
- 传统的封建家庭
- 外界无法有效联系的学校
- 传销组织
- 精神病院
- 民族、国家
- **集体的具体表征**:服装、口号、食物、房间、作息时间、课间体操

### 主角的三种选择(【原文】p.23)
| 选择 | 含义 |
|---|---|
| 反叛型 | 坚决揭露反抗制度 |
| 天真型 | 对规则制度无知 |
| 公司型 | 已经顺从体制 |

### 三种牺牲(结局)(【原文】p.23)
| 牺牲 | 含义 |
|---|---|
| 加入组织 | 被同化,成为集体一员 |
| 毁灭群体 | 以个人之力反击成功(较少见) |
| 自杀 | 最后的反抗形式 |

### 两难(【原文】p.28)
**两种群体的思想观念中进行选择**

### 冲突本质(【原文】p.28)
**力量悬殊**(个人 vs 集体)

### 衍生类型(【原文】p.23)
| 衍生 | 例子 |
|---|---|
| 军事机构 | 全金属外壳 |
| 家庭群体 | 教父 |
| 商业群体 | 穿普拉达的女王 |
| 师徒群体 | 入殓师、死亡诗社、放牛班的春天 |

### 外来者视角(【原文】p.23)
- **"一定会有新人进入到这个集体"**
- **"新人 = 观众视角"**
- 看他如何一步步被同化或者反抗这个集体
- 例子:《穿普拉达的女王》、《飞越疯人院》

### 关键人物(【原文】p.23)
- **"一个完全赞同这个群体的关键人物"**
- ta 是集体的完全捍卫者,是外来者的最大劲敌
- 外来者所遭受的打击往往来源于这种人

### 被制度化的师徒模式(【原文】p.23)
- **师徒关系是典型的被制度化入口**——《入殓师》《大红灯笼高高挂》
- 新人由经验丰富的人带入群体,借此揭开体制

## Context & problem
**这个类型解决的问题是**:怎么写"不公平的斗争"?

当敌人不是一个人,而是一整套规则、一群人、一种制度——主角就处于**力量悬殊**的起点。被制度化的价值在于:**它不让主角轻易获胜**。主角能做的只有三件事:被同化、摧毁它(代价巨大)、自杀。

这种悲剧感让被制度化成为**艺术电影最偏爱的类型**——它能承载最深的批判和最沉重的主题。

## Claims

1. **冲突本质是"力量悬殊"**(【原文】p.28)
   - 主角再聪明、再勇敢,也很难在体量上战胜集体
   - 这让被制度化自带**悲剧底色**

2. **外来者视角 = 观众视角**(【原文】p.23)
   - 几乎所有被制度化电影都有一个"**新人入场**"的结构——《穿普拉达的女王》的安迪、《飞越疯人院》的麦克墨菲、《入殓师》的小林大悟
   - 这是观众的"**代入门**"——观众通过新人看集体

3. **关键人物(集体捍卫者)是主角最大的敌人**(【原文】p.23)
   - 不是集体最高领导,而是**最忠诚的守门人**
   - eg.《穿普拉达的女王》里真正给安迪施压的不是 Miranda 本人,而是办公室里的其他资深员工(尤其是 Emily)

4. **三种选择 + 三种牺牲 = 9 种组合**
   - 【你的推断】反叛型 × 自杀 = 最激烈的悲剧(《飞越疯人院》麦克墨菲)
   - 天真型 × 加入组织 = 最温和的成长(《穿普拉达的女王》)
   - 公司型 × 毁灭群体 = 最意外的反转(罕见)

5. **集体的表征是具体的**(【原文】p.23)
   - 服装、口号、食物、房间、作息时间、课间体操——这些**细节就是集体的权力在视觉上的展示**
   - 【我的理解】这让被制度化电影特别适合**视听表现**——可以通过这些细节细腻地传达压制感。

## Method / How to use

### 设计 4 步
1. **定集体的类型**(学校 / 家庭 / 公司 / 组织 / 国家)
2. **定集体的具体表征**(服装、口号、食物、空间规则、时间规则)
3. **定新人的选择**(反叛 / 天真 / 公司)
4. **定最终牺牲**(加入 / 毁灭 / 自杀)+ **关键人物**(谁是最大对手)

### 艺考编讲故事的被制度化套路
选一个"**有严格规则的小群体**":
- 集训班
- 夏令营
- 新公司
- 一个村庄
- 一所学校

让主角**作为新人进入**,**设计关键人物**(最资深的成员),走"初入 → 受挫 → 反抗 → 最终选择"的四段式。

### 被制度化 checklist
- [ ] 集体有**明确的规则**(不是抽象的"社会")吗?
- [ ] 规则有具体的表征(视听可见)吗?
- [ ] 主角是**以新人身份**进入的吗?
- [ ] 有没有一个"**关键人物**"作为集体的化身?
- [ ] 主角最后做出选择了吗?(反叛 / 天真 / 公司)
- [ ] 结局是三种牺牲之一吗?

## Debates & limits
- **"力量悬殊"的极限**。【学界常见解释】被制度化类型的主角几乎注定无法战胜集体——这是否太悲观?Snyder 在原书里承认"**有极少数情况下主角能毁灭集体**",但"**需要极大的代价**"。
- **外来者的必要性**。不是所有被制度化都有"新人入场"——《大红灯笼高高挂》的颂莲算新人,但《金陵十三钗》里的妓女不是新人。这说明"外来者视角"是**常见技法**而非**铁律**。
- 【我的理解】中国艺考里,被制度化是**最容易拿分的类型**——因为它能承载"对某种体制的批判",老师天然看重。但写作者要注意:**不要流于表面的反叛**,必须展示集体的诱惑(为什么有人真心认同)。
- 【你的推断】被制度化和变迁仪式可以混血——主角可能一开始反叛(被制度化)但最终转向接受自己(变迁仪式)。《东京物语》就是这种混血。

## Directing takeaways
1. **开场要用"集体的视听表征"压住主角**——制服、钟声、统一动作。让观众立刻感受到规则。
2. **关键人物要比最高领导更重要**。最高领导常常是"抽象符号",关键人物是"具体的压迫"。
3. **外来者入场时的"一两个细节"**(走错门、穿错衣、说错话)是立刻确立"新人身份"的最快方法。
4. **让主角尝试"融入"过**——不能一开始就反抗。融入的过程才能展示集体的诱惑。
5. **结局必须明确是"三种牺牲"之一**——不要含糊。含糊会让观众觉得主角"逃掉了",丢失了类型的悲剧力量。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 被制度化 = 反乌托邦 | 被制度化的集体可以是监狱、家庭、公司、学校——不必然是反乌托邦。 |
| 被制度化 = 个人 vs 社会 | 不是整个"社会",而是**有明确边界的小集体**——观众能在 2 小时内理解它的规则。 |
| 外来者 = 主角 | 外来者通常是主角,但不必然。有时主角是"被体制化已久的老人",新人是配角。 |
| 关键人物 = 反派 | 关键人物**不是坏人**——他只是集体规则的执行者,他自己也是被制度化的。这让他成为最复杂的角色。 |
| 被制度化 = 悲剧 | 被制度化可以是喜剧(《穿普拉达的女王》),关键看"牺牲"是哪种——"加入组织"可以是轻喜剧。 |
