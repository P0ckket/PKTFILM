# 变迁仪式型(Rites of Passage)

## TL;DR
**变迁仪式(Rites of Passage)** 是 Snyder 10 类型里的"内在转变"型——**主角因生活中的苦难和折磨而产生转变**。三元素是:**一个"生活的问题"(无形的、无法轻易处理的) + 一个"错误的方法"(试图用自己的力量解决但行不通) + 一个"真正解决问题的方法"(意识到改变不了世界,于是开始改变自己)**。它的主题是"**人如何与不可控的现实相处**",两难是"**接受现实 vs 逃避、自我欺骗**"。青春期、中年危机、老年、沉迷、死亡都是它的子类。

## Definitions

- **变迁仪式**:主角因生活中的苦难和折磨而产生的内在转变。
- **生活的问题**:无形、模糊、无法轻易处理的现实困境。
- **错误的方法**:主角一开始选择的应对方式——通常是逃避或自欺。
- **真正的方法**:主角意识到"改变不了世界,开始改变自己"。

## Key terms

### 三元素(【原文】p.22)
| 元素 | 特征 | 例子 |
|---|---|---|
| 生活的问题 | 无形的、模糊的、无法轻易处理、现实问题——自己无法解决 | 青春期、中年危机、老年、伴侣死亡、生老病死 |
| 错误的方法 | 试图依靠自己的力量解决但行不通;逃避 / 消极 | 不停的恋爱(被嫌弃的松子) |
| 真正的方法 | 意识到改变不了世界,于是开始改变自己 | 需要拥有独立的人格(被嫌弃的松子) |

### 两难(【原文】p.28)
**接受现实 vs 逃避 / 自我欺骗**
"残酷的现实下解决办法的选择 / 心境的选择"

### 重点总结(【原文】p.22)
- 寻常问题
- 过激方式解决
- 回归正确方法
- **主人公最后不一定都能战胜困难**——这是与麻烦家伙的关键区别

### 衍生类型(【原文】p.22)
| 衍生 | 特征 | 例子 |
|---|---|---|
| 中年阶段 | 上有老下有小,经过波折改变现状 | — |
| 分离仪式 | 分手、离婚、离家、争夺抚养权 | 一次别离 |
| 青春期阶段 | 青春期、成长的冲动、成年考验 | — |
| 老年阶段 | 面对自然衰老、疾病缠身、渐渐逼近的死亡 | — |
| 沉迷阶段 | 吸毒、酗酒带来的痛苦 | — |
| 死亡类型 | 出现即将死去或已经离世的人物 | 凡夫俗子 |

### 案例拆解(【原文】p.22)
**《被嫌弃的松子的一生》**:
- 主人公:松子
- 生活的问题:渴望爱和被爱
- 错误的方法:不停的恋爱
- 真正的方法:松子需拥有独立的人格

## Context & problem
**这个类型解决的问题是**:怎么写"没有敌人"的故事?

变迁仪式的最大特点是——**没有外部反派**。主角的敌人是"生活本身"——青春会结束、父母会老、爱人会死、身体会衰。这些事没法"战胜",只能**接受**。

传统剧作的"冲突 → 胜利"模式在这里失效。Snyder 的贡献是指出:**变迁仪式的胜利是"心境的胜利",不是"外部的胜利"**。松子没有"赢"——她甚至没活下来——但她**变了**。这种变化就是成功。

## Claims

1. **生活的问题是"无形的"**(【原文】p.22)
   - 与"敌人明确"的麻烦家伙/被制度化不同,变迁仪式的问题**没法指着说"就是他"**。
   - 问题是**状态**,不是**对手**。

2. **错误的方法是"过激"的**(【原文】p.22)
   - 主角一开始用力过猛——逃避、酗酒、滥情、自欺
   - 【我的理解】这是变迁仪式的 "Fun and Games + Bad Guys Close In" 节拍——主角以为自己的方法管用,但事情越来越糟。

3. **"改变世界 → 改变自己"的觉悟**(【原文】p.22)
   - 【原文】"主角意识到自己改变不了这个世界,于是开始改变自己"
   - 这是变迁仪式的 "Dark Night of the Soul" 节拍的核心转折。

4. **不一定都能战胜困难**(【原文】p.22)
   - 【原文】"主人公最后不一定都能战胜困难"
   - 变迁仪式接受**悲剧结局**——只要主角"变了",就成功了。
   - 【你的推断】这让变迁仪式在文艺片里比商业片里更常见——商业片观众更期待明确胜利。

5. **类型的覆盖范围=人生的所有阶段**(【原文】p.22)
   - 青春期、成年、中年、老年、死亡——这五个子类其实**覆盖人的一生**
   - 【我的理解】这使得变迁仪式成为**所有人都能共鸣**的类型——每个观众都在经历某个"变迁"。

## Method / How to use

### 设计 4 步
1. **定"生活的问题"**:选一个无形的、无法轻易解决的困境
2. **设计"错误的方法"**:主角第一反应的应对方式(通常是逃避或自欺)
3. **让错误方法逐步失败**:每一次尝试让情况更糟
4. **触发觉悟时刻**:主角意识到"要变的是自己",而不是外部

### 艺考编讲故事的"变迁仪式"套路
选一个"**无法战胜的生活状态**":
- 父亲的疾病
- 母亲的衰老
- 爱情的结束
- 青春期的迷茫
- 童年的阴影

主角一开始**用错误的方法应对**(逃避、对抗、自欺)——让他逐步触壁——最后**接受并改变自己**。

### 变迁仪式 checklist
- [ ] 生活的问题是"无形的"吗?(不是具体敌人)
- [ ] 主角有一个"错误的方法"吗?
- [ ] 错误方法是否逐步失败?
- [ ] 主角是否有一个**觉悟时刻**(从改变世界转向改变自己)?
- [ ] 主角是否**变了**?(结局可以赢可以输,但必须变)

## Debates & limits
- **"主角不一定战胜困难"的商业风险**。【学界常见解释】变迁仪式是文艺片最常见的类型,但商业市场接受度较低——因为观众没有"爽感"。
- **变迁仪式与"成长片(coming-of-age)"的关系**。成长片通常是"青春期阶段"子类,但变迁仪式还包括中年/老年——比成长片更宽。
- 【我的理解】艺考编讲故事写变迁仪式很难,因为 2000 字很难写清"渐变的心境"。建议选择**短时间内集中发生的变迁**(如一次葬礼、一次重聚、一次雨夜)而不是"一生的变迁"。
- 【你的推断】东方叙事传统对变迁仪式非常友好——《饮食男女》《东京物语》《一一》《海街日记》都是典型变迁仪式。相比之下,好莱坞商业片里的变迁仪式往往要和其他类型混血(如《毕业生》是变迁仪式 + 伙伴之情)。

## Directing takeaways
1. **变迁仪式不要"给敌人"**。即使有对立人物,也要把他塑造成"生活的化身",而不是坏蛋。
2. **觉悟时刻最好是"安静的"**——不是嘶吼,不是打斗,而是一个沉默的瞬间(看着雨、看着窗、看着一张旧照片)。
3. **结局不必胜利**。观众看完要感到"生活还会继续,但主角已经不一样了"。
4. **用季节/时间作为变迁的外显标志**——从夏到冬、从黑发到白发、从同一地点的不同光线。
5. **错误方法的戏要写足**——不要太快让主角觉悟,让观众先看到他如何撞南墙。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 变迁仪式 = 成长片 | 成长片是变迁仪式的**一个子类**(青春期阶段)。 |
| 变迁仪式 = 悲剧 | 变迁仪式可以是悲剧,也可以是喜剧——关键是"变",不是输赢。 |
| 变迁仪式 = 心理戏 | 变迁必须**外显化**——通过行动、道具、关系变化展示,不是独白。 |
| 生活的问题 = 困境 | 生活的问题是**持续状态**,困境是**具体事件**。两者性质不同。 |

## Open questions
- **变迁仪式与东方"无常"哲学**的关系——东方文化里的"接受"是否和 Snyder 的"改变自己"完全对应?
- **集体变迁仪式**是否存在?——一群人共同经历变迁(如《海边的曼彻斯特》里的家庭)。
- 【待核实】Snyder 原书对本类型的 beat sheet 是否与原笔记的"寻常问题 → 过激方式 → 回归正确方法"三段式一致?
