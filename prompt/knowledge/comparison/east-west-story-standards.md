# 东西方故事价值标准对比

## One-liners
- **东方**:人应该是什么样子——寄托真善美的精神和品质
- **西方**:人可能是什么样子——窥见现实中人性之百态

## Matrix
| 维度 | 东方标准 | 西方标准 |
|---|---|---|
| **价值取向** | 文以载道(teach) | 探索人性(explore) |
| **核心问题** | 人应该是什么样子 | 人可能是什么样子 |
| **偏好品质** | 真善美、道德楷模 | 复杂、灰色、真实 |
| **适配考试** | **统考** | **校考** |
| **代表传统** | 儒家叙事、教化寓言 | 希腊悲剧、现代小说 |
| **英雄定义** | 道德高尚的人 | 经历考验后变化的人 |
| **反派定义** | 道德败坏的人 | 有理由的对手 |
| **结局偏好** | 善恶有报、正义获胜 | 开放式、代价式、悲剧也可 |
| **观众期待** | 感动、共鸣、uplift | 思考、震撼、unsettled |

## Non-overlapping differences
| 维度 | 东方独有 | 西方独有 |
|---|---|---|
| 叙事伦理 | "教化"是故事的义务 | 故事没有道德义务 |
| 主角缺陷 | 缺陷是"需要克服的"→ 最终变好 | 缺陷可以是"本质的"→ 不一定变好 |
| 家庭叙事 | 家和万事兴 / 血浓于水 | 家庭是冲突的来源(不必和解) |
| 集体叙事 | 个人服从集体(正面) | 个人对抗集体(正面) |

## Confusions
| 易混 | 正解 |
|---|---|
| 东方 = 保守,西方 = 先锋 | 东方也有先锋(第五代/第六代),西方也有保守(好莱坞商业片) |
| 统考只能写东方标准 | 统考**偏好**东方标准,但也接受有深度的西方式角色 |
| 校考必须写西方标准 | 校考**奖励**人性复杂度,但不排斥东方价值 |
| 真善美 = 煽情 | 真善美是价值取向,不是表达方式。可以用克制的方式表达真善美 |

## Analysis recipe
1. **先判断考试类型**(统考 / 校考)
2. **统考 → 偏向东方标准**:主角有缺陷但最终克服,结局有正向价值
3. **校考 → 偏向西方标准**:主角可以灰色,结局可以不完美,但必须有深度
4. **两者共同要求**:**故事的记忆点** = 人物个性鲜明 + 情节曲折精彩(p.40)
5. **两者共同要求**:**动机-行动-矛盾**三角必须清晰(p.40)

### 专业标准(p.40)——超越东西方的硬指标
- **整篇故事能不能用一句话概括?**
  - 一句话故事 = 做什么 + 结局
  - 如果故事中缺乏主人公、情节线杂乱,则难以用一句话概括
- 这是**东方和西方共同的专业标准**——无论价值取向如何,故事必须清晰。
