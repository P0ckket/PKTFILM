# 形式主义 vs. 现实主义（Formalism vs. Realism）

## One-liners

- **形式主义（Formalism）**：用电影技术手段主动赋予戏剧行动以意义——摄影机不是记录工具，而是表意工具。
- **现实主义（Realism）**：尊重电影的纪录本质，最小化人工干预，让现实自己说话——观众保有情绪自主权。

---

## Matrix

| 维度 | 形式主义（Formalism） | 现实主义（Realism） |
|---|---|---|
| **核心主张** | 电影技术"通过形式赋予戏剧行动以意义" | 电影是纪录媒介，"强调影像作为纪录工具的价值" |
| **理论来源** | 苏联，1920年代，谢尔盖·爱森斯坦（Sergei Eisenstein） | 法国/德国，1930-40年代，安德烈·巴赞（André Bazin）/ 齐格弗里德·克拉考尔（Siegfried Kracauer） |
| **先驱人物** | 乔治·梅里爱（Georges Méliès, 1861-1938），"在银幕上制造幻想" | 卢米埃兄弟（Lumière Brothers, 1895-1905），拍摄日常生活 |
| **主观性** | 高：主观性（从人物或摄影机视角叙事） | 低：尽量客观，给观众留下判断空间 |
| **剪辑** | 蒙太奇（montage）、快速剪切、跳切 | 连续性剪辑（continuity editing）、长镜头 |
| **声音** | 画外配乐（nondiegetic music），强化情绪操控 | 画内音为主（diegetic sound），拒绝情绪操控 |
| **灯光** | 表现性布光（expressionistic lighting），高对比 | 自然光或最小化人工光 |
| **摄影机角度** | 动态角度：大量俯仰、荷兰角、极端视角 | 平视为主，减少主观干预 |
| **景深** | 浅焦，引导注意力 | 深焦（deep focus），让观众自选注意点 |
| **拍摄地点** | 摄影棚为主，人工布景 | 实景拍摄（on-location） |
| **对观众的态度** | 主动引导情绪与解读 | 【原文/作者观点】"更民主，因为它不操控观众的情绪（more democratic, since it does not manipulate audiences' emotions）" |

## Non-overlapping Differences

### 形式主义独有

- **蒙太奇理论**：爱森斯坦认为两个镜头的碰撞产生第三层意义（思想火花），镜头不只是记录，而是思维单元。
- **表现性布景（Expressionistic Set Design）**：布景本身以失真的方式表达人物心理（如德国表现主义电影的扭曲建筑）。
- **音画对位（Counterpoint）**：音乐与画面形成张力，而非简单配合，产生更复杂的语义。

### 现实主义独有

- **深焦摄影（Deep Focus）**：前景至背景同时清晰，模拟人眼真实感知，观众可自主决定关注哪里（奥逊·威尔斯/格雷格·托兰德在《公民凯恩》中的贡献）。
- **长镜头（Long Take）**：尊重时空连续性，不切断现实的流动；安德烈·巴赞认为长镜头才是电影最诚实的形式。
- **非职业演员**：意大利新现实主义（Italian Neorealism）大量使用非职业演员，追求未经训练的真实感。

## Confusions

| 混淆点 | 澄清 |
|---|---|
| 形式主义 = 视觉华丽 | 错误。形式主义是关于技术手段的主动叙事介入，不等于画面精美；一部手持粗粒影像也可以是形式主义的 |
| 现实主义 = 无技术 | 错误。深焦、长镜头、实景等都是高度技术化的选择，只是这些技术指向"减少介入感"的美学目标 |
| 现实主义 = 好，形式主义 = 坏 | 错误。二者是不同美学哲学，各有其批判性传统和适用场景 |
| 形式主义必须用蒙太奇 | 不准确。表现性布光、荷兰角、音乐操控都是形式主义工具，蒙太奇只是其中最著名的一种 |

## Analysis Recipe

**分析一部影片的形式主义/现实主义倾向**（【我的理解】）：

```
第一步：看剪辑风格
├─ 快速剪切/蒙太奇 → 形式主义
└─ 长镜头/连续性剪辑 → 现实主义

第二步：看摄影机角度
├─ 大量极端角度/荷兰角 → 形式主义
└─ 平视为主/眼平摄影 → 现实主义

第三步：看布光
├─ 高对比/表现性布光 → 形式主义
└─ 自然光/均匀照明 → 现实主义

第四步：看声音
├─ 大量非叙事性配乐 → 形式主义
└─ 以环境音/画内音为主 → 现实主义

第五步：综合判断
── 大多数影片是两者的混合，指出在哪些维度偏向哪一端，比强行二选一更有分析价值
```

---

## Directing Takeaways

1. **形式主义的陷阱是"为技巧而技巧"**：每一个技术干预都要有叙事理由——炫技但无功能是初学者的常见错误。
2. **现实主义的陷阱是"无聊"**：去掉技巧干预不等于自动产生意义，仍需在构图、走位、演员表演上精心设计。
3. **大多数商业片是混合体**：用连续性剪辑（现实主义框架）处理基础叙事，用蒙太奇/音乐（形式主义工具）处理情绪高潮。
4. **艺考分析时明确坐标**：不必把影片强行归入某一极，但要能说明"在哪个层面倾向哪种美学，产生了什么效果"。

|---|---|---|
| 《战舰波将金号》(Battleship Potemkin, 1925) | 谢尔盖·爱森斯坦 | 【原文/作者观点】蒙太奇理论的奠基性示范，敖德萨阶梯段落经典蒙太奇 |
| 《惊魂记》(Psycho, 1960) | 希区柯克 | 【原文/作者观点】淋浴场景的快速蒙太奇剪辑；52个镜头在45秒内完成 |
| 《理查三世》(Richard III, 1995) | 理查·朗克雷尼 | 【原文/作者观点】构图与镜头紧迫感强化人物的道德阴谋性 |
