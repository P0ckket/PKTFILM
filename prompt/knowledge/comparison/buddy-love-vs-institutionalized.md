# 伙伴之情 vs 被制度化(两种"关系型"故事)

## One-liners
- **伙伴之情**:两个人因互补而靠近 → 因冲突而分离 → 因需要而重聚
- **被制度化**:一个人进入集体 → 被同化或反抗 → 被吞噬或摧毁集体

## Matrix
| 维度 | 伙伴之情 | 被制度化 |
|---|---|---|
| **核心关系** | 1 对 1(双主角) | 1 对多(个体 vs 集体) |
| **关系方向** | 互补、互利 | 单向压制 |
| **三元素** | 不完整主角 + 伙伴 + 复杂情况 | 有规则的集体 + 选择 + 牺牲 |
| **冲突本质** | 信息错位(误解对方) | 力量悬殊(打不过集体) |
| **两难** | 社交关系 vs 自我秉性 | 两种群体观念中选择 |
| **典型弧线** | 厌恶→互补→分离→重聚 | 进入→受压→反抗→被同化/毁灭/自杀 |
| **结局情感** | 温暖、治愈、成长 | 悲壮、沉重、批判 |
| **适配情绪** | 喜剧/温情/浪漫 | 悲剧/严肃/艺术 |
| **"观众代入"方法** | 通过主角 1 的缺陷代入 | 通过新人/外来者代入 |
| **关键配角** | 伙伴(催化剂) | 关键人物(集体捍卫者) |

## Non-overlapping differences
| 维度 | 伙伴之情独有 | 被制度化独有 |
|---|---|---|
| 关系平等性 | 两人地位对等(虽不同但互补) | 个体永远处于弱势 |
| 分离的意义 | 分离是为了更好的重聚 | 分离几乎不存在(被吞噬就是结局) |
| 主题生成 | 两人碰撞 = 主题 | 个体抗争 = 主题 |
| 喜剧可能性 | 高(《穿普拉达的女王》可以是喜剧) | 低(喜剧很少见) |

## Confusions
| 易混 | 正解 |
|---|---|
| 伙伴之情有"复杂情况" = 集体? | 伙伴之情的"复杂情况"是**两人关系的外部障碍**,不是"集体"。eg. 种族矛盾是外部障碍,不是一个组织。 |
| 被制度化有"关键人物" = 伙伴? | 关键人物是**集体的化身**,不是伙伴。他的功能是压制主角,不是帮助主角。 |
| 《穿普拉达的女王》是哪个? | 它同时含有两者——安迪的伙伴之情线(与Emily/Nigel)+ 被制度化线(与时尚行业)。**但核心是被制度化**(Snyder归类)。 |
| 都有"新人进入" = 一样? | 伙伴之情的"新人"是为了**遇见伙伴**;被制度化的"新人"是为了**揭示集体的规则**。入场目的不同。 |

## Analysis recipe
当你不确定一个故事属于哪种类型时:
1. **主角面对的是"一个人"还是"一个体系"?**
   - 一个人 → 伙伴之情
   - 一个体系 → 被制度化
2. **结局是"两人关系变化"还是"个人 vs 集体的结果"?**
   - 关系变化 → 伙伴之情
   - 个人vs集体结果 → 被制度化
3. **主角的弱点是"人际能力"还是"独立性"?**
   - 人际能力缺陷(不会与人相处) → 伙伴之情
   - 独立性缺陷(不敢对抗规则) → 被制度化

### 混血类型
很多优秀电影是两者的混血:
- **《死亡诗社》**:被制度化(学校体系)+ 伙伴之情(师生关系)
- **《放牛班的春天》**:被制度化(教养院)+ 伙伴之情(老师+学生)
- **《入殓师》**:被制度化(师徒体系)+ 变迁仪式(主角内心变化)
