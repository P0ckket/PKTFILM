# 《百元之恋》案例拆解

## TL;DR
原笔记 p.48-49 把《百元之恋》作为**人物弧光机制的完整教学案例**——从缺点到目标的完整链条 + 五事件大纲的实操示范。这是本库中**最接近艺考编讲故事实战**的案例：一个有缺陷的女主，一个配角，一段关系变化，一个"没有完美完成"的结局。

## 人物弧光链条(p.48)

```
缺点 ====> 困境 =======> 欲望 =========> 目标
起点                                      终点
```

### 女主的弧光
| 环节 | 内容 |
|---|---|
| **缺点** | 懒惰、懦弱、自卑、执行力差、没有动力改变自己 |
| **困境** | 不能接纳自己 |
| **欲望**（抽象） | 接纳自己、改变自己 |
| **目标**（具体） | 我要打完一场正式的拳赛 |
| **弧光终点** | 修正自己的缺点 |

### 配角
- **身份**：退役拳手
- **功能**：与主角互补，参与/影响主角目标的实现
- **关系轨迹**：二人关系由坏变好

### 欲望 vs 目标的区分（p.48）
| 维度 | 欲望 | 目标 |
|---|---|---|
| 性质 | **抽象** | **具体** |
| 示例 | 我想变得富有 | 我要努力工作提升薪资 / 我要去抢劫银行 / 天天买彩票等中奖 |
| 在故事中的位置 | 开场建立 | 激励事件确立 |

**同一个欲望可以生出完全不同的目标**——目标的选择决定故事类型。

## 五事件实操（p.48-49）

### 开场事件（p.48）
- 女主在家里懒惰，被家人排挤，发生争执
- **功能**：暴露缺点 + 可怜（被排挤）+ 冲突开场

### 激励事件（p.48）
- 女主离开家后去到拳馆，看到自己仰慕的配角的退役拳赛——但配角打输了
- 拳赛结束，女主去找配角，表达仰慕 + 想像他一样打拳
- **配角泼冷水**，主角不服气、很伤心
- **决定学拳击、打比赛、证明给配角看**
- **功能**：配角出场 + 冲突产生 + 具体目标确立

### 转折事件（p.49）
- 女主虽然一腔热血，但**执行力差、很难专注**
- 配角进场——女主训练到深夜，配角回来收拾东西
- **配角过去"爹味指导"**，女主表面很抗拒
- 配角对主角产生帮助，女主可能真的把话听进去了
- **意识到了自己的缺点**
- **功能**：小麻烦 + 意识到缺陷 + 关系开始改善

### 危机事件（p.49）
- **主角没有办法拿到打拳赛的资格**——除非通过考核
- **没有人愿意带她训练**
- 主角去找配角帮她训练
- 在训练过程中：**主角修正缺点 + 配角关系慢慢改善**
- **功能**：大麻烦 + 逐步改善缺陷 + 逐步改善关系

### 高潮事件（p.49）
- **女主拳赛——可能没赢，但是打完了**
- 比赛结束，主角配角激动的拥抱
- **功能**：目标未完美完成 + 缺陷弥补 + 关系顶点

## 与五事件大纲模板的完美对应

| 事件 | 模板要求 | 《百元之恋》实现 |
|---|---|---|
| 开场 | 冲突开场 + 可怜可爱>缺陷 + 原始欲望 | 争执 + 被排挤(可怜) + 接纳自己(欲望) |
| 激励 | 配角出场 + 冲突 + 具体目标 | 退役拳手 + 泼冷水 + 打完一场拳赛 |
| 转折 | 小麻烦 + 意识缺陷 + 关系改善开始 | 执行力差 + 听进去了 + 抗拒→接受 |
| 危机 | 大麻烦 + 改善缺陷 + 改善关系 | 没资格 + 训练修正 + 配角关系深化 |
| 高潮 | 目标未完美 + 缺陷弥补 + 关系顶点 | 可能没赢但打完了 + 拥抱 |

**【我的理解】**：《百元之恋》是**五事件大纲的标杆案例**——每一条 checklist 都完美命中。用它做模板练习，写任何题目都不会跑偏。

## 三种弧光启动方式（p.48，从案例中提炼）

| 启动方式 | 模式 | 适用题材 |
|---|---|---|
| **被动型** | 一个我本来很讨厌/有过节/有误会的人突然找上我，需要我帮助 → 我最终决定帮他 | 伙伴之情 |
| **救赎型** | 主角出于苦衷做损人利己的事(道德瑕疵)，看到行为如何伤害配角 → 决定改过自新帮助配角 | 变迁仪式 |
| **责任型** | 一个需要成长的主角突然被委托一个责任/人物(照顾家人/教导学生)→ 一开始排斥 → 慢慢改变 → 完成责任完成成长 | 伙伴之情/变迁仪式混血 |
