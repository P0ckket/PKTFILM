## 如何持续完善

### 加入新材料
1. 把新 PDF 放到 `sources/raw-extract/` 下提取
2. 在对应的 `story-elements/` 或 `concepts/` 文件里**追加内容**，不要新建重复文件
3. 更新文件里的 `Changelog` 字段（格式：`- YYYY-MM-DD vX.X — 描述`）
4. 更新本 README 的对应行

### 新建一篇笔记
复制以下模板粘贴到新 `.md` 文件：

```markdown
---
name: 笔记标题
description: 一句话描述（用于索引）
type: concept | theory | comparison | director-note
status: seed | draft | stable
tags: []
sources:
  - type: notebook | book | film
    cite: "来源"
    locator: "页码/时间码"
created: YYYY-MM-DD
updated: YYYY-MM-DD
---

# 标题

## TL;DR

## 核心内容

## Links & relations

## Changelog
- YYYY-MM-DD v1.0 — 初版
```
