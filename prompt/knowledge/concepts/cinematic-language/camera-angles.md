# 摄影机角度（Camera Angles）

## TL;DR

【原文/作者观点】"摄影机角度是表达场景情绪或传达人物处境与社会地位的重要工具（Camera angle is an important tool to express the mood or convey a character's standing or situation）。"不同角度在观众潜意识中构建权力关系与情绪基调，是编导分析的核心视听语言元素之一。

---

## Definitions

**摄影机角度（Camera Angle）**：摄影机相对于被摄对象在垂直/水平/旋转维度上的位置关系。与景别（shot size）不同，角度关注的是"从哪里看"，而非"看多少"。

---

## Key Terms

| 术语（中文） | 定义 | 英文 |
|---|---|---|
| 平视角度 | 摄影机与被摄对象眼睛齐平，最中性客观 | Eye-Level Shot |
| 俯视角度 | 摄影机高于对象向下拍，使对象显得弱小脆弱 | High-Angle Shot |
| 仰视角度 | 摄影机低于对象向上拍，使对象显得强大/威胁 | Low-Angle Shot |
| 斜角（荷兰角） | 摄影机水平倾斜，画框倾斜，制造不安感 | Dutch (Tilted) Angle |
| 鸟瞰角度 | 摄影机直接从正上方向下拍，呈现平面图式 | Bird's-Eye View / Overhead Shot |
| 正面角度 | 正对被摄主体，直接、对称 | Frontal Angle |
| 侧面角度 | 90°侧拍，强调轮廓与运动方向 | Side / Profile Angle |
| 背面角度 | 拍摄主体背部，制造神秘感或跟随感 | Rear / Back Angle |

---

## Context & Problem

【学界常见解释】角度与景别往往被学生混淆或合并处理，但它们是两个独立维度：同一个近景可以是俯拍或仰拍，产生截然不同的情绪效果。中国艺考课程通常以"平/俯/仰/正/侧/背"六个维度覆盖摄影机位置，英文文献在此基础上单独列出"荷兰角（Dutch Angle）"和"鸟瞰（Bird's-Eye）"作为特殊类别。

---

## Main Content

### 角度情绪矩阵

| 角度 | 摄影机位置 | 对被摄对象的视觉效果 | 常见情绪 / 叙事功能 |
|---|---|---|---|
| 平视（Eye-Level） | 与对象眼睛齐平 | 中性，无失真 | 客观叙述，纪录片常用；观众与对象地位平等 |
| 俯视（High-Angle） | 高于对象，朝下 | 对象被压缩、缩小、看起来脆弱 | 脆弱、受控、渺小；权威视角（监视感） |
| 仰视（Low-Angle） | 低于对象，朝上 | 对象被拉高，看起来强大、压迫 | 张力、威胁、不适、英雄感（取决于语境） |
| 荷兰角（Dutch Angle） | 机身水平倾斜 | 画面倾斜，构图失衡 | 心理扰动、焦虑、心理不稳定 |
| 鸟瞰（Bird's-Eye） | 正上方俯拍 | 对象呈平面图案，失去立体感 | 建立空间规模；宿命感；全知视角 |
| 正面（Frontal） | 正对对象 | 直接、对称、坦诚 | 对峙、坦白、自我展示 |
| 侧面（Side/Profile） | 90°侧拍 | 强调轮廓与运动 | 叙事推进、人物内心隐藏 |
| 背面（Rear/Back） | 拍主体背部 | 隐藏表情，跟随运动 | 神秘、跟随、目的地感 |

### 经典案例

| 影片 | 角度选择 | 叙事 / 情绪功能 |
|---|---|---|
| 《低俗小说》(Pulp Fiction, 1994) | 汽车后备箱内仰拍 | 【原文/作者观点】低角度 + 从密闭空间仰视，制造人物的压迫感与观众的不安 |
| 《碟中谍》(Mission: Impossible, 1996) | 多处荷兰角 | 【学界常见解释】荷兰角强化心理紧张与视觉不稳定感，暗示情报世界的道德倾斜 |
| 《李尔王》(King Lear, 2018) | 航拍/鸟瞰全场 | 【原文/作者观点】鸟瞰确立空间规模与人物之间的权力关系格局 |
| 《国王的演讲》(The King's Speech, 2010) | 对国王的轻微低角度 | 即使在脆弱场景中，低角度仍维持皇室的象征性高度 |

## Method / How to Use

**艺考分析角度的三问**（【我的理解】）：

1. **摄影机比对象高还是低？** → 高=权力在上（压制），低=权力在下（仰视）
2. **画框是否水平？** → 倾斜=心理扰动，水平=秩序感
3. **观众被置于什么位置？** → 俯视视角让观众感到优越或全知；仰视让观众感到渺小或被威胁

## Debates & Limits

- 【学界常见解释】仰拍不总是"英雄化"——在恐怖片中仰拍杀手会产生威胁感而非崇拜感，角度意义高度依赖其他视听元素（光线、音乐、叙事语境）。
- 【学界常见解释】荷兰角被过度使用后（尤其在 90 年代好莱坞），已成为近乎陈词滥调的"紧张信号"，当代导演使用需更加克制。
- 【你的推断】在分析时应避免将角度意义孤立化——同一个低角度，配上柔和自然光与配合音乐，读法可能与配上硬光 + 不和谐音完全相反。

---

## Directing Takeaways

1. **默认从平视出发**：每次偏离平视都是一次有意识的选择，需要理由支撑。
2. **用角度建立权力地图**：在同一场景中，对两个人物使用不同角度，可以在不说一句台词的情况下建立他们的权力关系。
3. **荷兰角是应急工具**：对长片克制使用；短片/艺考视频可以用一两处来增强风格感，但不要滥用。
4. **鸟瞰做开幕/收尾**：鸟瞰具有"神之视角"色彩，适合做段落的宏观定位或主题性收尾，而非日常叙事。

## Common Confusions

| 混淆点 | 正确区分 |
|---|---|
| 仰拍 vs. 英雄化 | 仰拍让对象视觉上更大，但"英雄"还是"威胁"取决于叙事语境 |
| 俯拍 vs. 道德评判 | 俯拍不等于导演在"审判"人物；也可以只是建立空间信息 |
| 荷兰角 vs. 摄影机不稳 | 荷兰角是有意为之的倾斜；手持摄影的不稳是另一种效果（见摄影机运动笔记） |
| 平视 vs. 无风格 | 平视是最接近"透明"的角度，但本身就是一种风格选择（客观性、可信度） |
