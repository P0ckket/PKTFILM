# 场面调度与走位（Blocking & Mise-en-Scène）

## TL;DR

【原文/作者观点】"走位（Blocking）是人物、道具与场景布置之间的物理关系（the physical relationships among characters, props, and the set）。"走位是场面调度（mise-en-scène）的核心组成部分——谁站在哪里、面对谁、距离多远，在不说一句台词的情况下就已经建立了人物的权力关系、情感距离与主题立场。

---

## Definitions

**走位（Blocking）**：在拍摄前导演与演员共同确定的人物在场景中的位置、移动路径与相互关系。

**场面调度（Mise-en-Scène）**（法语，意为"置于画面中"）：影响银幕上视觉构成的所有元素的总称，包括：场景设计、道具、服装、灯光、演员表演、走位与摄影机位置。走位是其中最直接涉及人物关系的维度。

---

## Key Terms

| 术语（中文） | 定义 | 英文 |
|---|---|---|
| 走位 | 人物与道具在场景空间中的物理位置安排 | Blocking |
| 场面调度 | 影响画面视觉构成的所有元素总称 | Mise-en-Scène |
| 180°轴线规则 | 所有摄影机必须在两角色连线同侧的规则，维持空间连贯性 | 180-Degree Rule |
| 轴线 | 场景中两个主要人物或运动对象之间的假想连线 | The Line / Action Line |
| 越轴 | 违反180°规则，摄影机越过轴线 | Crossing the Line |
| 半圆阵形 | 舞台传统：演员呈半弧形排列，自然面向观众方向 | Semi-Circular Formation |
| 空间压缩 | 摄影机长焦压缩前景与背景距离，影响走位的视觉读法 | Spatial Compression |

---

## Context & Problem

【学界常见解释】走位的概念来自舞台剧传统，但在电影中因"摄影机本身是一个观众位置"而产生了新的复杂性——摄影机角度与运动会主动构建走位的含义，而不只是被动记录。

【学界常见解释】180°轴线规则是好莱坞经典剪辑（continuity editing）的基础约定，目的是让观众在剪切后不失去空间方位感。违反它往往需要有明确的叙事理由。

## Main Content

### 走位揭示的四个维度

| 走位要素 | 揭示的叙事信息 |
|---|---|
| 人物间距离 | 情感亲密度 / 心理距离 / 权力安全距离 |
| 人物相对高度 | 权力关系（站着的人 vs. 坐着的人） |
| 面对方向 | 开放（面向彼此）vs. 封闭（背对/侧对） |
| 空间位置（前/后/中） | 叙事主体地位；被排挤/被主导 |

### 180°轴线规则

**规则**：假设场景中有两个人物，在他们之间划一条假想线（轴线/action line）。所有摄影机位置必须在这条线的同一侧。这样在剪切时，两个人物的左右位置关系保持不变，观众不会迷失方向。

**图示**：
```
人物A ────────────────── 人物B
           轴线

[摄影机1]  [摄影机2]  [摄影机3]
↑ 所有摄影机在轴线同一侧
```

**越轴的意义**：【原文/作者观点】越轴会产生令观众感到迷失或不安的跳切，但有时导演刻意利用这一效果：
- 表达人物关系突然逆转
- 制造混乱或心理扰动
- 打破观众的"安全感"

### 越轴的经典案例

| 影片 | 走位 / 越轴处理 | 叙事功能 |
|---|---|---|
| 巴兹·鲁赫曼《罗密欧与朱丽叶》(Romeo + Juliet, 1996) | 电梯场景：可拆卸墙板 + 移动推轨车绕行人物 | 【原文/作者观点】通过移动摄影机绕行人物，而非固定180°规则，制造包裹感与双人世界的封闭感 |
| 《美国出生的中国人》(American Born Chinese, 2023) | 晚餐桌走位 | 【原文/作者观点】桌子两侧的走位安排暗示无法调和的世界观对立 |
| 《少数派报告》(Minority Report, 2002) | 空间安排强调意识形态对立 | 【原文/作者观点】人物的空间排列直接体现系统与个人的意识形态对抗 |

### 演员走位的舞台传统

【原文/作者观点】舞台传统中，演员常形成半圆弧形排列，自然朝向观众方向。在电影中，摄影机会**压缩场景中的真实距离**，因此导演在设计走位时需要考虑镜头焦距对空间感知的影响：长焦压缩距离（近景演员和远景道具看起来更近），广角拉伸距离（近景物体显得更大）。

## Method / How to Use

**走位分析的三层问题**（【我的理解】）：

**第一层：物理层**
- 人物在哪里？相互距离多远？谁在高处谁在低处？

**第二层：关系层**
- 这个位置关系暗示了什么权力/情感关系？人物是在靠近还是疏远彼此？

**第三层：主题层**
- 这个空间安排如何呼应影片的核心主题？（如"无法逾越的距离"或"强迫的亲密"）

## Debates & Limits

- 【学界常见解释】180°规则是约定而非自然法则——让-吕克·戈达尔（Jean-Luc Godard）等新浪潮导演频繁越轴，恰恰是对好莱坞"透明剪辑"幻觉的政治性拒绝。
- 【学界常见解释】"走位暗示关系"的解读存在过度诠释风险：有时演员的走位只是技术限制（光线/场地）的结果，而非叙事选择。分析时需结合导演意图的其他证据。
- 【你的推断】随着移动摄影（手持/无人机/长镜头）的普及，固定轴线的概念在当代电影中越来越灵活，但理解其规则仍是分析"为何感到混乱"的基础工具。

## Directing Takeaways

1. **走位是无声的剧本**：在开拍前，用走位草图规划每个人站在哪里，你会发现很多台词可以删掉——因为空间已经说了。
2. **距离即关系**：两个人物相距1米 vs. 3米，在银幕上传递的情感温度完全不同。
3. **高度是廉价的权力工具**：让一个人物站着，另一个坐着，不需要一句台词就建立了此刻的权力状态。
4. **越轴前想好理由**：如果你想越轴，确保它是叙事选择（表达混乱/逆转），而不是操作失误。

## Common Confusions

| 混淆点 | 正确区分 |
|---|---|
| 走位 vs. 场面调度 | 走位是场面调度的子集；场面调度还包括灯光、服装、布景等 |
| 180°规则 vs. POV规则 | 180°规则维持空间连贯性；POV规则（eyeline match）维持视线方向连贯性，二者相关但独立 |
| 越轴（crossing the line）vs. 跳切（jump cut） | 越轴是空间方位错误；跳切是同一对象时间跳跃；都可能令观众迷失，但原因不同 |
