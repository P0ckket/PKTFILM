# 画幅比例（Aspect Ratio）

## TL;DR

【学界常见解释】画幅比例（Aspect Ratio）是银幕宽度与高度之比，它在物理上决定了"观众看到多少世界"，在心理上塑造着封闭感、史诗感与亲密感。从 1.33:1 的早期学院标准到 2.76:1 的 Ultra Panavision，每种比例都携带着独特的情绪语法。艺考编导需要能够解释"为什么这部电影用这个画幅"。

---

## Definitions

**画幅比例（Aspect Ratio）**：画面宽度除以高度的比值，通常写作 W:H（如 1.85:1 表示宽度是高度的 1.85 倍）。

**变形镜头（Anamorphic Lens）**：拍摄时水平压缩画面（2×），放映时解压还原，产生 2.35-2.39:1 超宽银幕效果的光学系统。

**画幅遮挡（Letterbox）**：在非原生比例银幕上放映时，上下两条黑边，保留原始画幅的显示方式。

## Key Terms

| 术语 | 英文 | 含义 |
|---|---|---|
| 学院比例 | Academy Ratio | 1932年 AMPAS 确立的 1.37:1 标准 |
| 平银幕 | Flat | 1.85:1 美国标准宽银幕，不使用变形镜头 |
| 宽银幕/变形宽银幕 | Scope / Anamorphic | 2.35-2.39:1，使用变形镜头拍摄 |
| IMAX | IMAX | 1.43:1（胶片）或 1.90:1（数字）的大画幅放映标准 |
| 散景 | Bokeh | 焦外虚化的光斑形状（变形镜头产生椭圆形散景） |

| 1.37:1 | 学院标准（Academy Standard） | 1932年后好莱坞黄金时代标准 | 经典好莱坞黄金时代绝大部分影片 |

### 为什么画幅影响情绪

**窄幅（1.33:1 — 1.66:1）**
- 封闭感（confinement）、亲密感（intimacy）、"被包裹"的安全或压抑
- 常用于：心理剧、家庭剧、个人叙事、刻意制造年代感或不适感
- 案例：《灯塔》(*The Lighthouse*, 2019) 使用 1.19:1 近正方形，极度强化封闭与窒息

**标准宽（1.85:1）**
- 平衡感（balance）、舒适感（comfort）、最接近人眼水平视野
- 中性，不强迫情绪，适合大多数叙事类型
- 现代主流商业叙事片首选

**超宽（2.35:1 — 2.39:1）**
- 史诗感（epic）、空旷感（vastness）、水平线条强烈
- 视觉上强调横向运动与宽阔风景
- 适合：公路片、西部片、动作片、太空片

**变幅叙事技巧**
- 【学界常见解释】《布达佩斯大饭店》(*The Grand Budapest Hotel*, 2014) 用 1.33:1 / 1.85:1 / 2.35:1 三种画幅对应三个叙事时代，画幅本身成为时代标记
- 《独行侠》(*The Lone Ranger*, 2013) 开篇用 1.33:1 切换到 2.35:1，视觉上完成历史与当代的跨越

### 变形镜头（Anamorphic）的标志性特征

变形宽银幕（Anamorphic / Scope）因其独特光学特性产生视觉风格标识，成为"电影感（cinematic look）"的代名词之一：

| 特征 | 描述 | 视觉效果 |
|---|---|---|
| 椭圆形散景 | Oval Bokeh | 焦外光斑呈水平椭圆形而非圆形 |
| 水平光晕 | Horizontal Lens Flares | 强光源产生水平延伸的蓝/蓝紫色光束 |
| 边缘形变 | Edge Distortion | 画面两侧轻微桶形畸变，产生"包裹感" |
| 2× 画面压缩 | 2× Squeeze | 拍摄时压缩画面，放映时解压还原为 2.35-2.39:1 |

【你的推断】变形镜头的水平光晕已成为文化符号——J.J. Abrams 在《星际迷途》中刻意强化这一效果，代表对传统"电影感"的致敬与招唤。

## Method / How to Use

**艺考选择画幅的决策逻辑**：
1. 这个故事的核心情绪是**封闭（内向）还是开阔（外向）**？
2. 角色与环境的关系是**融合还是对抗**？
3. 是否需要用画幅本身来**标记时代或叙事层**？

```
封闭/压抑/内向 → 1.33:1 — 1.66:1
平衡/中性/通用 → 1.85:1
史诗/开阔/水平动作 → 2.35:1 — 2.39:1
超史诗/极端宽幅叙事 → 2.76:1
变幅叙事（时代/梦境） → 多比例混用
```

## Debates & Limits

- 【学界常见解释】"宽银幕 = 史诗感"是可以被刻意颠覆的——《灯塔》用近正方形画幅拍摄史诗级心理恐怖，正是反常规的形式主义操作。
- 【你的推断】流媒体时代大多数观众通过 16:9 屏幕观看，letterbox（上下黑边）对部分观众产生"小窗感"，这是导演选择超宽画幅时需要考虑的发行现实。
- 画幅不能单独定义情绪——与构图、光线、色调共同作用时才能发挥最大效力。

## Directing Takeaways

1. **画幅是情绪的物理形状**：选择画幅不是技术决策，是叙事立场声明。
2. **窄幅制造压力**：当你想让角色"无处可逃"时，把画幅压窄。
3. **宽幅给角色呼吸**：角色在宽银幕中显得更渺小，这可以是力量也可以是孤独。
4. **变幅叙事要慎用**：画幅切换会打破"视觉透明性"，必须有充分的叙事理由。

## Common Confusions

| 混淆点 | 正确区分 |
|---|---|
| 变形宽银幕 vs. 球面宽银幕 | 变形（Anamorphic）用特殊镜头压缩画面；球面（Spherical/Flat）直接拍摄，无压缩 |
| 1.85:1 vs. 2.39:1 的选择 | 1.85 更通用，2.39 更史诗；不是越宽越好 |
| IMAX 画幅 vs. 超宽银幕 | IMAX 是 1.43-1.90（接近方形），超宽银幕是 2.35-2.39（极宽），方向相反 |
| 画幅 vs. 分辨率 | 画幅是宽高比，分辨率是像素数量；二者独立 |
