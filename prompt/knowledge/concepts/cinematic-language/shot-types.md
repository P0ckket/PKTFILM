# 景别类型（Shot Types）

## TL;DR

【学界常见解释】景别是摄影机与被摄对象之间距离（以及焦距选择）在画面上形成的"画框截取范围"。从大远景到极端特写，每一个景别都携带不同的叙事功能与情感信号。艺考编导需要能在剧本/故事板中精确命名并说明景别的选择理由。

---

## Definitions

**景别（Shot Size / Shot Type）**：摄影机对被摄对象的截取范围，通常以人体为参照基准定义。不同景别不仅改变信息密度，也改变观众与人物的心理距离。

---

## Key Terms

| 术语（中文） | 定义 | 英文 |
|---|---|---|
| 大远景 | 宏观环境全貌，人物极小或不见 | Extreme Long Shot (ELS / XLS) |
| 全景 | 包含人物全身与背景，建立人物-环境关系 | Long Shot (LS) |
| 中景 | 腰部以上，兼顾动作与表情 | Medium Shot (MS) |
| 中近景 | 胸部以上，平衡面部表情与肢体语言 | Medium Close-Up (MCU) |
| 近景 / 特写 | 聚焦面部，强调情绪与心理状态 | Close-Up (CU) |
| 大特写 | 充满银幕的局部细节（眼睛/手/道具） | Extreme Close-Up (ECU / XCU) |
| 定场镜头 | 确定时间地点与空间关系，常为鸟瞰或航拍 | Establishing Shot |
| 主观镜头 | 呈现人物视点所见，可引发"男性凝视"等议题 | Point-of-View Shot (POV) |
| 反应镜头 | 紧接主观镜头，呈现人物对所见之情绪反应 | Reaction Shot |
| 跳切 | 同一被摄体的两个相似镜头直接相接，产生时间跳跃感 | Jump Cut |
| 淡入淡出 | 渐显/渐隐，常用于时间流逝或段落转换 | Fade In / Fade Out |

---

## Context & Problem

【学界常见解释】景别的命名规范并不完全统一——好莱坞工业标准、欧洲电影学派与中国艺考课程各有差异。中国教材常用"大远景/全景/中景/近景/特写/大特写"六级体系，英语文献有时细分出 Medium Long Shot（MLS）、Medium Close-Up（MCU）等过渡档位。

【原文/作者观点】艺考重点不在于死记档位数量，而在于能够说明"为何在此刻选用此景别"——即景别的**叙事功能**与**情感功能**。

## Main Content

### 景别功能速查表

| 景别 | 画面截取 | 主要功能 | 常见情绪 / 效果 |
|---|---|---|---|
| 大远景（ELS） | 宏观地景，人物微小 | 建立地理、权力、孤独感 | 渺小、壮阔、疏离 |
| 全景（LS） | 人物全身 + 背景 | 人物与环境的关系，行动可见 | 客观、陈述性 |
| 中景（MS） | 腰部以上 | 对话场景主力景别，动作+表情兼顾 | 中性、日常 |
| 中近景（MCU） | 胸部以上 | 新闻采访标准构图；情感升温时替代全景 | 亲近但不逼迫 |
| 近景/特写（CU） | 面部为主 | 强调内心状态，拉近观众共情距离 | 紧张、亲密、暴露 |
| 大特写（ECU） | 局部细节充满画幅 | 放大道具象征意义；极端情绪强化 | 窥探、震撼、威胁 |
| 定场镜头 | 宽画幅环境 | 段落开头定向，告知时间/地点/规模 | 客观、全知 |
| 主观镜头（POV） | 模拟角色视点 | 制造代入感；可带有意识形态含义（如男性凝视） | 沉浸、窥探 |
| 反应镜头 | 接收信息的角色面部 | 引导观众情绪解读；完成叙事节拍 | 情绪传导 |
| 跳切（Jump Cut） | 同一对象不同时刻 | 时间压缩；不安/躁动风格 | 焦虑、现代性 |
| 淡入淡出 | 渐变画面 | 章节感、时间流逝 | 平滑、诗意 |

### 经典案例

| 影片 | 景别选择 | 叙事 / 情感功能 |
|---|---|---|
| 《国王的演讲》(The King's Speech, 2010) | 麦克风极端特写（ECU） | 【原文/作者观点】放大焦虑对象；麦克风成为权力与恐惧的象征 |
| 《国王的演讲》 | 国王视角主观镜头（POV） | 观众进入焦虑主体位置，建立共情 |
| 《低俗小说》(Pulp Fiction, 1994) | 汽车后备箱低角度大远景 | 仰拍 + 全景组合，人物显得压迫性，观众处于弱势位置 |
| 《择日再死》(Die Another Day, 2002) | 望远镜双圆形画框 POV | 望远镜造型主观镜头，呈现 Jinx 的视角，强化窥视感 |

## Method / How to Use

**艺考编导景别选用口诀（【我的理解】）**：

1. **开场定位** → 用定场镜头或大远景建立时空坐标
2. **进入叙事** → 全景 / 中景呈现行动与关系
3. **情绪高潮** → 推近至特写或大特写，或用 POV + 反应镜头对
4. **结尾余味** → 拉远回大远景，形成"宇宙视角"的留白

**写景别的剧本格式示例**：
```
CLOSE-UP — 她的手指捏紧了那把钥匙。
```

## Debates & Limits

- 【学界常见解释】景别命名体系在不同传统中不统一（六档 vs. 更细分的七-九档），艺考答题时应标注自己使用的参照系。
- 【原文/作者观点】POV 镜头可能携带意识形态意涵（如"男性凝视"理论），需要在影片分析中有意识地辨认。
- 【你的推断】跳切的"时间加速"效果在习惯流媒体的观众那里已降低冲击力，当代使用跳切更多是风格标签而非叙事必要。

---

## Directing Takeaways

1. **景别即心理距离**：你把镜头推多近，就在邀请观众靠多近。慎用极端特写，因为观众进入后很难"出戏"。
2. **POV 需谨慎**：每个主观镜头都是一个立场声明，问自己"我是否真的想让观众完全站在这个人的位置？"
3. **跳切不是免费的**：跳切会打破"透明剪辑（invisible editing）"的幻觉，慎重选择使用时机。
4. **定场镜头是礼貌**：给观众时空定向，减少认知负担，然后再推进叙事。

## Common Confusions

| 混淆点 | 正确区分 |
|---|---|
| 全景 vs. 定场镜头 | 定场镜头是功能性概念（确立时空），不一定是全景；可以是大远景甚至俯拍 |
| 近景 vs. 特写 | 部分中文教材把二者合并称"特写"，英文区分 CU（面部）与 ECU（局部细节）；以上下文为准 |
| 主观镜头 vs. 越轴 | POV 是视点，越轴（180°规则违反）是空间连续性问题；二者独立 |
| 跳切 vs. 硬切 | 跳切特指**同一对象**两个相似构图的直接剪切；普通硬切（hard cut）是不同场景间的直接切换 |
