# 灯光技术（Lighting Techniques）

## TL;DR

【原文/作者观点】"电影灯光同时创造照明与阴影。光影的相互作用表达场景情绪、定义人物性格，并在银幕上塑造电影空间（Film lighting creates both illumination and shadow. The interplay between light and shadow expresses the mood of a scene, defines a character, and shapes the cinematic space on screen）。" 灯光不只是"让演员看清楚"的技术手段，而是叙事与情感语义的直接载体。

---

## Definitions

**灯光（Lighting）**：在电影拍摄中，对光源的类型、方向、强度、色温与阴影的系统性设计与控制。光不仅影响曝光，更是导演"说话"的视觉语言之一。

---

## Key Terms

| 术语（中文） | 定义 | 英文 |
|---|---|---|
| 硬光 | 直射光，阴影边缘清晰，对比强烈 | Hard Light |
| 软光 | 经过散射（半透明遮光板/反射）的柔和光，阴影渐变 | Soft Light |
| 三点布光 | 主光+补光+背光的标准布光体系 | Three-Point Lighting |
| 主光 | 三点中最亮的主要光源，45°斜前方 | Key Light |
| 补光 | 主光对面的辅助光，柔化主光阴影 | Fill Light |
| 背光 | 被摄体背后的光，勾勒轮廓，分离主体与背景 | Backlight / Rim Light |
| 高调布光 | 均匀明亮，阴影极少，高对比度色调 | High-Key Lighting |
| 低调布光 | 主要区域为暗部，强高光对比，阴影主导画面 | Low-Key Lighting |
| 逆光 | 光从被摄体背后照射，产生剪影效果 | Backlighting |
| 顶光 | 光从正上方照射，眼窝下陷，表情诡异 | Top Lighting |
| 顺光 | 光从正面照射，阴影少，表情暴露 | Front/Fill Lighting |
| 侧光 | 光从侧面照射，强调面部立体感与阴影 | Side Lighting |

## Main Content

### 布光类型功能矩阵

| 布光类型 | 光质 / 结构 | 阴影特点 | 主要情绪 / 类型偏好 |
|---|---|---|---|
| 硬光（Hard Light） | 直射，无散射 | 边缘锐利，浓重阴影 | 威胁感、恐怖、权威；适合犯罪/惊悚 |
| 软光（Soft Light） | 散射或反射，漫射 | 渐变柔和，几乎无阴影 | 浪漫、温柔、梦幻；适合爱情场景 |
| 高调（High-Key） | 均匀充足照明 | 极少阴影，整体明亮 | 欢乐、喜剧、广告感；不引起注意 |
| 低调（Low-Key） | 无补光，高对比 | 大面积阴影主导画面 | 黑色电影（film noir）、恐怖、道德模糊 |
| 逆光/剪影（Backlight） | 背后光源 | 主体变剪影，轮廓光 | 英雄登场 or 威胁未知；神秘感 |
| 顶光（Top Light） | 正上方光源 | 眼窝/鼻下浓阴影 | 正常强度=魅力感；过强=威胁/诡异 |
| 侧光（Side Light） | 侧面光源 | 面部一半亮一半暗 | 人物内心矛盾；双重性；道德模糊 |

### 三点布光体系详解

```
       [背光 Backlight]
            ↓
      ┌─ 主体 ─┐
  [主光 Key]    [补光 Fill]
  45°斜前方     对面柔化阴影
```

【原文/作者观点】三点布光原则：
- **主光（Key Light）**：最强，从正面45°角打来，定义主要光影
- **补光（Fill Light）**：主光对面，强度较低，柔化主光产生的硬阴影
- **背光（Backlight / Rim Light）**：人物后方，用于在背景前勾勒轮廓，使主体"立起来"
- 主光与补光通常以45°角设置

### 经典案例

| 影片 | 布光选择 | 叙事 / 情绪功能 |
|---|---|---|
| 《黑客帝国》(The Matrix, 1999) | 墨菲斯（Morpheus）：硬侧光+顶光 | 【原文/作者观点】营造威严与权威感，强化其作为导师的地位 |
| 《泰坦尼克号》(Titanic, 1997) | 罗丝（Rose）：温暖软光，二维感 | 【原文/作者观点】柔光使她看起来如画中人，服务于爱情叙事的浪漫化 |
| 《天使爱美丽》(Amélie, 2001) | 经典三点布光 | 【原文/作者观点】标准三点布光体现法式温暖风格，光线本身不喧宾夺主 |
| 《黑豹女王》(The Woman King, 2022) | 均匀高亮布光 | 【原文/作者观点】针对深色皮肤演员刻意优化光量分布，体现布光的社会正义意识 |
| 《和莎莫的500天》(500 Days of Summer, 2009) | 同一场景两种布光 | 【原文/作者观点】同地点用不同布光方案暗示同一段记忆的主观温度变化（甜蜜/失落） |
| 《银翼杀手》(Blade Runner, 1982) | 布拉德伯里大楼低调布光 | 【学界常见解释】强高对比低调布光是赛博朋克/黑色电影的视觉标志，道德灰色世界的光影隐喻 |

### 布光与肤色的社会议题

【原文/作者观点】深色皮肤会吸收更多光线，需要比浅色皮肤更大的光量投入才能获得同等的面部细节表现。这在电影工业历史上长期被忽视，导致深肤色演员在银幕上"消失"或显得扁平。当代布光师在多元化叙事中必须有意识地处理这一问题。

## Method / How to Use

**布光决策的三问**（【我的理解】）：

1. **这个角色/场景的情绪基调是什么？** → 明亮积极（高调）or 阴暗复杂（低调）
2. **光从哪里来？** → 方向决定阴影位置，阴影决定面部读法
3. **有没有需要刻意强调或隐藏的东西？** → 背光可以制造神秘；顶光可以让眼睛消失在阴影里

**艺考分析布光的答题框架**：
```
布光类型 → 光线方向与质感 → 阴影效果 → 情绪/叙事功能 → 电影案例佐证
```

## Debates & Limits

- 【学界常见解释】"低调=恶"、"高调=善"是过度简化：《美国丽人》（American Beauty）用明亮高调光拍摄压抑的中产叙事，反而制造讽刺距离感。
- 【学界常见解释】自然光（natural light）布光风格（如特伦斯·马利克作品）拒绝人工三点体系，追求真实感——这本身也是一种布光哲学选择。
- 【你的推断】随着 LED 和数字摄影机的普及，布光的"规则"已被大幅拓展；但对艺考学生而言，理解经典体系（三点布光、高低调对立）仍是分析入门的必要基础。

## Directing Takeaways

1. **光即态度**：每一个布光方案都是导演对人物/场景的情绪判断，不存在"中性布光"。
2. **低调不等于"黑暗影片"**：低调布光是一个技术选择，控制的是对比度，而非整体亮度。
3. **三点布光是起点不是终点**：三点布光给你一个可控的基础，然后打破某一个点（去掉补光？加侧光？）来制造你需要的情绪偏移。
4. **注意演员肤色**：在拍摄深色皮肤演员时，主动增加主光量或调整色温，确保面部细节不丢失。

## Common Confusions

| 混淆点 | 正确区分 |
|---|---|
| 高调（High-Key）vs. 曝光过度 | 高调是低对比度的均匀明亮；曝光过度是高光区域失去细节 |
| 低调（Low-Key）vs. 画面太暗 | 低调有精确的高光区域；"画面太暗"是控制失败，无高光锚点 |
| 背光（Backlight）vs. 逆光（Contre-Jour） | 术语基本同义；Backlight更强调三点布光中的功能性背光，逆光更常指剪影效果 |
| 软光 vs. 低光 | 软光描述光质（散射度），与亮度无关；低光是关于光量多少 |
