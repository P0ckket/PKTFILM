# 道具象征(Prop Symbolism)

## TL;DR
**道具(Prop)** 在电影里有两种功能:**工具性**(执行情节的物件)和**象征性**(传达主题/人性的意向)。原笔记 p.1、p.6、p.11、p.36 散落记录了多组道具条目——**紧张道具、车的意向、面具的作用、产生悬念的道具设置、人物内心转变的标志**等。本卡把它们整合成**按功能分类的速查表**,并给出"道具设计四维度"的通用法则。

## Definitions

- **道具(Prop)**:电影场景中除布景和演员外的可移动物件。
- **道具象征**:道具承载的超出物件本身的意义(主题、人物、情绪)。
- **紧张道具**:其存在本身就制造紧张情绪的道具(钉子、炸弹、计时器)。
- **悬念道具**:通过其"来源未知 / 去向未知"产生悬念的道具(匿名信、神秘礼物)。
- **仪式道具**:通过其在仪式化场景中的出场产生意义的道具(戒指、烟火)。

## Key terms

### 道具设计四维度
| 维度 | 含义 |
|---|---|
| 工具维度 | 道具在情节里做什么? |
| 象征维度 | 道具传达什么抽象含义? |
| 时间维度 | 道具在故事里出现几次?前后是否有变化? |
| 关系维度 | 道具与主角的关系(拥有/失去/寻找/逃避)? |

## 道具速查表(按功能组织)

### 一、紧张道具(p.1)
**目的**:让观众仅凭道具出现就开始紧张。
| 道具 | 紧张来源 |
|---|---|
| 钉子 | 伤害威胁 |
| 硫酸 | 不可逆伤害 |
| 炸弹 + 计时器 | 时间压力 |
| 电线 + 破皮 | 触电风险 |

**使用原则**:紧张道具必须**先露面、后使用**——露面时不用,使用时已有期待,这就是悬念。

### 二、悬念道具(p.36)
**目的**:通过"来源/去向不明"制造未知感。
| 道具 | 悬念来源 |
|---|---|
| 匿名信件 | 发件人不明 |
| 陌生电话 | 通话人不明 |
| 神秘礼物 | 赠予者不明 |
| 失踪的物品 | 去向不明 |

**使用原则**:悬念道具必须**带着问号出场**——观众看到它就想"这是哪来的?谁给的?去哪了?"

### 三、意向道具(p.11)——车的意向
p.11 给出了单一道具的**多维度意向包**,以"车"为例:
| 车的意向 | 含义 |
|---|---|
| 时间机器 | 载主角跨越时间/空间(公路片) |
| 奢侈与阶级 | 车的品牌 = 人物社会位置 |
| 自由与掌控力 | 握住方向盘 = 掌握人生 |
| 独立与孤独 | 驾驶舱 = 一个人的空间 |

**使用原则**:同一道具可承载多种意向,具体选哪一种由剧情定。

### 四、身份道具(p.11)——面具
| 面具的作用 | 含义 |
|---|---|
| 线索 | 破案道具 |
| 隐藏身份 | 不让他人知道你是谁 |
| 暴力美学 | 风格化符号(《小丑》《V 字仇杀队》) |
| 恐怖氛围 | 心理威胁 |

### 五、仪式道具(p.12)
**仪式感场景的道具配置**:
- 坐在车顶
- 跪地行礼
- 交换戒指
- 烟火表演
- (p.36 补充)车灯与舞台灯的融合、特殊时间点(生日、毕业典礼)、独特的服装和化妆

### 六、人物内心转变的标志性道具(p.36)
**目的**:通过道具变化展示主角的内在转变,无需台词。
| 转变标志 | 道具化表达 |
|---|---|
| 着装改变 | 发型、化妆 |
| 兴趣转变 | 从喜欢安静变为热衷冒险的道具 |
| 人际变化 | 从依赖到独立的物件使用方式 |
| 象征性物品的使用 | 抛弃旧的或使用新的 |

### 七、人物塑造辅助道具(p.6)
p.6 "课前热身"里给出的两个道具分析:
**齿轮**
- 关键零部件 → 正常运转 → 真相揭露
- 整体与个体 → 团结合作、配合天衣无缝
- 按部就班 → 千篇一律、枯燥乏味的生活
- 工业化象征

**手套**
- 保护
- 隐藏、遮掩
- 人物符号化象征
- 手套颜色/样式变换 → 人物立场的变换
- 掌控 → 地位
- 揭示特殊职业身份(杀手)
- 阻碍、障碍 → 限制手部活动

## Context & problem
**道具象征解决的问题是**:电影要讲主题,但不能直白——道具是"省力的主题容器"。主角不说"我想自由",但他把鸟笼的门打开。主角不说"我不再依赖母亲",但他把母亲送的项链摘下。**一个道具顶三句台词**。

原笔记的价值在于:**把道具按功能分类**,让考生能在写作时"**按需选物**"——不是随意设计道具,而是问自己"这场戏需要紧张道具 / 悬念道具 / 意向道具,然后查表选"。

## Claims

1. **同一道具可承载多重意向**(【原文】p.11 "车")
   - 车可以同时是时间机器、阶级符号、自由、孤独
   - **选择哪一层取决于剧情语境**和道具的出现方式。

2. **道具设计必须结合"时间维度"**(【原文】p.36)
   - 【原文】"象征性物品的使用:抛弃旧的或者使用新的"
   - 一个**没有变化**的道具,只是布景;一个**前后变化**的道具,才是象征。

3. **紧张道具的作用机制是"悬念前置"**(【我的理解】)
   - 钉子出现 → 观众想"谁会踩到?"
   - 炸弹出现 → 观众想"什么时候爆?"
   - 这是 Hitchcock 说的"观众知道炸弹,演员不知道 = 悬念"。

4. **身份道具是"符号化"的捷径**(【原文】p.6 手套)
   - 【原文】"手套:人物符号化象征"
   - 一个反复出现的物件,会让观众自动把它和人物绑定。
   - **最经典的符号化道具**:维托·柯里昂的红玫瑰、邦德的马提尼、蝙蝠侠的斗篷。

5. **仪式道具的功能是"放大情绪"**(【原文】p.12, p.36)
   - 仪式感场景的道具(戒指/烟火/车灯)不是情节必需,但能**让情绪显得更重**。
   - 【我的理解】这对应于剧作中的 "moments of grace"——故事需要慢下来,让情绪沉淀。

6. **道具变化是内心变化的外显**(【原文】p.36)
   - 这是好剧作的铁律:**内在变化必须有外在标志**。
   - 道具是最稳定的外在标志——比表情、对白都可靠。

## Method / How to use

### 道具设计 4 问
写完剧本后,针对每个重要道具问:
1. 工具维度:它在情节里做什么?(不能只是布景)
2. 象征维度:它传达什么抽象含义?
3. 时间维度:它出现几次?前后变了什么?
4. 关系维度:它和主角的关系经历了什么变化?

### 艺考编讲故事的"一道具三用"法
选一个核心道具,让它在故事里**至少出现 3 次**:
- **第一次(开场事件)**:道具首次亮相,工具功能
- **第二次(转折事件)**:道具状态改变或意义翻转
- **第三次(高潮/结局)**:道具作为主题锚点出现,完成象征闭环

eg. 一块怀表:
- 第一次:主角每天上发条(仪式感,紧迫感)
- 第二次:怀表在危机事件里停了(时间停止 = 心死了)
- 第三次:结局中主角把怀表送给配角(放下时间 = 放下执念)

### 紧张道具的"不使用"技巧
紧张道具最大的技巧是:**出现了但没用到**。
- 手枪放在桌上一整场戏但没开火 = 紧张持续上升
- 炸弹装好了但没爆 = 悬念延长
- **使用就是释放,不使用就是蓄积**。

## Debates & limits
- **道具 ≠ 全部靠设计**。有些杰作的道具来自演员即兴(eg.《教父》开头那只猫是现场抓来的)——过度"设计"会让道具显得僵硬。
- **"道具必须有象征"是教条**。现代电影里很多道具只是工具,没有象征——强行要求每个道具都承载意义会让剧作累赘。
- 【我的理解】考场上反而要"强行象征化"——因为考官看不到拍摄,只能看文字,道具必须写出意图才拿分。
- 【你的推断】p.36 的"人物内心转变标志"实际上是一个独立的概念——**通过道具表达弧光**,不只是象征。这个用法比"紧张道具/悬念道具"更高级,应单独出一张练习卡。

## Directing takeaways
1. **每场重要戏都要问:这场戏的关键道具是什么?** 如果回答不出——戏的视觉层面是空的。
2. **不要滥用象征道具**。全片重点道具最多 3 个,太多观众记不住。
3. **紧张道具的使用要"慢"**——让它停留在画面里,观众的焦虑会自己升级。
4. **内心转变必须有道具标志**。写危机戏时,让主角拿起/放下某件物件——不是替代台词,是让内心外显。
5. **道具的"出场仪式"很重要**——第一次出现的方式决定观众如何看待它。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 道具 = 布景 | 布景是不可移动的环境,道具是可移动的物件。更本质的:道具与人物**有互动**,布景通常没有。 |
| 道具象征 = 符号 | 象征是"物代表某个抽象概念",符号是"物代表另一个具体的事物"。象征更深,符号更浅。 |
| 紧张道具 = 危险物品 | 紧张道具是"让观众觉得危险的物品"——纸箱也可以是紧张道具(如《七宗罪》结尾)。 |
| 仪式道具 = 宗教用品 | 仪式道具是"出现在仪式化场景里的道具",包括戒指、烟火、蛋糕、蜡烛等——不一定宗教。 |

## Open questions
- **更完整的"道具功能分类"**——p.1/6/11/36 给了 5-6 种功能,是否还有其他类别?(eg. 情感道具、记忆道具)
- **东方电影里的道具传统**(茶具、扇子、玉器)是否有独立的意向系统?
- **数字时代的新道具**(手机、外卖、监控)的意向规律是什么?——原笔记完全没涉及,这是时代空白。
