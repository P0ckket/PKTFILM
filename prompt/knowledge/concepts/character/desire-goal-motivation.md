# 欲望 / 目标 / 动机(Desire / Goal / Motivation)

## TL;DR
原笔记里关于"人物为什么动起来"给出了两条并行的线:
- **p.33 的三要素**:身份 / 欲望 / 动作——**静态结构**
- **p.48 的驱动链**:**缺点 → 困境 → 欲望 → 目标**——**动态流程**

前者回答"人物是什么",后者回答"人物怎么动"。**欲望是抽象的(想变富),目标是具体的(去银行抢劫)**。欲望分两档:**原始欲望**(名利权情)和**高级欲望**(奉献、牺牲、正义)。动机则是把欲望催化成行动的外部原因。

## Definitions

- **欲望(Desire)**:主角想要达成的**抽象状态**。eg. "想变富""想被爱""想自由"。
- **目标(Goal)**:把欲望落到地面的**具体操作**。eg. "去抢劫银行""结婚""离开这个国家"。
- **动机(Motivation)**:触发欲望落到目标的**外部/内部原因**。eg. "母亲生病需要钱""收到一封信"。
- **缺点(Flaw)**:人物内在的不完美——在动态链的起点,是主角"还没开始成长"时的底色。
- **困境(Predicament)**:主角的缺点带来的具体麻烦(见 [[../narrative/inciting-incident]])。

## Key terms

### 静态结构(p.33)
**身份 + 欲望 + 动作 = 人物塑造三要素**

| 要素 | 对应概念卡 |
|---|---|
| 身份 | [[./character-trinity]] 三位一体 |
| 欲望 | 本卡 |
| 动作 | [[./action-vs-movement]] 行动不是动作(待建) |

### 动态链条(p.48)
**缺点 → 困境 → 欲望 → 目标 → 行动 → 结局**

- 起点:一个有缺点的主角,在生活中遭遇困境
- 困境激发欲望(抽象愿望)
- 欲望落到具体目标
- 主角为目标在不同空间做出不同行动
- 最终目标实现或不实现(而实现过程中,缺点被修正)

【原文 p.48】"缺点====>困境=======>欲望=========>目标"
【原文 p.48】"起点:一个人物想去达到某个目标 → 终点:人物是否达到目标"

## Context & problem
**这个区分要解决的问题是**:学生写人物时经常把"欲望"和"目标"混为一谈,结果主角"只有一个想法没有动作"(想变富但整部片什么都没做),或者"只有动作没有想法"(一直在做事但不知道为什么)。

p.48 的关键洞见是:**欲望必须落到目标才能写**。"我想变富"是无法拍摄的,"我要去抢劫银行"才能拍。把欲望和目标拆开,让两者之间形成**抽象到具体**的映射关系——这是写作时的第一道过滤器。

同时 p.48 还给出"目标 vs 实现方式"的弹性:同一个欲望可以有多种目标。eg. 想变富可以 (a) 努力工作提升薪资 (b) 抢劫银行 (c) 天天买彩票等中奖。**选哪个目标 = 透露人物性格**。

## Claims

1. **欲望是抽象的,目标是具体的**(【原文】p.48)
   - 欲望:我想变得富有
   - 目标:我要努力工作提升薪资 / 我要去抢劫银行 / 天天买彩票等中奖
   - 【我的理解】选哪个目标,等同于"主角是什么样的人"——勤奋的人选努力工作,冲动的人选抢劫,懒惰的人选彩票。

2. **原始欲望的四维**:名、利、权、情(【原文】p.41)
   - 名:名誉、声望
   - 利:金钱、物质
   - 权:权力、地位
   - 情:感情、关系
   - 这是人物动机的"底层操作系统"。

3. **高级欲望**:奉献、牺牲、正义(【原文】p.41)
   - 高级欲望通常出现在**人物弧光后半**——主角从追求原始欲望,转变为追求高级欲望。
   - 【我的理解】这对应于"人物弧光"理论中的 "want → need" 跳跃。原始欲望是 want(想要的东西),高级欲望是 need(真正需要的东西)。

4. **缺点是起点,缺点的修正是终点**(【原文】p.48)
   - 【原文】"改变自己,缺点会在故事中被修正,困境会被解决"
   - 【原文】"缺点→困境→欲望→目标"是一条单向推进链
   - 如果结尾时缺点没有修正,要么主角是悲剧人物(《了不起的盖茨比》),要么故事有问题

5. **动机是激励事件的催化核心**(见 [[../narrative/inciting-incident]])
   - 动机 ≠ 目标,动机催生目标(【原文】p.21)
   - 动机 = 外部事件 + 人物内在欲望相互作用的结果

6. **动机分三档,催化难度递增**(【原文】p.6-7,见 [[../../theory/three-act-structure]] )
   - 生存 / 职业(最易催化)
   - 人物关系 / 情感线(需铺垫)
   - 发展 / 理想(最难,需多次催化)

## Method / How to use

### 写作时的四步拆解
1. **写出主角的缺点**(一个具体的性格或行为问题)
2. **让缺点带来具体困境**(缺点如何造成麻烦)
3. **困境催生抽象欲望**(主角想摆脱什么、得到什么)
4. **把欲望翻译成具体目标**(可拍摄的动作目的)

eg. 《百元之恋》(p.48-49):
- 缺点:懒惰、懦弱、自卑、执行力差
- 困境:不能接纳自己
- 欲望:接纳自己、改变自己(抽象)
- 目标:打完一场正式的拳赛(具体)

### 欲望升级练习
把一个原始欲望升级为高级欲望:
- 原始:我想赢(名) → 高级:我想证明有人像我这样的人值得被尊重
- 原始:我想要钱(利) → 高级:我想让家人过得好
- 原始:我想要她(情) → 高级:我想成为值得被她爱的人
- → 这种升级通常发生在**中点**或**灵魂黑夜**节拍之后

### 检查表
| 检查项 | 通过条件 |
|---|---|
| 欲望是否抽象? | 能用"想要…"开头说 |
| 目标是否具体? | 能写成一个动作/地点 |
| 目标能否被摄影机拍到? | 能 = 通过 |
| 缺点是否会阻碍目标? | 是 = 通过 |
| 结尾缺点是否改变? | 是(除非是悲剧) = 通过 |

## Debates & limits
- **want vs need 的映射**。【学界常见解释】在西方剧作理论中,Dara Marks 的 *Inside Story*(2009)和 Michael Hauge 的 *Writing Screenplays That Sell* 都提出 **want vs need** 的对立——人物想要的(want)和真正需要的(need)不一样,故事是从前者到后者的转变。原笔记的"原始欲望 / 高级欲望"与这一框架高度同构。
- **"缺点必须修正"的例外**。有些杰作的主角缺点**没有修正**(《革命之路》的 Frank)或反而**恶化**(《华尔街之狼》的 Jordan)。这些是"反弧光"或"负向弧光"——结构上仍可用"缺点→困境→欲望→目标"描述,但结果反向。
- 【我的理解】"欲望 / 目标"的严格区分是写作训练初期的工具,写到后期很多人会把它合并——但对艺考考生,严格分开最保险。
- 【你的推断】p.41 的"名利权情"四维和 p.48 的"缺点→困境→欲望→目标"链条其实是**两个不同老师的教案在同一本笔记里共存**的痕迹。前者更"价值论",后者更"结构论"。可以互补使用。

## Directing takeaways
1. **剧本第一页必须写清目标**。欲望可以含糊,目标不能含糊——否则观众不知道看什么。
2. **目标越具体越好**。"找到幸福"是烂目标,"在女友生日前把戒指送到她手上"是好目标。
3. **把欲望藏在潜台词,把目标放在台词或动作**。主角可以明确说"我要打完这场拳赛"(目标),但"我想接纳自己"这种话是用事情说,不是用话说。
4. **缺点决定目标的选择**。同一个欲望有多个可能目标,主角选哪个取决于他的缺点——这个选择本身就是角色塑造。
5. **高级欲望的出现不能突兀**。从原始到高级的跳跃需要**代价**——通常是灵魂黑夜里主角失去了某样东西。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 欲望 = 动机 | 欲望是"想要什么",动机是"为什么现在想要"。动机是欲望的**时间触发器**。 |
| 目标 = 欲望 | 目标是欲望的**可操作版本**。同一个欲望可以对应多个目标,选哪个体现性格。 |
| 动机 = 原因 | 动机是**让欲望在特定时间激活**的原因,不是欲望本身的来源。 |
| 高级欲望 = 好,原始欲望 = 差 | 两者没有优劣之分,高级欲望往往在人物弧光**后半**出现。前半必须有原始欲望,否则主角太"圣人"。 |
| 缺点 = 弱点 | 缺点是人物性格上的"不完美",弱点是被外部利用的薄弱点。不完全相同。 |

## Open questions
- 【待核实】"名利权情"四维的出处——是老师的归纳还是传统表达?
- **"高级欲望"是否必须出现?**——有些好故事主角始终只追求原始欲望(喜剧、动作片)。这种情况是否意味着"没有弧光"就一定差?
- **目标的"层级"是否需要进一步拆分?**——有学者区分 "objective / super-objective"(短期目标 / 总目标),p.48 未涉及,待补。
