# 救猫咪时刻(Save the Cat Moment)

## TL;DR
**救猫咪**是 Blake Snyder 2005 年在 *Save the Cat!* 里提出的概念——指主角出场后做的一件**无功利的美好小事**,功能是让观众"喜欢上他"。这件事不需要大,不需要勇敢,不需要聪明,只需要**无偿**:对孩子好、帮助下属、捡回丢失的物件、与动物互动。没有这个时刻,观众就不会主动站在主角一边,后面无论发生什么困难,观众都不会真的揪心。

## Definitions

- **救猫咪(Save the Cat)**:主角在故事早期(通常是开场事件 / 铺垫节拍内)做的一件无功利的美好小事,用来锁定观众的好感。
- **无功利**:主角做这件事**得不到回报**——不是在完成任务,不是为了赢得某人,不是有目的地讨好。
- **可爱时刻 vs 救猫咪时刻**:可爱是[[character-trinity|三位一体]]的**一个维度**,救猫咪是可爱的**一个具体操作**——可爱更宽,救猫咪更窄。

## Key terms

| 维度 | Snyder 原书 | 原笔记 p.1 + p.33 |
|---|---|---|
| 定义 | 主角做一件让观众喜欢他的事 | 让主角做一件美好的小事 |
| 时机 | 故事早期 | 开场事件 / 主角入场 |
| 强度 | 可大可小,但必须无功利 | "小事",适用于强势角色 |

### 原笔记列出的救猫咪操作(p.1)
- 与孩子互动
- 对待下属或员工的善意
- 找回失物
- (延伸)关心家人(但这属于"可爱的道德层面",更宽于救猫咪)

## Context & problem
Snyder 在原书里讲了一个故事:他看了无数**结构完美但观众讨厌主角**的剧本——所有情节点都对,主角也确实是主角,但你就是不关心他死活。Snyder 发现这些剧本都缺一个东西:**主角出场后,没有让你有一秒钟喜欢他**。

从此"救猫咪"成了剧作圈的流行术语。它回答的问题是:**观众凭什么要跟这个陌生人走 120 分钟?**

答:**因为你给他看了这个陌生人身上最好的一瞬间**。

原笔记 p.33 特别指出"**救猫咪适用于强势角色**"——这是对 Snyder 原意的重要补充。强势角色(医生、律师、富商、老板、杀手)在观众眼里天然带"反派潜质",如果不做救猫咪,观众会默认他是坏人。而弱势角色(农民工、学生、外卖员)天然可怜,救猫咪的优先级可以往后放。

## Claims

1. **救猫咪必须无功利**(【学界常见解释 / Snyder 原书】)
   - 如果主角救猫是因为"猫主人是美女",那不是救猫咪,是撩妹。
   - 如果主角救猫是因为"任务需要他救",那不是救猫咪,是职业操守。
   - 无功利 = 没人看也会做。

2. **救猫咪时间点要早**。(【原文 + Snyder 原书】)
   - Snyder 原书说"在前 10 页以内"(对应开场画面 + 铺垫节拍之间)
   - 艺考编讲故事:**在开场事件的 400 字以内**(同样是故事的前 1/5)
   - 过晚 = 观众已经在怀疑你的主角是不是好人

3. **救猫咪不必"惊天动地"**(【原文】p.33)
   - 【原文】"让主角做一件美好的小事"
   - 越**不引人注目**的善意,越显得真——炫目的善行像作秀,微小的善行像人性。

4. **强势角色尤其需要**(【原文】p.33)
   - 反向应用:当你的主角是医生、律师、警察等"自带权威"的身份时,救猫咪的优先级最高。
   - 【我的理解】这也解释了为什么经典片里的强势主角常常配一条"关心老母亲"或"对小动物好"的支线(eg.《教父》维多·柯里昂喂猫)。

5. **救猫咪不是"伪善",关键是动机的纯粹**
   - 【你的推断】这里容易被误解成"公式化做好事"——但 Snyder 原书明确反对"为了救猫咪而救猫咪"。真正的技巧是:救猫咪时刻必须**与主角的真实性格一致**,让它看起来是"顺便发生的"。

6. **救猫咪与"反救猫咪"**(【学界常见解释】)
   - Snyder 的续作 *Save the Cat! Goes to the Movies* 提出**反救猫咪(Kick the Cat)**——让反派主角做一件小恶事,锁定观众"恨"他的情绪。逻辑镜像:无论是爱还是恨,关键是**观众的情感锚点必须早早锁住**。

## Method / How to use

### 救猫咪设计 checklist
在写完开场事件后问自己:
- [ ] 主角做了一件小的善事/好行为吗?
- [ ] 这件事是**无偿**的吗?(没人看、没奖励、没目的)
- [ ] 这件事发生在**开场事件的 400 字内**吗?
- [ ] 这件事**符合主角的性格**吗?(不是硬塞)
- [ ] 主角是**强势角色**吗?(如果是,必须有;如果弱势,可以晚)

### 艺考三段式救猫咪模板
1. **展示主角的困境**(可怜)
2. **主角顺手做一件小的善事**(救猫咪 = 可爱)
3. **激励事件降临**(打破平衡)

eg.
> 外卖员(可怜,职业卑微)在雨中送餐,看到一只被雨淋湿的流浪猫躲在纸箱里。他把自己的雨披脱下来盖在纸箱上,自己冒雨继续送。(救猫咪 = 无功利的小善事)转角一辆车撞上他。(激励事件)

### 救猫咪素材清单(艺考可用)
- 让座给老人但不声张
- 帮孩子捡掉在地上的玩具
- 给流浪动物喂食
- 替同事背锅
- 归还捡到的钱包
- 路上捡起垃圾
- 不计较被踩脏的鞋
- 对服务员说谢谢

### 反例(避免)
- 做好事但被人看见并被夸奖(有功利:社会认可)
- 救了一个后面会帮主角的人(有功利:情节工具)
- 救的对象是主角亲友(变成"尽义务",不再是救猫咪)
- 救完就走不留痕迹——但**镜头特意聚焦**,让观众知道是"给观众看的"(这是 Snyder 原书说的"保留隐形观察者视角")

## Debates & limits
- **公式化批评**。【学界常见解释】在 Snyder 这本书流行后,好莱坞一度出现"人人都救猫咪"的现象,观众也学会了识别这种套路。Peter Dochtery 等人批评 "Save the Cat" 让好莱坞变成流水线。Snyder 原书其实已经预判了这点,强调**救猫咪要与角色本性一致**,否则变成"塑料微笑"。
- **反英雄的救猫咪不适用**——如《出租车司机》的崔维斯、《小丑》里的亚瑟·弗莱克,这些角色的观众共情机制不是"救猫咪",而是**共享主角的痛苦视角**。救猫咪是好莱坞商业片的主流手段,但不是唯一手段。
- 【我的理解】在艺考场景下,救猫咪几乎是必需品,因为考官不会花时间琢磨反英雄的同情机制。
- 【你的推断】中国电影传统里的"义举"(水浒好汉式的仗义)其实也是救猫咪的变体,但它往往**有对象、有回报、带江湖道德色彩**——严格说不是 Snyder 意义上的"无功利"。两者可以互鉴。

## Directing takeaways
1. **每个强势角色出场 5 分钟内必须救一次猫**,哪怕只是抚摸一下狗。
2. **救猫咪最好在"做别的事的途中"发生**,不要独立成戏——独立成戏会显得刻意。
3. **对照组技巧**:在救猫咪的同一场戏里,让另一个人做**不救猫咪**的反应(冷漠路过、抱怨),可以让主角的善意更醒目。
4. **救猫咪和可怜可以同场发生**。eg. 一个被欺负的主角反手去帮被更多人欺负的人——双钩子同时生效。
5. **不要在台词里解释救猫咪**。主角不能说"我就是见不得这种事"——这句话一出,救猫咪就失效了。让行动自己说话。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 救猫咪 = 救真的猫 | 这是泛称。只要是"无功利的美好小事",什么都可以。 |
| 救猫咪 = 可爱的全部 | 救猫咪是可爱的**一种具体操作**,可爱还有道德/性格/外形/技能四层(见 [[./character-trinity]])。 |
| 救猫咪 = 主角是好人 | 主角可以是灰色人物,救猫咪只证明他"还有人性的一面"。《小丑》里就没用救猫咪,但大量反派用救猫咪保住观众好感。 |
| 反派也需要救猫咪 | 反派用的是**反救猫咪 (Kick the Cat)**——做一件小恶事,锁定观众"反感"的情绪。 |
| 救猫咪=开场主角出场戏 | 主角出场戏是时间位置,救猫咪是戏的**内容**。两者常常重合,但概念不同。 |

## Open questions
- 【待核实】Snyder 原书对"救猫咪时刻"的**最小强度**(可以多小?)有明确界定吗?
- **东方叙事里的对应物是什么?**——孔子说"见义勇为"——是救猫咪吗?还是更强的"义举"?
- **"救猫咪"在纪录片和非虚构作品里是否适用?**——非虚构人物是真人,救猫咪是设计的;两者如何结合?
