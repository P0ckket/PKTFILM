# 人物三位一体:可怜 / 可信 / 可爱

## TL;DR
**可怜 + 可信 + 可爱** 是艺考编讲故事里人物身份设计的"三角模板"——给主角同时赋予三种共情标签:**可怜**(让观众同情)、**可信**(让观众觉得真实)、**可爱**(让观众喜欢)。原笔记 p.1 给出的优先级是"**可爱 > 可信 > 可怜**",而 p.33 把它纳入"人物塑造三要素(身份 / 欲望 / 动作)"的**身份**一栏。它不是描写三条单独的性格,而是**同一个人身上同时存在的三层钩子**,缺一个观众就会松手。

## Definitions

- **可怜(Pitiable)**:让观众产生同情的条件——职业卑微、被欺负、不幸遭遇、缺陷、贫困。是让观众**愿意站到主角一边**的首要钩子。
- **可信(Credible)**:让观众觉得人物"是真人"的条件——缺陷、坏习惯、脾气、不完美的道德。是让人物**不像神**的保险。
- **可爱(Lovable)**:让观众主动**想看这个人过得好**的条件——做一件美好的小事、帮助弱者、关心家人、展现善意。Blake Snyder 的"**救猫咪(Save the Cat)**"时刻就是"可爱"的具体操作。
- **原始欲望 / 高级欲望**:见 [[./desire-goal-motivation]]。可怜/可信/可爱解决"观众为什么跟这个人",欲望解决"这个人为什么动起来"。

## Key terms

| 层 | 操作 | 原笔记例子 |
|---|---|---|
| 可怜 | 职业卑微的人、穷、送外卖 | p.1 |
| 可怜 | 无缘无故被欺负、社会环境不好 | p.33 |
| 可怜 | 忽视和遗弃、成瘾问题、疾病或伤残 | p.1 |
| 可信 | 缺陷 / 坏习惯、脾气耿直、抢车 | p.1, p.33 |
| 可爱 | 值得被爱、救猫咪、关心家人 | p.1, p.33 |
| 可爱 | 让主角做一件美好的小事,适用于强势角色 | p.33 |

### 可爱的四个层面(p.1)

| 层面 | 手段 |
|---|---|
| 道德层面 | 关心家人(赚钱给妈妈) |
| 性格层面 | 自尊心强(坚持澄清自己) |
| 外形层面 | 帅 / 有魅力 |
| 技能层面 | 观察和推理(照片细节、袜子、记忆力好、准确记住地址) |

## Context & problem
**这套方法解决的问题是**:艺考考生写主角时两个极端错误——要么把主角写成**完美英雄**(没有可信),观众觉得假;要么写成**彻底惨剧**(只有可怜),观众压抑不想看。

p.1 给出的硬规则是 **"可爱 > 可信 > 可怜"**——可爱最重要。因为:
- 只可怜 → 观众看的是悲剧新闻,不是故事
- 只可信 → 观众看的是记录片,没有投入
- 只可爱 → 观众看的是偶像剧,没有力度
- 三者齐备 → 观众**愿意跟他走完 2000 字 / 120 分钟**

## Claims

1. **优先级是可爱 > 可信 > 可怜**(【原文】p.1)
   - 【我的理解】这与"先有观众认可,再让观众相信,最后让观众同情"的心理顺序一致——但可怜往往是**最先出场的钩子**(开场事件),可爱是**最强的留存力**(第二幕之后),可信是**全程黏合剂**。

2. **可怜不是"惨",是"值得同情"**(【原文】p.1, p.33)
   - 精神折磨(私自拆信朗读、网络欺凌、公然污染、公然嘲弄)
   - 物理困境(忽视、成瘾、疾病、伤残)
   - 社会处境(职业卑微、被欺负)
   - 关键是**主角不是咎由自取**——如果是自作自受,观众不会同情。

3. **可信 = 缺陷 + 坏习惯**(【原文】p.1, p.33)
   - 【原文 p.33】"缺陷 / 坏习惯"是可信的两种标准实现
   - eg. 脾气耿直(优点的极端化)、抢车(小过错但不算罪)
   - 【我的理解】可信的本质是"主角身上有**你也有**的东西",所以你相信这个人是真人。

4. **可爱 = 值得被爱**,最经典操作是**让主角做一件美好的小事**(【原文】p.33)
   - 这就是 Blake Snyder 的 *Save the Cat!* 里的"救猫咪"——主角在出场不久做一件让观众喜欢他的小事,这件事不用大,但必须**无功利**。
   - p.1 列出的"救猫咪行为":
     - 与孩子互动
     - 对待下属或员工的善意
     - 找回失物
   - **p.33 特别指出**:"可爱技巧**适用于强势角色**"——因为强势角色最需要"救猫咪"来防止观众觉得他是反派。

5. **可怜+可信+可爱的具体实例(p.33 的《叫我第一名》拆解)**
   - **可怜**:被同学欺负、被父亲嫌弃、找工作受到歧视
   - **可信**:妥瑞氏症、自卑
   - **可爱**:坚持、乐观、帅、老师身份
   - → 这是一个教科书级的三角平衡

6. **p.33 的《荒岛求生》拆解**
   - **可怜**:遇难到荒岛
   - **可爱**:选择怀表(爱)+ 安葬机长(善良)+ 没有立马拆快递(道德和职业操守)
   - **可信**:注重工作忽略了家人

7. **身份 / 欲望 / 动作是人物塑造的三要素**(【原文】p.33)
   - 身份 = 三位一体(可怜可信可爱)
   - 欲望 = 原始(名利权情) + 高级(奉献、牺牲、正义)
   - 动作 = 得到观众认可的连续行动
   - 三位一体解决的是**身份**,不能替代欲望和动作。

## Method / How to use

### 三角平衡检查(1 分钟)
写完主角初稿后,用这张表自检:

| 维度 | 有没有具体事件支撑? | 分值 0-2 |
|---|---|---|
| 可怜:主角遭遇了什么他不该承受的事? | | |
| 可信:主角身上有什么缺陷/坏习惯? | | |
| 可爱:主角做了什么美好的小事? | | |

**总分 < 4 = 不及格**(观众不会跟)
**4-5 = 及格**
**6 = 优秀**(三角齐备)

### 写作顺序建议
- **先定可怜**(让观众有初始共情)
- **再定可爱**(让观众愿意留下)
- **最后定可信**(让观众相信这个人是活的)

### 可爱的"救猫咪"时刻放在哪?
放在**开场事件里**,和可怜**同一场戏或相邻场戏**。不能等到中段——等到中段,观众已经走了。

### 反向检查(Anti-pattern)
- 主角只剩可怜 → 悲情片质感,没有动力
- 主角只剩可信 → 平庸,不记得
- 主角只剩可爱 → 甜腻,没有张力
- 主角可怜 + 可爱(缺可信) → 玛丽苏
- 主角可怜 + 可信(缺可爱) → 黑色写实,不适合艺考
- 主角可信 + 可爱(缺可怜) → 小资,没有共情钩子

## Debates & limits
- **三位一体不是全球剧作通用法**。【学界常见解释】好莱坞传统(Syd Field / Linda Seger)更强调 "character wants" 和 "character needs" 的对立;Robert McKee 讲 "gap between expectation and reality"——这些框架都没有"可怜/可信/可爱"这样的表述。
- **三位一体是艺考本土化产物**。【你的推断】这个说法最可能源自中国艺考教学法,它把"观众共情机制"简化到三个最直观的形容词,是为了让**高中生快速上手**。在专业剧作里,同样的效果用"flawed protagonist with likability"更常见,但这个长。
- **"可怜"的度很难把握**。【我的理解】太惨会变"苦情戏",太轻会没钩子。p.1 暗示的平衡是"**可怜要具体,不要笼统**"——用具体事件(抢车、丢单车)而不是形容词(悲惨、不幸)。
- **"可爱优先"有争议**。有老师认为"可信优先"更合理,因为"不可信的可爱"是假的。p.1 给的优先级是"可爱 > 可信 > 可怜",但 p.33 没再强调优先级——说明这个排序本身不是铁律,只是一种教学口径。

## Directing takeaways
1. **主角出场 30 秒必须可见"可怜"+"可爱"的证据**。考官看前 300 字决定是否感兴趣。
2. **可怜用"事"不用"词"**。不要写"主角很可怜",写"主角晚上吃半块冷馒头"。
3. **可爱要做,不要说**。不要写"主角是善良的人",写"主角把最后一块馒头分给流浪猫"。
4. **可信是**"缺陷"**不是"缺点"**。缺点是道德瑕疵,缺陷是性格特征——后者让人物活,前者让人物脏。分清。
5. **强势角色必须有可爱时刻**。写律师、医生、富商时尤其要注意——否则观众会默认他是反派。
6. **三位一体 + 五事件大纲的耦合**:开场事件展示可怜 + 可爱,激励事件触发欲望,转折/危机事件暴露可信的缺陷如何阻碍目标,高潮事件靠克服缺陷解决。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 可怜 = 悲惨 | 可怜是"不该承受的事",悲惨是"承受了过多的事"。后者让人想逃离,前者让人想靠近。 |
| 可信 = 真实 | 可信是**性格层面的真实**(缺陷、坏习惯),不是**题材层面的真实**(写实主义)。 |
| 可爱 = 外形帅 | 外形只是可爱的**四个层面之一**(p.1)。道德、性格、技能也能产生可爱。 |
| 救猫咪 = 爱动物 | 救猫咪是**泛称**,指任何"无功利的美好小事"。帮老奶奶过马路、让座、捡垃圾都算。 |
| 三位一体 = 三个性格 | 三位一体是**同一个人的三层钩子**,不是三组独立性格。 |

## Open questions
- 【待核实】"可爱 > 可信 > 可怜"这个优先级的出处——是原笔记老师的个人教学,还是某本艺考教材的规定?
- **"可怜"在文化差异里是否稳定?**——西方剧作更少强调"可怜",更多强调 "flawed but compelling"。是否因为中国观众更习惯"同情驱动"?
- **"救猫咪"是否就是可爱的全部?**——p.1 列出的救猫咪只有 3 项(孩子/下属/失物),但可爱的层面有 4 个(道德/性格/外形/技能)。两者关系还需更精细的拆分。
