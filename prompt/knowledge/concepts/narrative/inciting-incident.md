# 激励事件(Inciting Incident)

## TL;DR
**激励事件**是故事前 1/4 发生的、**彻底打破主角生活平衡**的第一个重大事件。它同时出现在两种剧作术语体系里:三幕剧的 **情节点 1 (Plot Point 1)**、十五节拍表的 **催化剂 (Catalyst)**——名字不同,指同一个节点。它的作用是"**让观众相信主角动机的那件事**",为整个故事提供动力。原笔记 p.6-7 给出了激励事件的三种触发形态(突变/缓变/他人带来)、三档动机催化难度、以及"难 + 一线生机"的内在要求。

## Definitions

- **激励事件(Inciting Incident)**:彻底打破主角生活平衡、让主角被迫进入新目标的第一个重大事件。
- **生活平衡(Status Quo)**:故事开始时主角的"正常状态"。激励事件正是对此的破坏。
- **催化剂(Catalyst)**:Blake Snyder 在 *Save the Cat!* 里对同一概念的命名。
- **情节点 1(Plot Point 1)**:Syd Field 在 *Screenplay* 里对同一概念的命名。

## Key terms

### 原笔记 p.6-7 的硬指标
| 指标 | 要求 |
|---|---|
| 时间位置 | 故事前 1/4 |
| 状态 | 动态、有冲突、时间明确(不是静态、模糊) |
| 冲突强度 | 充分发展 |
| 方向性 | 清晰明确地催化主角动机 |
| 前因 | 往往不是单一事件,需要"伏笔"铺垫 |
| 功能 | "第一幕的落幕代表了冲突锁定" |

### 困境的三种触发形态(p.6)
| 形态 | 定义 | 例子 |
|---|---|---|
| 突变 | 突然发生的一件事打破生活平衡 | 亲人突发车祸 |
| 缓变 | 之前就埋下祸根,在节点显现 | 长期隐瞒的真相被揭穿 |
| 他人带来 | 来自三观/社会认知巨大差异的人 | 陌生人带来的委托 |

### 动机分三档(p.6-7)
| 类型 | 催化难度 | 示例 |
|---|---|---|
| 生存 / 职业 | 最易 | 求生故事、警察医生接到任务 |
| 人物关系 / 情感线 | 需铺至少一个伏笔事件 | 亲人失联、爱人出走 |
| 发展 / 理想 | 最难,需多次催化 | 坚持艺术梦想 |

## Context & problem
**这个概念解决的问题是**:学生写开端时经常磨洋工——主角吃饭、上学、回家、睡觉,到 1/3 还没进入故事。激励事件的存在是一个**硬性时间刻度**,强制要求你在前 1/4 处给出"打破平衡"的那一下。

它还解决另一个问题:**主角为什么动?** ——不是因为编剧让他动,而是因为**他原本的生活被摧毁了,不得不动**。激励事件是"不得不"的制造者。

## Claims

1. **激励事件必须彻底打破平衡,不是小麻烦**(【原文】p.7)
   - 【原文】"通过彻底打破主角生活平衡,进而激发观众期待的第一个重大事件"
   - 【原文 p.7 三个例子】
     - 《亲爱的》:男主的孩子被拐卖
     - 《十七岁的单车》:男主为追女孩买的单车丢了
     - 《这个杀手不太冷》:玛蒂尔达全家被恶警杀死,变成孤儿

2. **激励事件 ≠ 第一次冲突**
   - 之前可以有伏笔、小摩擦,但那些不是激励事件本身
   - 激励事件是"**锁定**"——这之后主角不能回到原状态

3. **主角必须因激励事件产生动机**(【原文】p.6)
   - 如果激励事件发生了,主角却没动——那不是激励事件
   - 动机的出现是激励事件成立的**验证**

4. **困境的三种触发形态对应不同的写作难度**
   - 突变最易写,但容易"巧合感"
   - 缓变最考验伏笔铺陈
   - 他人带来是"离水之鱼"的典型触发(见 [故事核与人物反差](../../theory/story-core-and-contrast.md))

5. **"难 + 一线生机"的硬要求**(【原文】p.7)
   - 【原文】"难、一线生机:给主人公一线生机"
   - 困境必须难到让观众担心,同时必须留缝让观众相信"还有可能"
   - **没有生机的困境 = 停止的故事**

6. **写开端的最佳方法是倒推结尾**(【原文】p.7)
   - 【原文】"清楚地知道你的结尾,结局意味着什么,解决方法"
   - 【我的理解】这对应"backward outlining"——先决定主角死活/胜负,再倒推激励事件应该是什么。

## Method / How to use

### 激励事件合格检查 6 条
- [ ] 发生在故事前 1/4 吗?
- [ ] 是动态的、有冲突、时间明确吗?(不是静态的、模糊的)
- [ ] 彻底打破了主角的生活平衡吗?
- [ ] 催化了主角的动机吗?(不只是"发生了一件事")
- [ ] 铺了伏笔吗?(不是空降)
- [ ] 留了一线生机吗?

### 艺考开场事件模板(三件套)
在第一幕 300-400 字里完成:
1. **展示生活平衡**(主角正常状态,含三位一体的钩子)
2. **救猫咪时刻**(锁定观众好感)
3. **激励事件**(打破平衡,锁定目标)

### 三种触发形态选哪种?
- 写短故事 → 突变(最快展开)
- 写长故事或想炫技 → 缓变(伏笔出彩)
- 写反差或错位 → 他人带来

### 动机档位选择
- 保底:生存/职业(eg. 求生、执行任务)
- 拉分:人物关系/情感(eg. 寻亲、救爱人)
- 冲分:发展/理想(需要多次催化,难但高分)

## Debates & limits
- **时间位置的灵活性**。【学界常见解释】Robert McKee 在 *Story* 里并不坚持"前 1/4",他的标准是"**能合理放多晚放多晚**"——只要观众还没睡着。《教父》的激励事件(劫持)发生在差不多 1/4,但《记忆碎片》的激励事件需要观众倒推才能确定。
- **"彻底打破"的度**。一些文艺片的激励事件非常微弱(一封信、一场雨),但足以成为转折。p.7 的"彻底"二字更适合商业片,作者电影可以用更柔的触发。
- 【我的理解】在艺考场景下,激励事件应该足够强——因为短篇 2000 字没有空间用微弱事件做转折。
- 【你的推断】p.7 把"动机分三档"放在激励事件一节,实际上是在说:**动机决定了激励事件的强度需求**。理想型动机需要更强的激励事件,职业型动机用较弱的事件就够。

## Directing takeaways
1. **激励事件的镜头要够重**。不要用对话说"你家被烧了",要让主角亲眼看到火。
2. **激励事件必须是主角能感受到的**——不能是"背景里的新闻",必须进入主角的生活。
3. **激励事件前的铺垫是"可怜+可爱"的展示时间**,不要浪费。
4. **激励事件 + 救猫咪时刻最好相邻或重叠**,让观众的两种情绪(同情+喜爱)同时被激活。
5. **不要在激励事件之后立刻"说明"**——让主角的反应和后续动作说明一切。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 激励事件 = 开场事件 | 激励事件是**情节点**,开场事件是**第一幕的起点**。开场建立平衡,激励打破平衡。 |
| 激励事件 = 第一个冲突 | 激励事件是"锁定目标"的事件,之前的摩擦都是伏笔,不是激励事件。 |
| 激励事件 = 戏剧性场面 | 激励事件可以很安静(一封邮件、一个眼神),重点是"打破平衡",不是声光效果。 |
| 激励事件 = 情节点 2 | 情节点 2 是第二-三幕间的"一线生机",时间位置完全不同。 |

## Open questions
- 【待核实】"发生在故事前 1/4"的硬数字出处——Syd Field 的 25% 规则?还是中国艺考的通行教学?
- **多重激励事件**是否存在?——有些片(eg.《低俗小说》)同时有多个"激励事件",对不同人物产生不同动机。
- **激励事件与"无动机故事"**:散文化电影(侯孝贤)常常没有明显的激励事件,这种片如何在艺考语境下被评价?
