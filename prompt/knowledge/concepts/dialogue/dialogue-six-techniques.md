# 六种台词技巧(Six Dialogue Techniques)

## TL;DR
原笔记 p.24 列出了**六种台词技巧**:**打断、反问、压缩、角色、沉默、避免准确直接的回应**——它们的共同目标是让台词脱离"生活对话",变成"故事对话"。p.11 进一步给出"生活对话 vs 故事对话"的五条对比,p.24 还按**家长 / 成人 / 孩子**三种人物口径给出典型语句。这六技本质是回答一个问题:**怎么让台词少说正事、多传信息?**

## Definitions

- **台词技巧**:让台词承载情节/性格/潜台词的手段。
- **打断**:一句话没说完被另一个人截断,制造节奏和冲突。
- **反问**:用问句而非陈述句,回避直接回答。
- **压缩**:用最少的字表达最大的信息,省略主语/宾语/连接词。
- **角色**:让台词带上人物的角色口吻(家长/成人/孩子)。
- **沉默**:该说话时不说话,让"不说"本身成为台词。
- **避免"准确直接"的回应**:不让人物按对方提问的方向回答,而是偏移/跳转。

## Key terms

### 六种技巧(p.24)
| 技巧 | 目标 |
|---|---|
| 打断 | 制造对抗和节奏 |
| 反问 | 回避 + 施压 |
| 压缩 | 提高信息密度 |
| 角色 | 符合人物身份口吻 |
| 沉默 | 让"不说"说话 |
| 避免"准确直接" | 让潜台词出现 |

### 三种角色口径(p.24)
| 口径 | 特征 | 典型语句 |
|---|---|---|
| 家长 | 制定规则、滥用职权、话题终结者 | "爱听不听,不听滚蛋" |
| 成人 | 情绪稳定、善于分析、危急关头挑大梁 | "先别急,我们坐下来好好想一想" |
| 孩子 | 人性、天真、营造对话气氛 | "不嘛不嘛,我就要" |

### 生活对话 vs 故事对话(p.11)
| 生活对话 | 故事对话 |
|---|---|
| 忽略主语 | 明确目的 |
| 内容琐碎 | 角色性格 |
| 冗长 | 推进情节 |
| 重复 | 隐晦表达私密信息 |
| — | 只用来表达情绪或氛围 |

## Context & problem
**这套技巧解决的问题是**:学生写对话时有两种通病——
1. **太像生活**:你一句我一句,主语齐全,逻辑完整——但**没有戏剧性**。
2. **太像口号**:每句都在说道理、说主题——但**没有人物**。

六技加三口径的目标是让台词同时做到**两件事**:
- 符合人物(角色口径)
- 推进故事(六技巧)

而**不直接说正事**。

## Claims

1. **故事对话的底层规则:不直接回答**(【原文】p.24)
   - 【原文】"避免'准确直接'的回应"
   - 【我的理解】这是六技的**母规则**——其他五技都是"不直接回答"的具体形态:打断是不让对方问完、反问是反向提问、压缩是省略直接内容、沉默是不回答、角色是用身份代替回答。

2. **压缩是信息密度的关键**
   - 生活对话:"我昨天去超市买了点东西,然后呢,我就回来了,路上遇到了你妈"
   - 故事对话:"昨天在超市看见你妈了"
   - 【我的理解】压缩不是省字,是**省掉无关信息**。

3. **沉默是最难的技巧**(p.24 仅列出,未展开)
   - 【学界常见解释】Harold Pinter 的"Pinter Pause"(品特式停顿)是现代戏剧最著名的沉默技巧——关键时刻不说话,让观众自己填空。
   - 【我的理解】沉默 = 台词,但需要前面所有戏的铺垫才能让沉默变成"有分量的不说"。

4. **角色口径决定语言系统**
   - 家长讲规则,成人讲分析,孩子讲情绪——三种口径决定了对话的**句式、词汇、节奏**。
   - 【你的推断】同一个人物也可能在不同场合用不同口径(成人对孩子时讲家长,对领导时讲孩子)——这种**口径切换**本身就是人物塑造。

5. **生活对话 vs 故事对话的五条对比**(【原文】p.11)
   - 故事对话必须**明确目的**:每一句台词都应能回答"这句话为什么存在"。
   - 故事对话必须**体现角色性格**:同一个信息不同人物说法不同。
   - 故事对话必须**推进情节**:要么传递信息、要么改变关系、要么制造冲突。
   - 故事对话可以**隐晦**:潜台词比表层更重要。
   - 故事对话可以**只表达情绪**,不负载信息——但必须是有节奏的情绪。

## Method / How to use

### 六技巧 checklist(写完对话后)
逐句问:
- 能用"打断"代替完整句吗?
- 能用"反问"代替陈述吗?
- 能"压缩"到更短吗?
- 符合这个人物的"角色口径"吗?
- 这句话能不能用"沉默"代替?
- 是否在直接回答对方?如果是,偏移它。

### 写台词的 3 步法
1. **先写生活对话初稿**(想怎么说就怎么说)
2. **用五条故事对话规则检查**(p.11)
3. **用六技巧改写**(p.24)

### 练习:一句改五遍
原句:"我很爱你。"
- 用反问:"你还要我怎样?"
- 用压缩:"……是你。"
- 用打断:"我——算了。"
- 用角色(家长):"别问这种傻话了。"
- 用沉默:(看他很久,不说话)
- **这五种里哪一种最像你的人物?选它。**

## Debates & limits
- **"不直接回答"的边界**。【学界常见解释】Aaron Sorkin 的剧作以"快速、精准、直接"著称(《白宫风云》《社交网络》),几乎和"不直接回答"相反——但 Sorkin 的直接是**带智力攻击性**的直接,不是生活式的直接。规则有例外。
- **沉默技巧对表演依赖极大**。沉默若无对应的镜头、表情、动作撑住,会变成"剧本空白"。
- 【我的理解】六技巧是考场实用工具,但不是全部——真正的剧作大师会根据人物需求发明自己的技巧。
- 【你的推断】三口径(家长/成人/孩子)让人联想到 Eric Berne 的《人间游戏》(*Games People Play*,1964)里的**交互分析**(Transactional Analysis)里的"Parent / Adult / Child"三种自我状态——原笔记的老师可能受过这本书的影响。

## Directing takeaways
1. **每写一句台词,默数自己是不是在"直接回答"**。如果是——换一种方式说。
2. **压缩到"少一个字就不对"为止**。多余的字就是对戏的伤害。
3. **沉默比台词更有力**——但沉默必须被铺垫,不能是"我想不出词了"的沉默。
4. **家长 / 成人 / 孩子口径可以成为人物弧光的外在标志**。eg. 主角从"孩子式"说话变成"成人式"说话,就是成长的证据。
5. **潜台词靠"偏移"实现**:对方问 A,你答 B——B 和 A 的距离就是潜台词的深度。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 压缩 = 简短 | 压缩是省掉无关信息,**不是让台词变短**。有时候压缩会变长(因为需要绕开直接回答)。 |
| 沉默 = 没有台词 | 沉默是"应该说话时不说话",有**明确的期望对象**。没台词 ≠ 沉默。 |
| 反问 = 疑问句 | 反问不是真的在问,而是不想回答/在反击。 |
| 避免直接 = 模糊 | 避免直接是"偏移",不是"说不清"。偏移后观众应该能懂你的意思。 |

## Open questions
- 【待核实】六技巧的"角色"具体指什么?——p.24 没展开,只列出 3 种口径。可能是不同老师的不同命名。
- **第七、第八技巧是否存在?**——如反复(同一句话重复)、引用(用诗/歌词/谚语代替直接表达)。
- **东方对话传统的"含蓄"是否等同于"避免直接"?**——两者表面像,但东方含蓄更强调"点到为止",避免直接更强调"偏移"。需要进一步区分。
