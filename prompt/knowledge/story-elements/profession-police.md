# 警察职业素材库

## TL;DR
原笔记 p.58-59 做了一次**"警察这个职业怎么写真实"**的素材积累——从职业病到手的特征到警服到装备到警衔到二人关系模式。跟 [[./profession-doctor]] 一样,这是一张让"泛泛写一个警察"变成"内行写一个警察"的真实性清单。

## 一、警察的职业病(p.58)
- **爱数数**(狱警)
- **关门强迫症**(狱警)
- **走在队伍后面**
- **声音敏感症**(铃声应激障碍)
- **扫描强迫症**

**使用要点**:这些"职业后遗症"是塑造**可信维度**的利器——让主角在日常场景里表现出"警察的影子"(聚餐时走在最后、听到铃声全身绷紧),观众会立刻相信他是真警察。

## 二、警察的手(p.58)
| 警种 | 手的特征 |
|---|---|
| 特警 | 骨节分明,虎口、手心老茧、刀疤 |
| 交警 | 白手套;手臂肤色比正常人黑八度;紧急救援 |
| 法医 | 不主动和人握手 |

**使用要点**:"手"是最好的人物符号——一个特写就够了。法医"不主动和人握手"既是职业习惯也是人物距离感——一举两得。

## 三、警服系统(p.58)

### 衬衫颜色 = 等级
| 颜色 | 等级 |
|---|---|
| 蓝衬衫 | 基层 |
| 白衬衫 | 三级警监以上(25 年 +) |

### 警衔等级
1. 总警监、副总警监
2. 警监
3. 警督
4. 警司
5. 警员

### 警服配件
- 警衔、警号、胸徽、臂章

### 警号封存制度
- **因公殉职**:警号封存
- **重启**:其子女可以重启该警号

**使用要点**:"警号封存"本身就是一个**天然的悬念道具+象征道具**——一个被封存的警号被子女重启,就是"传承"主题的最短路径。

## 四、警用八大件(p.58)

4. 警用水壶

8. 强光手电

## 五、警察黑话(p.58)
- **雷子 / 条子**:犯罪分子对警察的称呼

## 六、二人关系模式(p.58)

### 《七宗罪》模式:老带新(p.58)
- **新警察 + 老警察**
- **对比轴**:经验资历、办案方式、性格
- 这是 [[../genre/buddy-love]] 的"同事之情"子类

### 卧底模式:双重身份(p.59)
- **卧底警察隐藏身份 = 双重身份**
- **迫不得已干坏事**
- **为了保住线索以身试毒**
- **隐姓埋名、改头换面**
- **外形标志**:汤头纹身叼烟、抽抽剩的烟头

**使用要点**:卧底是**麻烦家伙 + 被制度化的混血**——主角被迫进入犯罪集体(被制度化),却不是自愿的(麻烦家伙的"无辜")。

## 七、少数群体警察(p.59)
- **兔子警官**(《疯狂动物城》)
- **自身属性劣势**——体力/体型/身份
- **但也有独特优势**
- **真实案例**:义务执勤 32 年聋人"交警"

**使用要点**:少数群体警察是**变迁仪式 + 麻烦家伙**的天然混血——身份本身就是"生活的问题"(变迁仪式),同时不断遇到麻烦(麻烦家伙)。

## 八、情理与法理的冲突(p.59)
- 《我不是药神》
- 《亲爱的医生》

**使用要点**:这是警察故事**最高级的冲突设计**——法律说"他有罪",人情说"他是好人"。对应 [[../theory/conflict-six-types]] 的"现实与理想"冲突。

## Method / How to use

### 警察故事设计 checklist
- [ ] 选了哪个警种?(特警/交警/刑警/法医/狱警/缉毒/网警)
- [ ] 有没有至少 2 个**职业细节**?(手的特征/衬衫颜色/装备)
- [ ] 有没有**黑话**增加内行感?
- [ ] 二人关系模式选了哪种?(老带新/卧底/少数群体)
- [ ] 是否涉及**情理 vs 法理**的两难?
- [ ] 警察本人的**职业后遗症**有没有被写进去?

### 警察 × 类型叙事
| 类型 | 警察版本 |
|---|---|
| 麻烦家伙 | 普通交警被卷入大案 |
| 被制度化 | 新警察进入老警察体系(老带新) |
| 伙伴之情 | 新老搭档(《七宗罪》) |
| 变迁仪式 | 退休老警察面对余生 |
| 惊悚片 | 卧底被揭发 |

## Debates & limits
- **原笔记的警察素材聚焦中国警察体系**——衬衫颜色、警衔、八大件都是中国标准。写外国警察故事需要另行查证。
- 【我的理解】艺考编讲故事写警察的最大优势是**自带"生死考验"**——不需要额外制造,职业本身就是。但劣势是**容易写成"主旋律宣传片"**——必须给主角一个**个人缺陷**(职业后遗症/家庭矛盾)才能避免。
- 【你的推断】法医"不主动和人握手"可以展开成一个完整的伙伴之情故事——法医(封闭)+热情的新同事(开放),冲突就在这一握手。
