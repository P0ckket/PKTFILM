# 人物设计素材库(主角身份清单)

## TL;DR
这是一张**编讲故事用的主角身份速查表**——原笔记 p.1, p.11, p.14, p.18-19, p.33-34 反复出现的"怎么设计一个让观众立刻站在他一边的主角"素材。核心框架是 [可怜/可信/可爱 三位一体](../concepts/character/character-trinity.md) 的 **可怜+可信+可爱** 三维度,配合四大主角类型(**底层 / 英雄 / 女性 / 儿童**)的具体困境清单。艺考编讲故事 2000 字开场,主角身份设计 30 秒就要让老师认定"这个人值得看"——本卡给的是可直接挑的积木。

## 主角三维度的表现方式清单

### 一、可怜(职业/境遇/关系)(p.1, p.11, p.33, p.45)
- **职业卑微**:送外卖、环卫工、流水线工厂工人、流动商贩、驯兽员、飘飘车司机、杂技表演员、三和大神
- **经济困境**:穷、付不起医药费、交不起学费、没钱还债
- **家庭缺失**:母亲死掉、父亲离开、孤儿、被遗弃、留守儿童
- **遭受欺凌**:被同学欺负、被父亲嫌弃、找工作受到歧视(《叫我第一名》)
- **身份歧视**:智商不足受人欺负、遭受不公正待遇(《辛德勒的名单》)
- **生理缺陷**:不孕、阿尔兹海默症、哮喘、妥瑞氏症、残疾
- **精神折磨**:私自拆信朗读、网络欺凌、公然标记/污染、公然模仿/嘲弄
- **忽视和遗弃**:父亲孤独的死去、童年阴影

### 二、可信(缺陷/坏习惯)(p.1, p.11, p.33)
- **性格缺陷**:脾气暴躁古怪、自私、懦弱、悲观、执行力差(《百元之恋》)、自卑、懒惰、偏执、资历浅
- **道德缺陷**:啃老、作弊或盗窃、恋爱/婚姻不忠诚、利用他人、公报私仇
- **习惯缺陷**:抢车、酗酒、沉迷赌博
- **专业缺陷**:新警察(经验不足)、新医生(资历浅)、实习生

### 三、可爱(四层面)(p.1, p.33)
| 层面 | 表现方式 | 例子 |
|---|---|---|
| 道德层面 | 关心家人、赚钱给妈妈、救猫咪、对下属员工善意、找回失物、与孩子互动 | 荒岛求生安葬机长 |
| 性格层面 | 自尊心强、坚持澄清自己、不妥协的工作态度、对抗欺凌、拒绝施舍、乐观、坚持 | 叫我第一名 |
| 外形层面 | 帅、漂亮、有气质 | — |
| 技能层面 | 观察和推理(照片细节/袜子/记忆力)、嗅觉灵敏(说出香水品牌)、预测他人行为、揭露谎言、敏锐听力 | 神探夏洛克 |

**排序**:**可爱 > 可信 > 可怜**——看 [可怜/可信/可爱 三位一体](../concepts/character/character-trinity.md) 的优先级讨论。

## 四大主角类型库

### 一、底层人物(p.14, p.18)
**定位**:越底层 → 抗风险能力越弱 → 越容易陷入困境。
**写作要点**:要写出他们的 **"难"**——**低收入 + 不被尊重 + 难以照顾家庭**。

| 子类 | 代表作 |
|---|---|
| 老 | 一个叫欧维的男人决定去死、桃姐 |
| 病 | 滚蛋吧肿瘤君、送你一朵小红花、我们一起摇太阳 |
| 病 + 老 | 遗愿清单 |
| 残 | 阿甘正传、触不可及 |
| 超低收入 | 最后的棒棒 |

**三种健康问题**:
- 身体健康问题
- 心理健康问题(孤独感、社交隔离)
- 社会保障问题(年龄歧视、养老服务体系)

### 二、英雄人物(p.18)
**定位**:自带主角光环——但要写出"牺牲 / 创伤 / 道德压力 / 被道德绑架"的代价,才能避免工具人。

| 职业 | 细分 |
|---|---|
| 警察 | 治安警察、缉毒警察、狱警、海关、交警、网络警察 |
| 军人 | 陆军、海军、空军、火箭军、战略支援部队 |
| 老师 | 学科、教学地点、学生年龄段 |
| 消防员 | — |
| 医生 | 科室、患者群体(人/动物)、有无生命体征 |
| 检察官 | — |
| 律师 | — |
| 科研人员 | — |

**找冲突的三轴**:
- **工作环境**:对职业进行细致分类
- **确定时态**:未来、现在、过去
- **喜爱程度**:喜欢、不喜欢

### 三、女性角色(p.18-19)
**定位**:从**多重身份(家庭 / 社会)**切入寻找冲突。

| 困境 | 代表作 |
|---|---|
| 职场不公 | 穿普拉达的女王、实习生 |
| 年龄焦虑(20/30/40) | — |
| 生育压力 | 朱诺、四月三周两天 |
| 带娃/家务 | 82 年生的金智英 |
| 容貌焦虑 | 魔镜 |
| 穿衣自由 | — |
| 刻板印象 | — |
| 男性凝视 | — |
| 体力悬殊 | — |
| 重男轻女 | — |

### 四、儿童角色(p.19)
**定位**:借儿童视角反映成人社会问题。

| 困境 | 代表作 |
|---|---|
| 教育问题(父母+老师) | 放牛班的春天 |
| 成人规则的介入 | 狗十三、校合唱团的秘密 |
| 留守儿童 | 念书的孩子 |
| 被伤害 | 少年的你、素媛 |
| 没有赚钱能力 | 贫民窟的百万富翁 |
| 没有决定权/发言权 | 何以为家 |

## 人物的名字设计(p.15)
好名字本身就是人物的一部分:
- **墨菲** — 定律
- **陈真**(精武英雄)— 坚持真理与正义
- **程蝶衣**(霸王别姬)— 性别身份
- **赵子龙**(三国演义)— 英勇、忠诚
- 【我的理解】**名字是第一层人物设定**——艺考编讲故事 2000 字,起一个带含义的名字能节省大量铺垫字数。

## 人物小细节清单(p.1, p.2, p.32)

### 观察力的表现(p.1)
- 袜子颜色
- 预测他人行为
- 揭露谎言
- 敏锐的听力

### 嗅觉灵敏的表现(p.32)
- 说出香水品牌
- 关键时刻通过某种气味发现线索
- 闻到非常细微的气味(女人体香)

### 鼓起勇气的方式(p.1)
- 喝酒
- 取得小胜
- 他人的榜样
- 与亲友的交谈

### 敏感性格的表现(p.2)
- 换了地方无法适应
- 对新环境/不熟悉的人群感到不安
- 容易出汗、脸红
- 看电影容易共情

## 使用建议
### 开场 30 秒的主角身份设计 checklist
- [ ] 有没有一条 **可怜** 元素?(从 p.1 或 p.45 清单选)
- [ ] 有没有一条 **可信** 元素?(缺陷/坏习惯)
- [ ] 有没有一条 **可爱** 元素?(四层面任选)
- [ ] 主角属于四大类型中的哪一类?(底层/英雄/女性/儿童)
- [ ] 名字有没有含义?

### 案例速写(从素材库组装)
**女主 + 底层 + 女性组合**:一个 28 岁被家里催婚的女外卖员(可怜=底层女性),脾气暴躁怼过客户(可信),但每天把剩饭留给流浪猫(可爱)。
→ 30 秒内三维度齐全。

## Debates & limits
- **"四大类型"不是互斥**。《哪吒》的女主既是女性也是儿童,《触不可及》的主角既是底层也是残疾。类型是**视角切入点**,不是**标签**。
- 【我的理解】艺考编讲故事里,**女性 + 底层** 组合的分数天花板最高——因为它天然带"多重困境",也是老师最乐于看到的"社会关怀"向。
- 【你的推断】英雄人物反而是最难写的——因为写不好就变成"工具人",必须花额外字数写"代价"才能立住。艺考初学者慎选。
