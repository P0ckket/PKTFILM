# 医生职业素材库

## TL;DR
原笔记 p.50-52 用三页纸做了一次**"医生这个职业怎么写真实"**的详尽素材积累。从职业病到黑话到手术器械到科室排行到患者心理到家属困境——这是一张**写医疗题材故事**的真实性清单。艺考编讲故事选"医生"做主角,如果细节到位,会立刻和"泛泛写一个医生"的竞争者拉开差距。

## 一、医生的职业习惯(p.50)
- **写字像天书**
- **规范化洗手**
- **盯着人的某些部位反复看**
- **怪异方式开关水龙头/门**
- **再急,接触病人前也会先消毒**
- **在病房按压消毒液用手背**
- **在办公室或门诊用手心按压**

**使用要点**:这些"职业下意识"是**建立人物可信度**的最快手段——不用解释"他是医生",让他在吃饭前先规范化洗手,观众就知道了。

## 二、临床实操训练(p.50)
- **口腔医生**:镜像训练
- **腔镜练习**
- **缝合练习**

## 三、医院黑话(p.50)
| 黑话 | 真实含义 |
|---|---|
| 上台 | 上手术台 |
| 下台 | 下手术台 |
| 阳间 | 肝炎、梅毒、HIV |
| 看电影 | 腹腔镜手术 |
| 采摘 | 摘取供体脏器 |
| 仔细 | 当心,别乱说话 |

## 四、医院暗号系统(p.50)
| 暗号 | 含义 |
|---|---|
| 111 | 火灾 |
| 222 | 治安 |
| 333 | 集体中毒(需要大量医护人员) |
| 666 | 婴儿被盗 |
| 777 | 手电器产品故障 |
| 999 | 紧急救援 |

**使用要点**:暗号是**天然的悬念道具**——主角听到广播里播报"333",观众不知道什么意思,但看到所有医生同时起身奔跑——紧张感立刻到位。不需要解释,让动作说话。

## 五、医生的体贴小动作(p.50)
- **捂热听诊器**
- **触诊时搓手**(暖手)
- **经过病床前顺手盖被子**

**使用要点**:这三个小动作是**可爱(道德层面)**的完美载体——不用安排台词,一个"顺手盖被子"就够了。

## 六、成为医生的困境(p.50)
- **年限**:15 年以上的漫长培养
- **晋升**:需要出色科研能力
- **成本高**:经济 + 时间
- **容错率低**:一次失误可能终结职业
- **易受心理创伤**
- **医闹**

## 七、最累的五个科室(p.51)
1. **急诊科**
2. **重症医学科**
3. **手术室**
4. **妇产科**
5. **儿科**

## 八、医生的职业病(按科室)(p.51)
| 科室 | 易患疾病 |
|---|---|
| 外科医生 | 肾结石、尿路结石 |
| 内科医生 | 胃炎、胃溃疡、肥胖 |
| 放射科医生 | 白内障、皮肤病、血液系统疾病和癌症 |
| 重症医学科 | 肿瘤 |
| 麻醉科医生 | 痔疮 |
| 急诊科医生 | 失眠、抑郁症 |
| 儿科医生 | 全身外伤 |

**使用要点**:医生的职业病是**主角"可怜"维度**的绝佳载体——急诊科医生自己失眠,却每天拯救别人的生命。这种反差 = 天然的可怜+可爱。

## 九、患者心理(p.51)
- **病耻感**
- **"为什么" & "怎么办"** 的双重焦虑
- **病急乱投医**
- **是否为了更重要的事情推迟治疗**
- **器官移植的伦理焦虑**

## 十、家属困境(p.51)
- **隐藏情绪**(唯一不用跟医生演)
- **价位不同的药物如何选择**——便宜药 = 不孝?
- **时间条件**
- **病人间的攀比**
- **真相是否告知**——所谓"告知"是演一出戏(单方/双方)
- **害怕做决定**——"如果你妈妈患癌了,你会怎么办?"
- **是否积极抢救**

**使用要点**:家属困境是**编讲故事最好用的两难**——每一条都是"没有正确答案"的选择,天然产生主题。

## 十一、希波克拉底宣言(p.50)
这是医生职业的精神基石——"首先,不伤害"。用它做**主题锚点**:当主角面临"伤害一个人来拯救另一个人"的选择时,希波克拉底宣言就是道德天平的支点。

## 十二、医生故事的完整案例(p.51)——执行安乐死的宠物医生
| 元素 | 内容 |
|---|---|
| 主角 | 宠物救助员小简 |
| 原始欲望 | 救助更多的流浪动物 |
| 可爱 | 有爱心 |
| 缺陷 | 偏执、资历浅 |
| 配角 | 宠物救助所所长 |
| 矛盾 | 理想与现实的差距 + 被制度化 |
| 伦理问题 | 自我道德意识与法律之间做出选择 |
| 情节链 | 微小失误 → 真相还原 → 罪责判定 |
| 情感主题 | 不被接纳的孤独感 |

## Method / How to use

### 医生故事设计 checklist
- [ ] 有没有至少 2 个**职业下意识**细节?(洗手/捂听诊器/怪异开门)
- [ ] 有没有用**黑话或暗号**制造内行感?
- [ ] 选了哪个科室?科室的**特有困境**是什么?
- [ ] 患者/家属的**两难**是什么?
- [ ] 医生本人的**职业病**有没有被写进去?(自身也是患者)

### 医生 × 被制度化
医院是典型的**被制度化集体**——规则严格、等级分明、新人入场(实习生)、关键人物(主任)。可以直接对接 [被制度化](../genre/institutionalized.md) 的模板:
- **集体**:医院/科室
- **外来者**:实习医生/转科医生
- **关键人物**:科室主任/资深护士
- **集体表征**:白大褂、值班制度、交接班仪式、暗号系统

## Debates & limits
- **原笔记的医生素材偏向"细节积累"**而不是"故事结构"——它更像是编剧的田野调查笔记,而不是完整的医疗剧模板。
- 【我的理解】艺考编讲故事里,医生故事最大的陷阱是**写成"科普宣传片"**——堆砌专业术语但没有人物弧光。黑话和暗号是调味品,不是主菜。
- 【你的推断】中国近年影视对"医疗题材"的需求很高(《中国医生》《关于唐医生的一切》)——这些素材对校考面试的"行业分析"也有用。
