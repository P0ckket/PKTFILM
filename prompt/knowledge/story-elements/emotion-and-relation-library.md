# 情感与关系素材库

## TL;DR
这是一张**写人物关系变化**的素材表——怎么拉近两个人、怎么疏远两个人、怎么让人信任/和解/愧疚/冲动/思念。原笔记在多处反复列出了关系操作的具体动作、情境、原因。对应 [伙伴之情](../genre/buddy-love.md) 的四段式关系轨迹:厌恶 → 互补 → 分离 → 重聚,每一步都需要具体的"关系动作"来推进——本卡就是那些动作的仓库。

## 一、拉近人物关系

### 拉近方法一(p.2)
- **聊私人关系**
- **给予礼物**
- **分享食物或饮料**
- **透露自己的弱点**

### 拉近方法二(p.11)
- **同病相怜**
- **分享秘密**
- **对抗共同敌人**
- **照顾和帮助**

### 拉近方法三(p.32)
- **为对方报仇**
- **分享秘密**
- **共同兴趣和爱好**
- **感情的共鸣**

### 拉近关系的具体表现(p.32)
- **开玩笑**
- **倾诉心事**(分享自己的过去和秘密)
- **身体接触**(拍肩、握手、拥抱)
- **共同活动**(开车)

**核心原则**:拉近关系的本质是**交换脆弱性**——你向对方展示弱点,对方才会卸下防备。所以"透露弱点""分享秘密""同病相怜"反复出现——这不是巧合,是关系的底层逻辑。

## 二、人物关系疏远原因(p.3)
- **减少负罪感**
- **利益冲突**
- **生活阶段的变化**
- **过度的依赖**

**使用要点**:这四条对应 [伙伴之情](../genre/buddy-love.md) 的"分离段"——两人分道扬镳需要一个合理的原因,而不是"吵了一架"。

## 三、表达信任(p.11-12)
- **没有躲**
- **分享个人秘密**
- **放弃控制权**
- **共同承担风险**

**使用要点**:信任是**逐步建立**的——先是"没有躲"(被动信任),然后是"分享秘密"(主动信任),最后是"放弃控制权"(深层信任)。好的关系戏应该写出这个梯度。

## 四、表现和解的动作(p.16)
- **拥抱**
- **握手**
- **击掌或拳头相碰**
- **手牵手**

**使用要点**:和解不需要大段台词——一个动作就够。"拥抱"适合亲情,"握手"适合敌人和解,"击掌"适合队友,"手牵手"适合恋人或亲子。选对和解动作 = 选对关系属性。

## 五、父女之间爱的象征(p.16)
- **手表**
- **项链或珠宝**
- **信件或日记**
- **花朵或植物**

## 六、思念爱人的情景(p.45-46)
- **特定的音乐或歌曲**
- **翻看旧照片**
- **特殊的日期**
- **香水或者味道**

## 七、想要忘记痛苦的动作(p.45)
- **针灸**
- **饮酒**
- **剧烈体育活动**(跑步、拳击)
- **沉迷于工作或爱好**(《我们与恶的距离》)

## 八、人物内心转变的标志(p.36)
人物从"旧我"到"新我"的视觉化表现:
- **着装改变**(发型、化妆)
- **兴趣和爱好的转变**(从安静变冒险)
- **人际关系的变化**(从依赖他人到独立自主)
- **象征性物品的使用**(抛弃旧的或者使用新的)

**使用要点**:内心转变的标志对应 [人物弧光机制](../concepts/character/character-arc-mechanism.md) 的弧光终点——观众需要在视觉上"看到"人物变了,不只是"听到"人物说"我变了"。

## 九、过河拆桥的情节(p.45)
- **退票**
- **抢功**
- **无情统治**
- **陷害救人**

## 十、制造喜剧效果(p.3)
- **观众知角色不知**(dramatic irony)
- **身体喜剧**
- **模仿与讽刺**
- **言语幽默**

## 十一、葬礼出乎意料的情节(p.46)
- **复活**
- **遗嘱揭示**
- **未知亲戚出现**
- **爱情告白**

## 十二、人质情节的变体(p.12)
- **建立情感联系**(斯德哥尔摩)
- **内鬼揭露**
- **人质逆袭**
- **人质之间的冲突**

## Method / How to use

### 关系变化 × 五事件模板
| 事件 | 关系状态 | 具体技法 |
|---|---|---|
| 开场 | 陌生/对立 | 第一印象反感(伙伴之情四段式) |
| 激励 | 被迫接近 | 透露弱点 / 分享食物 |
| 转折 | 初步信任 | 分享秘密 / 对抗共同敌人 |
| 危机 | 分裂/背叛 | 利益冲突 / 生活阶段变化 |
| 高潮 | 和解/深层绑定 | 拥抱 / 击掌 / 手牵手 |

### 关系设计 checklist
- [ ] 两人有"交换脆弱性"的时刻吗?
- [ ] 关系有没有经历"拉近→疏远→重新拉近"的起伏?
- [ ] 和解的动作是否对应了关系的类型?(亲情/友情/爱情/敌对和解)
- [ ] 人物的内心转变有没有外显化?(着装/行为/道具变化)
- [ ] 关系变化有没有服务于主题?

## Debates & limits
- **"交换脆弱性"不是万能**。有些关系的建立不靠脆弱,而靠**能力互补**(如同事之情)或**共同经历**(如灾难之情)。脆弱性交换是拉近关系的"最快路径",但不是唯一路径。
- 【我的理解】艺考编讲故事里,**拉近关系比拉远关系更难写**——因为"拉远"可以靠一个事件(吵架/背叛),而"拉近"需要渐进。建议用"共同对抗一个外部威胁"来加速拉近——这比"聊天聊出了感情"节省字数。
- 【你的推断】原笔记的"过河拆桥"情节(退票/抢功/陷害)可以和 [被制度化](../genre/institutionalized.md) 的"关键人物"合用——关键人物对外来者做的就是"过河拆桥"。
