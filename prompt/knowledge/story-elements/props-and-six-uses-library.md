# 道具与六大用法素材库

## TL;DR
原笔记从 p.44 开始,用一半以上的页面(约 p.44-73)反复练习**"同一个道具的不同用法"**——以一个道具为单位,分别写成 **价值 / 象征 / 悬念 / 反差 / 转折 / 激励 / 危机 / 高潮** 八种不同情节功能。这是原笔记最核心的"造句训练"——**一个道具能解锁多少种情节可能性**,决定了写作者的素材调动能力。本卡把所有道具按功能八法做反向索引,把原笔记中分散的案例整合成**可直接取用的道具库**。

## 一、道具的八种功能(原笔记统称"道具用法")

| 用法 | 功能 | 对应节拍 |
|---|---|---|
| **价值**(Value)| 道具承载生命/生存/关键利益,失去它=失去一切 | Setup / Climax |
| **象征**(Symbol)| 道具代表抽象情感或主题 | Final Image / 主题表达 |
| **悬念**(Suspense)| 道具本身是谜题,推动追问 | Hook / Investigation |
| **反差**(Contrast)| 道具与人物/环境预期形成反差 | 开场 / 人物立体化 |
| **转折**(Twist)| 道具引发剧情转向 | Plot Point |
| **激励**(Catalyst)| 道具触发主角动机 | Inciting Incident |
| **危机**(Crisis)| 道具制造紧急处境 | Bad Guys Close In |
| **高潮**(Climax)| 道具在决胜时刻发挥作用 | Finale |

**原则**:一个道具**不止一种功能**。好的道具设计会让同一个道具在不同节拍承担不同功能——e.g. 一枚戒指开场是"价值"(求婚),中段是"转折"(丢失),结尾是"象征"(重逢)。

## 二、按道具类型的素材反向索引

### A. 生存/工具类道具

#### 火柴(p.47)
- **价值**:流浪荒野的小李找到最后一根火柴,点燃篝火渡过寒夜——生存关键
- **象征**:寒夜中小女孩点燃火柴,亮光照亮脸颊(《卖火柴的小女孩》原型)
- **悬念**:探险家仅剩三根火柴,数量成为生死悬念

#### 直升机(p.47)
- **价值**:雪崩中救援队调直升机救出被困
- **悬念**:陌生直升机在城市上空盘旋,居民不知来意
- **转折**:考察途中故障迫降荒岛,故事转向生存冒险

#### 信号弹(p.47)
- **价值**:被困海岛探险队靠最后一刻信号弹获救
- **转折**:小李误放信号弹引发敌人围攻
- **悬念**:营地外突现不明信号弹

#### 头盔(p.47, p.56)
- **价值**:车祸中佩戴头盔幸免于难 / 战场防护
- **激励**:小丽父亲(消防员)遗物,睹物思人
- **高潮**:两位竞赛对手攀登遇险,头盔成为和解象征;失联飞行员用头盔灯获救
- **悬念**:案发现场的血头盔上有奇怪符号
- **反差**:女生主角爱上极限运动,头盔挂满卧室

#### 麻药(p.61)
- **价值**:急诊室仅剩的剂量——给濒死孕妇还是警察?
- **危机**:手术台上麻醉师发现剂量用错
- **激励**:孩子因劣质麻药去世,母亲发誓揭露

### B. 身份/关系类道具

#### 信(p.53)
- **价值**:小村每人靠一封信联系外界;暴风雪封山,邮差冒死送信
- **悬念**:内容不完整的信 / 律师将密信交给林太太,写着"别相信任何人"
- **激励**:小林收到匿名信,暗示父亲死因不单纯

#### 戒指(p.70)
- **象征**:婚姻、束缚;女主摘下戒指象征对自由的追求
- **危机**:婚礼当天戒指丢失引发冲突
- **悬念**:失踪多年的戒指出现在陌生人手中

#### 手表(p.16, p.70)
- **象征**:父女之间爱的象征(p.16)
- **反差**:主人公发现真手表被换成假的
- **危机**:手表被朋友偷

#### 合照(p.56)
- **转折**:聚会发现亲密好友越来越远
- **悬念**:老相册里父母年轻合照背后写着奇怪数字
- **激励**:与父亲最后一张合照,决定实现父亲未竟的登山梦

#### 玩具(p.68)
- **象征**:破损娃娃象征单亲家庭孩子的孤独
- **价值**:离异父母争夺抚养权时,孩子执意带着破旧玩具熊

#### 娃娃(p.71)
- **象征**:失去的爱与遗憾——主人公失去最亲朋友后,娃娃成为怀念的象征

### C. 场景触发类道具

#### 蜡烛(p.63)
- **价值**:战争停电中唯一的光源
- **象征**:脆弱与希望(病重母亲每晚点燃为儿子祈祷)
- **危机**:被困地下室用蜡烛寻出口,火焰即将熄灭

#### 桥(p.53)
- **危机**:暴雨中桥突然崩塌,阻断逃离洪水道路
- **价值**:偏远村庄的新桥让孩子们不用再涉水上学
- **象征**:两家世代交恶的家庭因修桥成为邻居,孩子在桥上结友谊

#### 体育馆(p.73)
- **高潮**:多年恩怨的兄弟在突然相遇的比赛中释怀
- **价值**:见证主角三年训练从青涩到成熟
- **悬念**:深夜体育馆莫名亮灯

#### 铁轨(p.64)
- **危机**:小张赶火车时手机掉在铁轨上,情急之下跳下去捡

### D. 情感触发类道具

#### 眼睛(p.64)
- **价值**:宴会上陌生人带面具,只有一双深邃的眼睛在闪烁
- **危机**:侦探即将揭露杀手身份时突然失明
- **高潮**:少年被诬陷后失明,听觉敏锐后找出真相

#### 抽血(p.55)
- **转折**:抽血结果显示小王与父亲血型不符,揭开被领养身世
- **危机**:小李抽血报告显示患上了重病

#### 药(p.65)
- **价值**:母亲拼命工作只为给患病孩子买药
- **象征**:希望或依赖;坚持服药,但药已无效
- **激励**:父亲为救患病女儿去找黑市药商

#### 乐谱(p.66)
- **象征**:情感与回忆——老人弹奏的乐谱是已故妻子创作
- **悬念**:失踪作曲家留下的最后一页乐谱缺失
- **转折**:阁楼发现母亲年轻时的乐谱,化解母女矛盾

#### 日记(p.16, p.66)
- **象征**:父女/母女间爱的凭证(p.16)
- **价值**:母亲日记写着对女儿的愧疚(p.66)

#### 蝴蝶(p.65)
- **象征**:重生或消逝的灵魂(母亲在女儿坟前看到蝴蝶)
- **激励**:男孩在森林中看到蝴蝶蜕变,决心改变自己

### E. 身体/生理类道具

#### 手抖(p.53)
- **反差**:狙击手李刚手抖厉害,一枪击中目标
- **危机**:拆弹手抖差点剪错线
- **高潮**:深吸一口气抑制手抖投进篮筐

#### 烧伤(p.61)
- **转折**:失踪多年兄长被发现有独特烧伤痕迹
- **象征**:角色心理创伤——救人时的烧伤成为无法抹去的失败阴影

#### 眼泪(p.69)
- **反差**:主角以为母亲落泪是病情恶化,其实是化疗成功
- **危机**:女孩撒谎保护兄弟,一次失控的眼泪被反派识破
- **象征**:脆弱中的坚强

### F. 数字时代/物证类道具

#### 录音笔(p.57 变体)
- **激励**:作家把常用的笔送给年轻作者
- **危机**:记者意外得到一支特殊录音笔,里面记录惊天阴谋
- **高潮**:审判关键时刻曾消失的笔被出示

#### 名单(p.63)
- **危机**:泄露的名单记录证人姓名,必须抢在罪犯前找到
- **反差**:主角以为入选荣誉名单,打开才发现是被开除名单
- **重复**:多次梦到名字被划掉,焦虑加重

#### 名字(p.63)
- **象征**:遗产或家族荣耀
- **反差**:名叫"勇"的男孩性格胆小
- **转折**:告白信上写的不是她的名字,而是闺蜜的名字

### G. 仪式/特殊时刻道具

#### 葬礼(p.70)
- **反差**:亲戚们公开争吵,反应家庭破裂
- **悬念**:父亲葬礼发现神秘信封,里面有未解的遗嘱
- **高潮**:与多年未见的兄妹重新和解

#### 录取(p.63)
- **转折**:主人公收到录取通知书,发现是父母贿赂换来的
- **危机**:录取资格被不公正取消,必须在有限时间内证明清白
- **象征**:希望和重生

#### 生日(p.67)
- **价值**:孤儿院的第一个生日,院长准备蛋糕给予归属感
- **激励**:为了赶在孩子生日之前归家,主人公穿越战区
- **反差**:欢乐的生日宴会上父亲因心脏病倒下

## Method / How to use

### 道具"八法造句"练习
这是原笔记最推崇的基本功。选一个道具,挨个写八种用法:
1. **选一个道具**(任何都行,越日常越好:雨伞、钥匙、鞋带、唇膏)
2. **八种功能都写一遍**(价值/象征/悬念/反差/转折/激励/危机/高潮)
3. **每种功能用 2-3 句话写完整的情节片段**
4. **挑 2-3 条组合进同一个故事**

### 艺考编讲故事的"五事件 × 一道具"公式
选一个道具,让它在五个事件中都出现但功能不同——最强的情节黏合剂:
- **开场**:道具是"价值"(它很重要)
- **激励**:道具是"激励"(因为它触发目标)
- **转折**:道具是"转折"(它揭露真相)
- **危机**:道具是"危机"(它制造紧急)
- **高潮**:道具是"象征"(它代表主角的变化)

### 道具设计 checklist
- [ ] 这个道具有没有至少 2 种功能?
- [ ] 开场出现的道具,结尾有没有回扣?
- [ ] 道具的"物性"(材质、形状、重量、温度)有没有被写出来?
- [ ] 道具的主人/来源有没有故事?
- [ ] 道具是不是只服务于情节,而忽略了人物?

## Debates & limits
- **道具不能滥用**。一个故事里有 2-3 个有记忆点的道具就够了,7-8 个只会让观众记不住。选"最能承载主题"的那 1-2 个主道具。
- **原笔记的"八法"不严格**。价值 / 激励 / 危机 / 高潮属于**情节功能**,象征 / 反差属于**修辞功能**,悬念 / 转折属于**结构功能**——三类不同性质混在一起。但作为训练框架这样分很实用。
- 【我的理解】中国艺考 2000 字编讲故事,**一个主道具反复出现**比**多个道具并列**更稳——因为字数有限,反复出现同一道具能制造"回扣感",而多道具容易写成清单。
- 【你的推断】西方剧作理论里,这种"道具八法"更接近"Chekhov's gun 契诃夫之枪"的扩展——**如果一个道具出现在第一幕,它必须在第三幕响**。原笔记的"八法"可以看作契诃夫之枪的具体操作指南。
