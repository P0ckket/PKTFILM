# 场景与环境设置素材库

## TL;DR
这是一张**写故事时选场景、选环境、选时间点**的素材表。原笔记在 p.12-13, p.15, p.32, p.36, p.45 分散列出了大量"什么场景自带什么情绪"的条目。本卡整合为**六类场景**:紧张场景、危机环境、仪式感场景、抒情场景、特殊时间点、封闭/孤立地点。每类场景都标注了情绪色彩和适用的故事类型。

## 一、紧张场景(p.13, p.45)

### 经典惊悚场景(p.13)
- **沼泽、树林、老宅**
- **地下洞穴**
- **墓地**
- **地下室**

**共同特征**:封闭 / 黑暗 / 历史感 / 地下——都暗示"你不知道里面有什么"。

### 扩展紧张场景(p.45)
- **黑巷**
- **废弃建筑**(废弃的医院、学校或工厂)
- **密闭空间**(电梯、地下室)
- **森林、迷雾**
- **衣柜**(偷窥功能)
- **孤立的地点**(孤岛、荒漠、海中的孤舟)
- **夜间的公路**
- **迷宫、走廊**

### 让观众紧张的物品(p.13)
- 动物头骨、法阵、放有内脏的玻璃瓶
- 血迹斑斑的工具
- 阴森的洋娃娃
- 乌鸦或黑鸟

### 让观众紧张的情节环境(p.13)
- **时间限制**:危险越来越近
- **资源不足**
- **被困或困境**
- **模糊不明的威胁**

**使用原则**:紧张场景要配合 [[../genre/thriller-elements]] 的"叠加效应"——不是单独的场景就够,而是场景 + 声音 + 动作 + 道具同时运作。

## 二、危机环境(p.15)
- **沙尘暴**
- **暴风雪**
- **地震**
- **丛林**

**使用要点**:自然灾害型环境对应 [[../genre/dude-with-a-problem]] 的"自然麻烦 / 灾难麻烦"子类。危机环境越压缩(空间小 + 时间短 + 资源少),紧迫感越强。

## 三、仪式感场景(p.12, p.36)

### 经典仪式感(p.12)
- **坐在车顶**
- **跪地行礼**
- **交换戒指**
- **烟火表演**

### 现代仪式感(p.36)
- **车灯与舞台灯的融合**
- **特殊的时间点**(生日、毕业典礼)
- **独特的服装和化妆**(盛装出席、佩戴饰品)

**使用要点**:仪式感场景最适合做 **高潮事件** 或 **结尾回扣**——因为仪式本身就有"终结感"和"升华感"。

## 四、抒情场景(p.32)
- **演讲**
- **告白**
- **告别**
- **葬礼**

**使用要点**:这四类场景本身自带强情绪——但容易写成煽情。需要控制"台词量"——把台词让给动作和道具,抒情场景才不会腻。

## 五、特殊时间点(p.15)
- **生日**
- **周年纪念日**
- **比赛或重大事件**(毕业典礼)
- **最后期限**(Deadline)

**使用要点**:特殊时间点是"**内置紧迫感**"——不需要额外制造时间压力,日期本身就是限制。最后期限(Deadline)是所有紧迫感中最直接的——对应 [[../theory/fifteen-beat-sheet]] 里"坏蛋逼近"和"一无所有"的加速段。

## 六、封闭/孤立空间(综合)
从以上五类提取"封闭与孤立"的共同子集:
| 空间类型 | 情绪色彩 | 适用类型 |
|---|---|---|
| 地下室 | 恐惧、秘密 | 惊悚 |
| 电梯 | 密闭焦虑 | 惊悚、悬疑 |
| 孤岛 | 绝望、原始 | 灾难、生存 |
| 荒漠 | 渺小、孤独 | 变迁仪式、公路片 |
| 海中孤舟 | 漂泊、无助 | 灾难、麻烦家伙 |
| 迷宫 | 困惑、循环 | 心理惊悚 |
| 废弃建筑 | 历史、荒废 | 恐怖、悬疑 |
| 夜间公路 | 未知、逃离 | 公路惊悚 |

## 七、反转情节的场景设置(p.13)
场景本身也可以承载反转:
- **落入敌人的圈套**(安全空间变陷阱)
- **最信任的人是敌人**(家变战场)
- **预言的双重解释**(神圣空间变悲剧)
- **死而复生**(绝望空间变希望)

## Method / How to use

### 场景选择 checklist
- [ ] 这个场景有自带的情绪色彩吗?(紧张/仪式/抒情/危机)
- [ ] 这个场景是封闭的还是开放的?(封闭加紧迫,开放加孤独)
- [ ] 场景有没有时间限制?(最后期限/生日/纪念日)
- [ ] 场景能不能"反转"?(安全变危险 / 废弃变温暖)
- [ ] 场景有没有通过道具实体化?(黑巷里的一盏灯,比"一条巷子"更有力)

### 艺考 2000 字场景经济学
- 开场 400 字:**一个带情绪的单一场景**(不要换场景,在一个场景里完成人物出场+冲突开启)
- 中段 800 字:**两个场景切换**,对比(安全→危险 或 白天→夜晚)
- 高潮+结尾 800 字:**回到开场场景**——但光线/天气/道具发生变化(呼应变迁)

**字数有限,场景数量控制在 2-3 个**。把细节给**一个主场景**,比四处游走更有画面感。

## 天气/光线 × 情绪的快速对照
| 天气/光线 | 情绪 | 使用时机 |
|---|---|---|
| 雨天(p.61) | 洗涤、重生、失落 | 高潮、觉悟 |
| 暴风雪(p.15) | 绝望、隔离 | 危机 |
| 沙尘暴(p.15) | 窒息、末日 | 灾难开场 |
| 逆光(p.26) | 疏远、悬疑、神秘 | 反派出场 |
| 侧光(p.25) | 亦正亦邪、心事重重 | 人物内心戏 |
| 顺光(p.26) | 正面、安全、明亮 | 开场、回忆 |

## Debates & limits
- **原笔记的场景列表偏向"暗色调"**。紧张场景和危机环境占绝大多数,温暖/喜剧场景几乎没有列出。这是因为原笔记侧重惊悚/悬疑训练。
- 【我的理解】艺考编讲故事**不要全程用暗色调**——开场用一个温暖场景(明确"日常"),对比中段的紧张场景(打破日常),效果更好。
- 【你的推断】场景的"历史层"(一个有故事的地方 vs 一个空白的地方)往往被忽视——《闪灵》的旅馆之所以恐怖,不是因为它暗,而是因为它有**历史**。
