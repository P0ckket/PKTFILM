# 家庭冲突素材库

## TL;DR
这是专门写**家庭题材故事**的冲突设计清单。原笔记 p.4 的亲情主题、p.10 的家庭题材冲突设置、p.11 的家庭关系冲突点,加上 p.4 对《饮食男女》的深度拆解——整合成**家庭故事的冲突引擎库**。家庭故事是艺考编讲故事的**最安全选择**——因为每个评委都有家庭,共情门槛最低。

## 一、家庭题材的冲突设置(p.10)
- **厌烦婚姻**
- **外部诱惑**(出轨)
- **家庭责任与个人愿望的冲突**
- **经济压力**

## 二、家庭关系的冲突点(p.11)
- **孩子的抚养权**
- **教育理念的分歧**
- **财产或遗产争执**
- **疾病或成瘾问题**

## 三、亲情主题清单(p.4)

### 父母与子女
- **对父爱的赞美**
- **对母爱的赞美**

### 隔代祖辈
- **对手足之情的赞美**

### 亲戚关系
- **家和万事兴**
- **血浓于水**

### 兄弟姐妹
- **家是幸福的港湾**

## 四、《饮食男女》的家庭拆解(p.4)——家庭故事的标杆

### "家"的四重含义
1. **空间含义**:家是物理空间——各种矛盾冲突发生的"场"
2. **吸纳/糅合/分解矛盾的含义**:家是"烹饪"的地方——烹饪 = 处理矛盾
3. **完整性的挣扎与冲突**:父亲维护家的完整,但家在解体
4. **传统 vs 现代**:体现延续和继承(长辈与晚辈之间的"礼"与"规范") + 颠覆(传统的"家庭"观念、孝悌观与婚姻观)

### 餐桌的含义(p.4)
- **吃饭的地方**
- **将家庭成员聚集的地方**
- **彼此可以沟通交流的地方**
- **矛盾产生、激化、解决的地方**
- **两种态度**:"联络感情重视" vs "完成任务应付"

### 父亲形象(p.4)
**传统的中国式父亲的共性**:
- 严厉、慈爱、沉默寡言
- 有担当、有责任
- 观念保守——想坚守传统可又显得力不从心

### 家的解体(p.4)
- "家"从形式到实质的解体
- 以父亲为核心的传统家庭模式被解构
- 与之相连的崇尚孝道的家庭伦理道德也被颠覆

**【我的理解】**:《饮食男女》的拆解是原笔记质量最高的案例分析——它揭示了"家庭故事"的深层结构:**餐桌 = 微型舞台,烹饪 = 处理矛盾,父亲 = 旧秩序的化身,解体 = 主题**。任何中国家庭故事都可以从这个模板出发。

## 五、非常规家庭结构(p.4)
- **母亲角色缺失的环境**
- 当核心角色缺失时,家庭的动力学完全改变——其他成员必须**补位**,补位的过程就是冲突的来源

## 六、家庭故事可用的冲突六类(交叉 [冲突六大类型](../theory/conflict-six-types.md))
| 冲突类型 | 家庭版表现 |
|---|---|
| 信息错位 | 父母隐瞒病情 / 子女隐瞒恋情 / 家族秘密 |
| 力量失衡 | 家长制 / 长子的"权力" / 经济控制 |
| 资源稀缺 | 遗产争夺 / 一份工作养全家 / 一床一房 |
| 未知的深渊 | 家族历史的黑暗面 / "被遗忘的过去" |
| 现实与理想 | 家庭责任 vs 个人愿望 / 留守与远行 |
| 个体与集体 | 个人自由 vs 家庭期望 / 出走 vs 归顺 |

## Method / How to use

### 家庭故事设计 4 步
1. **定家庭结构**(完整/残缺/重组/隔代)
2. **定一张"餐桌"**(一个固定的场景承载矛盾产生→激化→解决)
3. **定冲突引擎**(从"冲突设置四条"或"冲突点四条"中选)
4. **定父母形象**(传统型/开放型/缺位型)+ **子女身份**(叛逆/顺从/在两者之间挣扎)

### 艺考 2000 字家庭故事套路
- **开场 400 字**:餐桌上/年夜饭/家庭聚会——表面和平,暗流涌动
- **激励 400 字**:一件小事引爆——孩子的一句话/一封信/一个电话
- **转折 400 字**:矛盾升级——老问题和新问题叠加(e.g. 遗产争夺 + 教育分歧)
- **危机 400 字**:家庭"解体"时刻——有人出走/有人崩溃/有人说出真话
- **高潮 400 字**:和解或决裂——但必须有**一个动作**(拥抱/握手/离去)而不是一段台词

### 家庭冲突 checklist
- [ ] 冲突是"家庭内部"的吗?(不是外部敌人强加的)
- [ ] 有没有一个"固定场景"承载矛盾?(餐桌/客厅/医院病房/车里)
- [ ] 父母形象是否有"两面性"?(严厉但慈爱/控制但心软)
- [ ] 子女有没有"两难"?(家庭义务 vs 个人追求)
- [ ] 和解/决裂是通过**动作**还是**台词**完成的?(动作更好)

## Debates & limits
- **"家庭题材"不等于"温情"**。原笔记 p.10 的冲突设置(厌烦婚姻/出轨/经济压力)完全可以写成家庭惊悚或犯罪(如《消失的爱人》《寄生虫》)——关键看"冲突走多远"。
- 【我的理解】中国艺考的"安全区"是**亲情和解型**(统考);**家庭撕裂型/犯罪型**更适合校考——因为校考老师更看重"敢写 + 深度"。
- 【你的推断】原笔记对"非常规家庭结构"只提了一句——但这恰恰是当下影视市场的热点(单亲/离异/重组/同性/跨代抚养)。未来可单独补充。
