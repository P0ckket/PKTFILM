# 穿着与性格表达素材库

## TL;DR
原笔记 p.60 专门讨论了**穿着如何建立人物**——服饰四要素(色彩/质地/样式/效果) + 穿着的四重功能(身份/场合/形象/时代) + 遮挡类服饰 + 性格特质的形成与表现。这是**视觉化建立人物**的操作指南——在编讲故事里,一句"他穿着……"就能替代一段人物描写。

## 一、服饰四要素(p.60)
| 要素 | 功能 |
|---|---|
| **色彩** | 情绪、身份、阵营(参考 [视听语言总览](../concepts/cinematic-language/visual-language-overview.md) 的色彩板块) |
| **质地** | 阶级、生活状态(丝绸 vs 粗布,崭新 vs 破旧) |
| **样式** | 时代、职业、性格(制服 vs 便装,保守 vs 夸张) |
| **效果** | 观众接收到的综合印象(威严/可怜/滑稽/神秘) |

## 二、穿着的四重功能(p.60)
1. **身份 = 服饰**:穿什么就是谁(白大褂=医生,制服=军人)
2. **区别场合 = 社交礼仪**:穿对衣服=人物的社会化程度
3. **塑造人物形象 = 某种特征**:一个标志性穿着 > 一段描述
4. **时代背景**:衣服直接交代"这是什么年代"

## 三、描写穿着的写作技法(p.60)
| 手法 | 效果 |
|---|---|
| **形容词 = 画面感** | "洗旧的蓝棉布外套"——瞬间建立可怜 |
| **动词 + 名词 = 人物动作和穿着结合** | "她卷起袖子"——穿着+动作同时建立性格 |
| **服装的变化 = 人物成长与蜕变** | 开场穿灰色→结尾穿红色——视觉化弧光 |

**使用要点**:服装变化是 [人物弧光机制](../concepts/character/character-arc-mechanism.md) 在视觉层面的**最简表达**——不用任何台词,让主角在高潮时换一件衣服,弧光就完成了。

## 四、遮挡/隔绝类服饰(p.60)
- **手套**——隐藏/保护/掌控/阻碍
- **帽子**——遮挡/身份/权威
- **墨镜**——隐藏情感/酷/距离感
- **口罩**——匿名/恐惧/隐藏
- **围巾**——温暖/束缚/怀旧
- **耳机**——隔绝外界/自我封闭
- **假发**——身份伪装/变身

**使用要点**:遮挡类服饰是**双重符号**——"戴上"的那一刻代表隐藏/保护,"摘下"的那一刻代表暴露/真实。摘下口罩的瞬间、摘下墨镜的瞬间——都是**微型高潮**。

## 五、性格特质(p.60)

### 性格的三个维度
- **偏好**:喜欢什么(口味/颜色/空间)
- **癖好**:不由自主的习惯(强迫症/收集/洁癖)
- **爱好**:主动选择的活动(运动/阅读/音乐)

### 性格特质的形成原因(p.60)
| 路径 | 说明 |
|---|---|
| **传承** | 从家庭/文化中继承的性格 |
| **归因(靠时间次数)** | 重复经历形成的习惯 |
| **归因(靠强烈情绪的重大事件)** | 创伤/顿悟形成的转变 |
| **特殊教育训练** | 军训/体育/艺术训练塑造的性格 |

### 性格特质的三层表现(p.60)
1. **具体表现**:个人实际行为
2. **一旦形成**:会影响与世界交互的方式
3. **造成影响**:行为怎么进一步改变了生活,形成了他的现况

**使用要点**:这三层对应 [人物弧光机制](../concepts/character/character-arc-mechanism.md) 的起点链——**形成原因(过去) → 具体表现(现在) → 造成影响(困境)**。写清这条链,人物就不是"设定",而是"有来历的人"。

## Method / How to use

### 穿着设计 checklist
- [ ] 主角的穿着能不能一句话概括他的身份?
- [ ] 穿着有没有一个"标志性元素"(帽子/手套/旧外套)?
- [ ] 穿着在故事中有没有**变化**?(变化=弧光)
- [ ] 有没有"遮挡类服饰"可以利用?(戴上=隐藏,摘下=暴露)
- [ ] 穿着的色彩是否与光线/场景协同?

### 性格设计快速三问
1. **他的性格是怎么形成的?**(传承/创伤/训练)
2. **这个性格让他做了什么?**(具体行为)
3. **这个行为让他陷入了什么困境?**(造成影响)

→ 回答完三问,一个有来历的人物就立住了。

## Debates & limits
- **原笔记对穿着的讨论偏向"符号化"**——把衣服当做信息载体,而不是审美对象。这是剧作视角而非造型设计视角。
- 【我的理解】艺考编讲故事里,**一个有记忆点的穿着细节 > 三行外貌描写**。"他穿着一双不匹配的鞋子"就够了。
- 【你的推断】遮挡类服饰(手套/墨镜/口罩)可以和 [道具象征通则](../concepts/symbolism/prop-symbolism.md) 的"身份道具"合并讨论——它们的功能高度重叠。
