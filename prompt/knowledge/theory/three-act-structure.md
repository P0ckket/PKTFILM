# 三幕剧结构(Three-Act Structure)

## TL;DR
三幕剧是西方主流剧作法里最基础的"段落比例"规定:故事按 **1:2:1** 的字数/时长比被切成开端(Setup)、对抗(Confrontation)、结局(Resolution)三段,段与段之间由两个**情节点(Plot Point)**承接,其中**第一情节点 = 激励事件**,负责"彻底打破主角生活平衡"并给出不可逆的戏剧目标。在艺考编讲故事场景中,它被进一步压缩成 **300 + 600 + 300 字** 的硬比例模板。

## Definitions

- **三幕剧**:一种叙事段落法。把故事视作"建置→对抗→解决"的三段式结构,每一段都有明确的戏剧任务。
- **第一幕 / 开端 / Setup**:明确主角存在、展露故事背景、设置悬念、预示冲突。
- **第二幕 / 中段 / Confrontation**:对抗段,冲突升级,主角反复试探并失败。
- **第三幕 / 结局 / Resolution**:主角对最终抉择的面对与结局的落定。
- **情节点 1(Plot Point 1)**:第一幕与第二幕之间的接缝,即**激励事件**所在位置。
- **情节点 2(Plot Point 2)**:第二幕与第三幕之间的接缝,主角找到"一线生机"。

## Key terms

| 术语 | 含义 | 来源 |
|---|---|---|
| 建置 = 铺陈 = 引子 | 开端三要素:主要人物 / 戏剧性前提 / 戏剧性情景 | 【原文】p.6 |
| 激励事件(Inciting Incident) | 彻底打破主角生活平衡的首个重大事件,发生在故事前 1/4 | 【原文】p.7 |
| 困境(Predicament) | "难"+"一线生机"——必须难,但必须留缝 | 【原文】p.7 |
| 动机(Motivation) | 分生存/职业、人物关系/情感线、发展/理想三类,理想最难催化 | 【原文】p.7 |

## Context & problem
**这个框架想回答一个问题**:当观众坐下来看一个故事时,他们在"什么时间点"期待"什么事情"发生?三幕剧用一句话给出答案:**前 1/4 打破平衡,中间 1/2 反复对抗,最后 1/4 收束命运**。

在艺考语境下,它还要回答一个更现实的问题:**2.5 小时(150min)考试,2000 字编讲故事,应该怎么分配字数?** 【原文】p.6-7 给出的硬模板是:

- 第一幕 300 字(开端 / 建置)
- 第二幕 600 字(对抗)
- 第三幕 300 字(结局)

> 注意:这 1200 字不是 2000 字全部;p.41 给出的是**五事件**结构(400×5 = 2000 字),即把三幕再细分。详见 [故事核与人物反差](../theory/story-core-and-contrast.md)。

## Claims

1. **开端(第一幕)的任务是三件事**(【原文】p.6)
   - 明确主角的存在:"不一定是一个品行端正的好人,但身上必须有可爱之处、有闪光点、有能让人共情的地方"
   - 展露故事背景:完整呈现世界架构,说明作品调性与类型,**可以特意隐瞒一些信息**
   - 设置悬念、预示冲突(eg. 小镇上发生的怪事、跟随主角的陌生人)

2. **建置(铺陈 / 引子)必须包含三要素**(【原文】p.6)
   - 主要人物
   - 戏剧性前提(dramatic premise)
   - 戏剧性情景(dramatic situation)

3. **情节点 1 = 激励事件**,必须是动态的、有冲突的、能够**锁定**第一幕到第二幕的转换。(【原文】p.6)
   - 【原文】"动态的、有冲突、充分发展了、明确的事件"
   - 【原文】"既要有冲突,还要清晰明确,能够催化主角的动机"
   - 【原文】"往往不能是一个单一事件造成的,所以会有所谓'伏笔'这样的前因设置"
   - **人物困境的三种形态**:突变(一件事突然打破平衡)/ 缓变(埋下的伏笔到节点显现)/ 他人带来(带来困境的人必须与主角三观有巨大差距)

4. **动机分三档,催化难度递增**(【原文】p.6-7)
   - 生存 / 职业:eg 求生故事、警察医生接到任务(最易催化)
   - 人物关系 / 情感线推动:需要多次催化,至少一个伏笔事件
   - 发展 / 理想:最难催化,需要多次催化

5. **困境的硬要求是"难 + 一线生机"**(【原文】p.7)
   - 【原文】"给主人公一线生机"——不能让观众彻底绝望,否则故事不能进

6. **写开端的最好方法是"先知道结尾"**(【原文】p.7)
   - 【原文】"清楚地知道你的结尾,结局意味着什么,解决方法,也就是你的故事怎么解决"
   - 【我的理解】这等同于编剧圈常说的 reverse engineering / backward outlining。

## Method / How to use

### 艺考编讲故事:300-600-300 字分配

| 段 | 字数 | 任务 | 必须出现 |
|---|---|---|---|
| 第一幕(开端) | 300 | 主角出场 + 背景 + 悬念 | 建置三要素 |
| 情节点 1 | 嵌入 | 激励事件 | 打破平衡 + 锁定目标 |
| 第二幕(中段) | 600 | 对抗、反复失败 | 动机受阻 → 升级 |
| 情节点 2 | 嵌入 | 一线生机 | 主角找到解决路径 |
| 第三幕(结局) | 300 | 最终抉择 + 落定 | 与开端呼应 |

### 判断自己的激励事件是否合格(来自 p.6-7 的清单)
- [ ] 是动态的、有冲突、时间明确的事件吗?(不是静态的、模糊的)
- [ ] 能清晰地催化主角的动机吗?
- [ ] 是否铺了"伏笔"(不是空降)?
- [ ] 打破的是**生活平衡**,而不只是"遇到一件小麻烦"?
- [ ] 发生在故事前 1/4 吗?

### 给动机分级
- 求生或职业任务开头 → 易催化,但天花板低
- 人物关系 / 情感 → 需要铺至少一个伏笔事件
- 理想 / 信念 → 需要多次催化,老师最高看

## Debates & limits
- **三幕剧的普遍性有学界争议**。【学界常见解释】Syd Field 的《Screenplay》把它推为"好莱坞主流范式",但 David Bordwell、Kristin Thompson 等人认为古典好莱坞更适合用"四段式(setup–complicating action–development–climax)"描述(Thompson, *Storytelling in the New Hollywood*, 1999)。三幕剧是**工作室与写作教学**的简化用法,不是电影本体论。
- **三幕与东方叙事传统的兼容性问题**。【我的理解】"起承转合"(四段)、中国章回体(多集群)、日本的"序破急"(三段但比例 1:4:1 而非 1:2:1)都不完全吻合 1:2:1。艺考语境下仍推三幕,是因为**考试评分以西式叙事为基准**(见 [东西方故事标准](../comparison/eastern-vs-western-story-standard.md))。
- 【你的推断】因此在"校考面试 + 西方电影评论"场景下用三幕剧最划算;在"分析国产文艺片 + 统考主题写作"场景下,三幕剧只是脚手架,不是天花板。

## Directing takeaways
1. **把激励事件往前放**:前 1/4 必须完成 setup + 激励,不能磨洋工。艺考编讲故事是 300 字前必须有转折。
2. **结尾反推开端**:先决定结尾谁生谁死、目标是否达成,再倒推"开端让观众相信动机的那件事是什么"。
3. **伏笔是激励事件的"前因储备"**:不要让激励事件凭空降临。考官最容易扣分的就是"突然出现"的事件。
4. **留一线生机**:写"困境"时强迫自己问:主角还能做什么?如果回答"什么都做不了"——这个困境太满,戏走不下去。
5. **动机分级决定写作难度**:统考可冲"理想型动机"拉分,但必须多次催化;时间不够时用"职业型动机"保底。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 情节点 = 情节转折 | 情节点是**结构接缝**,不是每个小转折都是情节点。一部长片通常只有 2 个情节点。 |
| 激励事件 = 第一次冲突 | 激励事件是"打破生活平衡"的**那一个**事件;之前的小摩擦是伏笔,不是激励事件本身。 |
| 300-600-300 是字数硬性 | 这是艺考指导的**起步模板**,不是铁律。学界的 1:2:1 是时间/页数比例,落地到字数时会根据考试规定调整。 |
| 三幕剧 = 所有故事的结构 | 三幕剧是"西方工作室主流",不是叙事本体论;东方/实验片有自己的段落法。 |
