# 冲突六大类型(Six Types of Conflict)

## TL;DR
原笔记 p.28 在"类型叙事总结"里塞入了一张**冲突六类型表**——不是按"人与人 / 人与自然 / 人与自己"这种经典三分,而是按**冲突的动力来源**分类:信息错位、力量失衡、资源稀缺、未知的深渊、现实与理想、个体与集体。每一种冲突对应若干类型叙事,可以帮你在"类型叙事总览"之后做第二次聚焦:**同一个类型在不同冲突驱动下,情感走向完全不同**。

## Definitions

- **冲突(Conflict)**:故事中推动主角行动的对立力量来源。
- **冲突类型**:按动力来源划分的冲突底层形态。
- **动力来源**:驱动冲突的核心机制(不是表面对手,而是"为什么对手能与主角对立")。

## Key terms

| 冲突类型 | 动力来源 | 典型载体类型叙事 | 【原文】p.28 |
|---|---|---|---|
| 信息错位 | 主角/观众掌握的信息不对称 | 伙伴之情 / 推理侦探类 | ✓ |
| 力量失衡 | 主角能力远低于对手或事件需求 | 愚者成功 / 麻烦家伙 | ✓ |
| 资源稀缺 | 供求关系失衡导致的竞争 | 竞赛 / 名利机会 | ✓ |
| 未知的深渊 | 恐怖产生于无法看清的事物 | 鬼怪屋 / 恐怖类型 | ✓ |
| 现实与理想 | 主角期望与现实世界的差距 | 变迁仪式 | ✓ |
| 个体与集体 | 个人力量 vs 集体力量的绝对悬殊 | 被制度化 | ✓ |

## Context & problem
**这个分类解决的问题是**:当你在写剧本时,说"这个故事里主角要面对困难",是不够的——因为"困难"有很多种,每一种对应不同的情感走向和观众体验。

传统经典分类(人与人/人与自然/人与自己)的问题是:**一部片通常同时包含三种**,无法区分。比如《海上钢琴师》既有"人与自己"(要不要下船)也有"人与环境"(船上的孤独)。p.28 的六类法更细、更好用,因为它按"**动力**"分,每部片通常只有一个**主动力**。

## Claims

1. **信息错位 = 侦探+伙伴之情的共通底层**(【原文】p.28)
   - 侦探片:观众已知信息不断 vs 新露出水面的事实矛盾
   - 伙伴之情:两人相互不了解对方 → 共同经历揭露

2. **力量失衡 = "普通人陷入大事"的底层**(【原文】p.28)
   - 愚者成功:愚者 → 大事(反差产生戏剧)
   - 麻烦家伙:普通人 → 大事(无辜卷入)
   - 也常见于人物关系设定(强势长辈 vs 弱势晚辈)

3. **资源稀缺 = 名利机会冲突的核心**(【原文】p.28)
   - "资源需求 → 供求关系失衡 → 竞争和竞赛"
   - 包含:名、利、机会三种资源
   - 【我的理解】这解释了为什么"高考题材""求职题材""体育竞赛题材"全部是资源稀缺冲突——它们共享同一个动力。

4. **未知的深渊 = 恐怖类型的核心**(【原文】p.28)
   - "恐怖产生的根源:未知"
   - 【学界常见解释】这与 Lovecraft 的 "cosmic horror"理论相通:**真正的恐怖不是看见怪物,而是知道有怪物但看不清**。

5. **现实与理想 = 变迁仪式的必然冲突**(【原文】p.28)
   - 这是"主角以为生活是某样,但发现是另一样"的冲突——不是外部敌人,是认知的错位。

6. **个体与集体 = 被制度化的本质,冲突本质是"力量悬殊"**(【原文】p.28)
   - 【我的理解】这是 p.28 自己给出的答案:被制度化的核心冲突是力量悬殊,所以它在结构上是"力量失衡"的一个子类,但被单独拉出来是因为它的"集体"是**非人的规则系统**,不是单一对手。

## Method / How to use

### 给故事定冲突类型的 1 分钟问诊

| 问题 | 如果回答"是",则属于 |
|---|---|
| 主角不知道对手的真面目吗? | 信息错位 |
| 主角是否明显弱于对手/事件? | 力量失衡 |
| 主角和其他人在抢同一样东西吗? | 资源稀缺 |
| 故事的恐惧感来自"看不见的东西"吗? | 未知的深渊 |
| 主角的痛苦来自"生活和期望不一样"吗? | 现实与理想 |
| 主角在对抗一个规则系统吗? | 个体与集体 |

### 写作自检
- **冲突类型选定后**,其他冲突应作为副线处理,不能与主冲突同等重要。
- **同一冲突类型可以换载体**。如资源稀缺可以用"高考"也可以用"求婚",但内部逻辑一致。
- **混合冲突是高阶技巧**。eg.《肖申克的救赎》主线"个体与集体"、副线"信息错位"(知更鸟的秘密)。

## Debates & limits
- **"六类冲突"不是学界公认分类**。【学界常见解释】西方叙事学更常用 Gustav Freytag 的"内在/外在冲突"二分,或 Lajos Egri 的"前提 → 冲突 → 解决"三角。p.28 的六类法更接近**创作实务归纳**。
- **六类之间不是互斥**。同一场戏可能同时有资源稀缺 + 力量失衡(eg. 穷人 vs 富人争工作)。使用时应选**主导动力**。
- 【你的推断】这六类其实可以按"客观性"排序:个体与集体、资源稀缺(外部可观察)→ 信息错位、未知(认知层面)→ 力量失衡(主角自身)→ 现实与理想(最内在)。靠近内在的冲突,越难拍、越依赖演员表演。

## Directing takeaways
1. **主导冲突决定镜头语言**。资源稀缺需要"对比镜头"(穷与富);未知的深渊需要"画面外"(框选+声画剥离);个体与集体需要"群像与孤像"的构图对比。
2. **冲突类型选错 = 类型叙事走不动**。eg. 想写变迁仪式却用力量失衡做动力——主角会变成"被打败的受害者",而不是"接受命运的人"。
3. **信息错位最适合短片**——因为信息可以快速揭露,适合考场 2000 字。
4. **未知的深渊最难驾驭**——需要靠视听而不是台词表达,初学者慎用。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 冲突 = 反差 | 冲突是故事动力,反差是表达技巧。详见 [反差 vs 冲突](../comparison/contrast-vs-conflict.md) |
| 冲突类型 = 类型叙事 | 类型叙事是"故事的困境结构",冲突类型是"困境的动力来源",前者是壳,后者是核。 |
| "个体与集体" = "人与社会" | 个体与集体强调"力量悬殊的规则系统";人与社会是更宽的哲学说法,包括温和的舆论压力等。 |
| 现实与理想 = 梦想破灭 | 现实与理想可以是"主角接受了另一种生活",不一定是梦想破灭——《海街日记》就是。 |
