# 十五节拍表(Fifteen-Beat Sheet)

## TL;DR
**十五节拍表**(Beat Sheet)是 Blake Snyder 在《救猫咪》(*Save the Cat*)里提出的剧作拆解工具,把三幕剧内部再切成 15 个具体**节拍(beat)**,每个节拍都有明确的戏剧任务和大致的时间位置。它不是对三幕剧的替代,而是对三幕剧的**细粒度嵌入**:三幕剧回答"故事分几段",十五节拍表回答"每段里具体发生什么"。原笔记 p.29-30 提取了其中最核心的 **5 组 15 个** 节拍要素。

## Definitions

- **节拍(Beat)**:故事中一个具有明确戏剧功能的最小单元,通常对应一场或几场戏。
- **十五节拍表**:Snyder 列出的 15 个标准节拍的顺序排布,是好莱坞商业类型片的常见骨架。
- **中点(Midpoint)**:第二幕中央的"伪胜利 / 伪失败"节点,标志剧情从"上升"转入"下坠"。
- **灵魂黑夜(Dark Night of the Soul)**:主角精神最低谷的节拍,位于"一无所有"之后、第三幕衔接点之前。

## Key terms

| 节拍 | 位置 | 功能 | 【原文】p.29-30 |
|---|---|---|---|
| 开场画面(Opening Image) | 第一幕开头 | 奠定基调/风格,展示主角起点,与终场画面对立 / 呼应 | ✓ |
| 主题呈现(Theme Stated) | 前 5 分钟 | 给主角出难题,主题不明显、不刻意 | ✓ |
| 铺垫(Set-Up) | 第一幕 | 主要人物全部出场,展示主角缺失、动机、将改变的东西 | ✓ |
| 推动 / 催化剂 / 激励事件(Catalyst) | 第一幕尾 | 打破主角生活平衡的"第一次剧烈打击" | ✓ |
| 争执 / 辩论(Debate) | 第一-二幕间 | 主角陷入两难,内心与周围人的争执 | ✓ |
| 第二幕衔接点(Break into Two) | 第一-二幕接缝 | 主角**主动**决定进入新世界(不能被骗入) | ✓ |
| B 故事(B Story) | 第二幕早段 | 通常是爱情故事,性格与环境与 A 故事巨大反差 | ✓ |
| 游戏(Fun and Games) | 第二幕前半 | 上升阶段,卸力,充分展现主角真实性格,"真正敌人尚未出现" | ✓ |
| 中点 / 伪胜利(Midpoint) | 第二幕正中 | 小高潮,似乎胜利但未完全解决;气氛开始紧张 | ✓ |
| 坏蛋逼近(Bad Guys Close In) | 第二幕后半 | 外部力量重整,内部矛盾纷呈,内外夹击 | ✓ |
| 一无所有(All Is Lost) | 第二-三幕间 | 屋漏偏逢连夜雨,与"伪胜利"互为相反 | ✓ |
| 灵魂黑夜(Dark Night of the Soul) | 第二-三幕间 | 观众感受主角绝望、蜕变、重生的精神过程 | ✓ |
| 第三幕衔接点(Break into Three) | 第三幕入口 | 主角找到解决问题的办法——一线希望 | ✓ |
| 结局 / 大高潮(Finale) | 第三幕 | 新世界诞生,取得最终胜利,形势无逆转可能 | ✓ |
| 终场画面(Final Image) | 结尾 | 与开场画面呼应,强调"新世界" | ✓ |

## Context & problem
**这个表格解决的问题是**:三幕剧告诉你"有三段",但写作时你会发现"第二幕 600 字写什么"这个问题悬空——对抗怎么展开?什么时候起伏?什么时候绝望?

Snyder 的答案是:**节拍感**。好莱坞观众在大量商业片的训练下,形成了对情节节奏的固定期待——大约 25 分钟有一个中点、大约 75 分钟主角会彻底崩溃、最后 15 分钟会反转——如果你踩不上这些节拍,观众会感到"故事松了"。十五节拍表把这种"隐性节奏"显性化。

在艺考语境下,它还回答了一个更朴素的问题:**编讲故事时,中段 600 字的剧情要怎么不塌?** 答案是:用"游戏→中点→坏蛋逼近→一无所有"这四个子节拍撑起第二幕。

## Claims

1. **节拍感是"观众期待"的外显化**,不是作者凭空的规定。(【学界常见解释】)

2. **开场画面必须与终场画面呼应**——这是戏剧性变化的外显证据。(【原文】p.29)
   > "开场画面与终场相互对立 / 呼应——戏剧性变化"

3. **主题的提出不能明显、刻意**。【原文】"前期主角不在意,后期会产生深远的影响"——这是好主题与坏主题的分水岭。

4. **激励事件在这里叫"推动 / 催化剂"**,与三幕剧的"情节点 1"是同一个东西的不同叫法(【原文】p.29)。两种命名法源自不同老师:
   - 三幕剧派(Syd Field):叫 Plot Point 1
   - Save the Cat 派(Blake Snyder):叫 Catalyst

5. **"游戏"节拍的本质是"卸力"**。【原文】"剧情轻轻精彩超乎观众预期,卸力,让观众 / 读者情绪放松"。——这是剧本最易被省略、但商业片最不能缺的节拍。

6. **"中点"与"一无所有"互为相反**。(【原文】p.30)
   - 中点 = 伪胜利,表面赢实则问题未解
   - 一无所有 = 屋漏偏逢连夜雨,精神导师/爱人/亲人的离去,旧世界消亡
   - 两者构成第二幕的"倒 U 形"情感弧线。

7. **进入第二幕必须是主角自己的选择**。【原文】"主角不能被诱入、骗入或不知不觉进入第二幕……必须自己作出决定"——这是判断主角是否"活着"的标志。

8. **开场的功能有三种**(【原文】p.29):
   - 表明主题(阿甘正传、战争之王)
   - 交代信息(霸王别姬、教父)
   - 隐藏信息(闪灵、记忆碎片)

## Method / How to use

### 拆片练习
拿到一部陌生的商业片,在看的时候标出 15 个节拍的时间码,对比"预期位置 vs 实际位置"。偏差大的位置通常对应导演的风格特征。

### 写作时用作 checklist
写完初稿后问:
- 开场画面和终场画面呼应了吗?
- 主题是不是藏在某场戏里,而不是台词直说?
- 游戏节拍有没有?(如果只有下坠没有上升,情感曲线会塌)
- 一无所有之后,灵魂黑夜写到位了吗?
- 主角进第二幕是不是自己决定的?

### 在艺考编讲故事中应用
| 五事件(p.41) | 对应 Snyder 节拍 |
|---|---|
| 开场事件 | 开场画面 + 铺垫 + 催化剂 |
| 发展事件 1 | 第二幕衔接点 + 游戏 |
| 发展事件 2 | 中点 + 坏蛋逼近 |
| 危机事件 | 一无所有 + 灵魂黑夜 |
| 结局事件 | 第三幕衔接点 + 结局 + 终场画面 |

## Debates & limits
- **公式化批评**。【学界常见解释】很多独立电影创作者批评《救猫咪》把剧作变成"填空游戏",导致好莱坞商业片同质化。Peter Dochtery 等人在 2013 年左右掀起过"Save the Cat Formulaic Hollywood"的讨论。
- **时间点是建议,不是铁律**。Snyder 原书给出的是"110 页剧本的页码",映射到国产短故事、长剧集、纪录片时不能硬套。
- **东方叙事的错位**。【我的理解】侯孝贤、小津这类导演的节拍显然不走"中点伪胜利 → 一无所有"的商业曲线,而是更接近"散文化"的段落累积。把这些作品硬套节拍表会误判它们"节奏塌了"。
- 【你的推断】十五节拍表最适合的应用域:**目标明确的商业类型片 + 艺考编讲故事**。不适合:作者电影、纪录片、散文电影、实验电影。

## Directing takeaways
1. **节拍表是写作时的体检表,不是大纲**。先写故事,再用它体检。
2. **中点和一无所有是第二幕的两个骨点**——如果缺一个,第二幕一定软。
3. **主角必须主动进入第二幕**。写完后检查:是谁按的"开始"按钮?如果不是主角,改。
4. **游戏节拍最容易被省略**,省略后电影变紧张但缺乏"角色魅力展示"时间——观众记不住主角是谁。
5. **开场与终场的镜像**是最廉价但最有效的结构技巧(eg. 《阿甘正传》羽毛飞起 vs 羽毛落下)。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 十五节拍 = 三幕剧的升级版 | 十五节拍是三幕剧的"细分视图",两者同时存在、互补。 |
| B 故事 = 次要角色线 | B 故事通常是爱情线,其核心功能是**反差 A 故事的气质**,从而让观众看到主角的另一面。 |
| 中点 = 高潮 | 中点是**伪胜利**,是小高潮,绝不是大高潮;大高潮在结局节拍。 |
| 一无所有 = 最低点 | 一无所有是外部状态的最低点,**灵魂黑夜**才是内部状态的最低点——两者连续发生。 |
| 每个节拍必须是一场戏 | 一个节拍可以跨多场戏,甚至一场戏可以包含两个节拍。它是功能单元,不是场次单元。 |

## Open questions
- 【待核实】Snyder 原书的十五节拍是否完全对应 p.29-30 的 15 个项?(原笔记已对 5 组做了分组标记:开场事件/激励事件/转折事件/危机/高潮结尾)
- **B 故事在非爱情类型里是什么?**——在侦探片里可能是"警搭档关系",在灾难片里可能是"家庭关系"。这需要更多案例补充。
- **第四堵墙**(p.29 开头提到)与十五节拍是什么关系?——原笔记把它和节拍表并列,但它不是节拍,更像一种"观众位置"技巧。待拆出独立概念卡。
