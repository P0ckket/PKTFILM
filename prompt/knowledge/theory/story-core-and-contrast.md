# 故事核与人物反差(Story Core and Character Contrast)

## TL;DR
**故事核**是艺考编讲故事的第一工作——把整个故事用**一句话**概括。公式是:**故事核 = 谁 + 做什么 = 人物 + 行动**,且**人物与行动之间必须构成反差**,以产生悬念和张力。原笔记 p.40 还把"反差"拆成**三层张力**:人物张力(过去 vs 当下、自我 vs 非我)、情节张力(事件 vs 人物、命运 vs 抉择)、场面张力(整体 vs 局部、和谐 vs 突兀)。这是**考场上 20 分钟构思的起点**——核没立住,后面五事件白写。

## Definitions

- **故事核(Story Core)**:能用一句话概括整个故事的核心命题,结构为"谁 + 做什么 + (结局)"。
- **反差(Contrast)**:人物与其行动、处境、目标之间的对立或不协调。
- **行动(Action)**:**不是动作(movement),也不是遭遇(encounter)**,而是"主动的、有目标的、克服不同方面阻力的连续动作"。(【原文】p.40)
- **一句话故事**:"做什么 + 结局"的极简公式,整个故事被压缩到一行。
- **离水之鱼(Fish out of Water)**:反差的经典配方——把特定类型人物推进超出他理解范围的世界。

## Context & problem
**这个公式解决的问题是**:艺考编讲故事 2000 字 / 20 分钟构思,**第一件事不是写人物小传,不是列事件,而是先立故事核**。如果故事核不成立(没有反差、没有行动、没有结局),后续所有段落都救不回来。

原笔记 p.40 给出的反例很直接:"如果故事中缺乏主人公、情节线杂乱,则难以用一句话概括"——倒推过来说:**能不能用一句话概括,是判断故事是否成立的底线测试**。

## Claims

1. **故事核的最小公式 = 谁 + 做什么**(【原文】p.40)
   - "谁"必须是具体的**人物**(不是群体,不是抽象)
   - "做什么"必须是**行动**(不是动作、不是遭遇)

2. **行动的三个硬指标**(【原文】p.40)
   - **主动的**(不是被动承受)
   - **有目标的**(不是无目的游荡)
   - **克服不同方面阻力的连续动作**(不是一次性的动作)

3. **人物和行动之间必须构成反差**(【原文】p.40)
   - 【原文】"让人物和行动之间必须构成反差,从而引起悬念,激发阅读兴趣"
   - 反差越大,悬念越强
   - 【举例 p.40】
     - "一个商人爱上了他雇来假扮他女朋友的妓女"——商人 ↔ 妓女的阶级反差
     - "一对新婚夫妇要在圣诞节这天跑遍双方父母的四个再婚家庭"——新婚的温情 ↔ 四家庭的混乱反差
     - "一个新入职的员工去参加公司的周末集体度假,结果发现有个人想杀他"——新人无辜 ↔ 谋杀反差
     - "被监视了三十多年的男人决心逃回现实世界"(《楚门的世界》)——主角所在的假世界 ↔ 真世界反差

4. **离水之鱼 = 反差的通用配方**(【原文】p.40)
   - "逆人物能力去设置目标和环境"
   - "特定类型人物被推进超出他理解范围的世界"
   - 【我的理解】这是一种**让主角错位**的技巧——把一个在 A 环境游刃有余的人放到 B 环境里,反差自然生成。

5. **反差有三个层次**(【原文】p.40)——这是全笔记最锋利的一条理论:

   | 层次 | 对立项 | 含义 |
   |---|---|---|
   | 人物张力 | 过去 vs 当下、自我 vs 非我 | 人物内部的分裂 |
   | 情节张力 | 事件 vs 人物、命运 vs 抉择 | 外部力量与主角的角力 |
   | 场面张力 | 整体 vs 局部、和谐 vs 突兀 | 视听层面的不协调感 |

   - 【你的推断】这三层对应三个操作层面:
     - 人物张力 → 人物设计(人物设置 p.1、人物三位一体)
     - 情节张力 → 叙事结构(三幕剧、十五节拍)
     - 场面张力 → 视听语言(构图、对比蒙太奇)
     - 一部好电影通常**三层张力同时运转**。

6. **故事的记忆点来自:人物个性鲜明 + 情节曲折精彩**(【原文】p.40)

## Method / How to use

### 20 秒故事核测试
拿到一个故事,试着用这样的句式说出来:
> "一个 **[形容词 + 身份]** 的人,决定 **[行动]**,结果 **[结局或悬念]**。"

示例:
- "一个**守规矩**的法官**包庇**了一起杀人案,结果发现凶手是自己的女儿。"(反差:守规矩 ↔ 包庇)
- "一个**胆小的**会计**决心**成为杀手,最后却爱上了目标。"(反差:胆小 ↔ 杀手)

如果你的故事说不出这样一句话——**核没立住**。

### 反差强度检查表
问自己:
- 人物属性(身份/性格/能力)与行动之间,有没有**矛盾**?
- 有没有**形容词一换就让故事变无聊**的关键反差词?(如果没有,反差不够强)
- 能不能升级反差?(从"守规矩 ↔ 违法"升到"法官 ↔ 替女儿顶罪")

### 三层张力的艺考应用
- **人物张力**:在开场事件里展示"过去/当下"或"自我/非我"的对比(eg. 主角提到一段被否定的过往)
- **情节张力**:在危机事件里放大"命运/抉择"(eg. 主角在无法选择的情境下必须选)
- **场面张力**:在高潮事件里用一个**反常的场面**(eg. 葬礼上跳舞、婚礼上痛哭)

### 离水之鱼公式化
1. 选一个"强能力领域"(医生、警察、法官、程序员)
2. 把主角放到"完全不依赖该能力"的环境(农村、原始部落、幼儿园)
3. 故事核完成:"一个 **[领域]** 专家被迫在 **[非对应环境]** 生存"

## Debates & limits
- **"一句话故事"是商业片思维**。【学界常见解释】Hollywood 的 *logline* 正是这个——一部长片必须能用一句话卖给制片人。这种模式不完全适合艺术电影(侯孝贤的《悲情城市》很难一句话概括)。
- **反差不是万能钥匙**。有些杰作靠**积累**而非反差(小津、蔡明亮的重复日常)。艺考评分以商业片为基准,所以"反差"规则在考场上有效;在艺术电影评论场景下应慎用。
- 【我的理解】"反差越大越好"这条规则有上限——**反差过大会变成荒诞**。eg. "小学生当总统"是反差,但缺乏可信度。关键是反差要在可信度内。
- 【你的推断】p.40 把反差拆成三层的做法,其实比"故事核"这个公式更有理论价值——它实际上是在说:**张力是叙事的最小构成单位**,而故事核只是"人物张力 + 情节张力"的一个应用切片。

## Directing takeaways
1. **没核不动手**。考场上先写故事核一行,如果这一行不能让人想继续看,换题。
2. **反差要写进故事核本身**,不能等后面补。"一个法官判案"不是故事核;"一个法官要替女儿顶罪"才是。
3. **"行动 ≠ 动作"是写作时最常踩的坑**。写"主角去超市"——那是动作;写"主角冒雨赶去超市买一盒母亲最后一次能吃的蛋糕"——那是行动。
4. **三层张力同时运转的时候最抓人**。写完初稿后检查:我的故事有人物反差(内)、情节反差(外)、场面反差(视听)吗?只有一层 = 及格;两层 = 良;三层 = 优。
5. **离水之鱼是最保险的反差配方**——不知道怎么设反差时,直接套"专家 + 错位环境"。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 故事核 = 主题 | 故事核是"谁做什么",主题是"想表达什么"。故事核可以没主题(差作品),主题不能没故事核。 |
| 反差 = 冲突 | 反差是**静态的不协调**(形容词层面),冲突是**动态的对立推动**(事件层面)。详见 [[../comparison/contrast-vs-conflict]] |
| 行动 = 动作 | 动作是物理运动(走、跑),行动是带目标的连续动作链。 |
| 人物张力 = 人物内心戏 | 人物张力是可以通过**外部细节**展示的(一张旧照片、一个习惯性动作),不等于独白。 |
| 场面张力 = 视觉冲击 | 场面张力是"整体与局部的不协调",不只是视觉炫技(一个安静的户内镜头里一扇敞开的门就是场面张力)。 |

## Open questions
- 【待核实】"故事核 = 谁 + 做什么 + (结局)"这个公式的出处——是否源自 Robert McKee 的 *Story*(1997)中的 "controlling idea"?
- **反差三层的理论贡献**:这是原笔记老师的原创,还是某本教材的归纳?若是原创,是本库最有价值的理论点之一,应独立建卡。
- **离水之鱼的反例**——是否存在反差反而失效的情况?(eg. 反差太大变成荒诞,观众出戏)
