# 类型叙事总览(Genre Narratives Overview)

## TL;DR
**类型叙事**不是按"动作片/爱情片/恐怖片"这种市场标签来分,而是按"故事的核心困境结构"来分。Blake Snyder 在 *Save the Cat! Goes to the Movies*(2007)里提出 **10 种** 普适性故事类型,每一类对应一组典型的**两难选择(dilemma)**。原笔记 p.20-28 整理了其中 5 类:**伙伴之情(Buddy Love)、麻烦家伙(Dude with a Problem)、变迁仪式(Rites of Passage)、被制度化(Institutionalized)、金羊毛 / 愚者成功**——并给出"**类型叙事 → 两难选择 → 主题**"的推导链条。

## Definitions

- **类型叙事(Genre Narrative)**:按"故事核心困境结构"分类的叙事范式,与市场分类(动作/爱情/恐怖)正交。
- **两难选择(Dilemma / Trade-off)**:主角面对两个互斥选项、任何一个都要付出代价的抉择时刻,是主题浮现的入口。
- **三元素**:Snyder 为每种类型设定的三个必要组件(eg. 伙伴之情 = 不完整的主角 + 伙伴 + 复杂情况)。

## Context & problem
**这个分类法解决的问题是**:为什么看起来完全不同的两部电影(《绿皮书》和《断背山》)在结构上如此相似?因为它们都是**伙伴之情型**——不完整的主角 + 补足他的伙伴 + 一个让两人被迫相处的复杂情况。

传统市场分类(喜剧/爱情/犯罪)没法解释这种结构相似性,因为它们按"表面情绪"分,不按"内部困境"分。Snyder 的贡献是把"困境结构"放到首位。

在艺考语境下,这个分类法还回答一个非常现实的问题:**考场上 20 分钟构思时间,怎么快速找到故事骨架?** 答案是:**先定类型叙事,再套两难选择,再写五事件**。

## Claims

1. **类型叙事 → 两难选择 → 主题,是单向推导链**(【原文】p.28)
   - 选定类型 ⇒ 确定两难 ⇒ 两难的解决方式即主题
   - 如:选"变迁仪式" ⇒ 两难是"接受/逃避" ⇒ 主题就是"**人如何与不可控现实相处**"

2. **每个类型都有"三元素"的最小集合**(【原文】p.20-23)
   - 伙伴之情:不完整主角 + 伙伴 + 复杂情况
   - 麻烦家伙:无辜主人公 + 祸从天降 + 生死考验
   - 变迁仪式:生活的问题 + 错误的方法 + 真正解决问题的方法
   - 被制度化:有规则的集体 + 一种选择 + 一种牺牲

3. **冲突有 6 种底层形态**(【原文】p.28),每种对应若干类型:
   - **信息错位** ─ 伙伴之情 / 推理侦探
   - **力量失衡** ─ 愚者成功 / 麻烦家伙
   - **资源稀缺** ─ 竞争竞赛 / 求名求利
   - **未知的深渊** ─ 鬼怪屋(恐怖产生的根源:未知)
   - **现实与理想** ─ 变迁仪式
   - **个体与集体** ─ 被制度化(冲突本质:力量悬殊)

4. **"类型 = 困境",所以类型是可组合的**(【你的推断】)
   - eg.《了不起的盖茨比》= 愚者成功 + 伙伴之情 + 被制度化的混血
   - eg.《肖申克的救赎》= 被制度化 + 变迁仪式的混血
   - 这种组合不违反 Snyder 的原意,Snyder 原书也承认"每部电影通常只有**一个主类型**,但可以含有副类型元素"。

## Method / How to use

### 给你手上的故事选一个主类型(20 秒决策法)
问 4 个问题:
1. 主角是**不完整的一个人**(需要补)吗? → 伙伴之情
2. 主角是**普通人被迫卷入大事**吗? → 麻烦家伙
3. 主角的问题是**无形的生活难题**(青春期、中年、死亡)吗? → 变迁仪式
4. 主角是在**对抗一个体制/集体**吗? → 被制度化

### 从类型倒推两难
拿到类型后,直接套 p.28 的两难对照表:

| 类型 | 两难 | 主题方向 |
|---|---|---|
| 金羊毛 | 外在目标 vs 内在目标 | 追寻 |
| 愚者成功 | 躺平 vs 破釜沉舟 | 信念 |
| 麻烦家伙 | 逃避 vs 面对 | 勇气 |
| 被制度化 | 两种思想体系 | 自由/归属 |
| 变迁仪式 | 接受 vs 逃避 | 和解 |
| 伙伴之情 | 社交 vs 自我 | 关系 |

### 艺考套路
1. 先从"**我最熟的生活困境**"选两难(eg. 家庭重男轻女 → 变迁仪式 或 被制度化)
2. 选类型
3. 按类型三元素填主角/冲突/解决方式
4. 再走五事件大纲

## Debates & limits
- **10 类不是铁律**。【学界常见解释】Snyder 的 10 分法并不是类型学界公认的分类法;Rick Altman(《Film/Genre》,1999)基于语义/句法两层的分类法更常被学术引用。Snyder 的 10 类更多是**编剧实务工具**,不是类型理论。
- **"类型是困境"的假设过强**。有学者认为类型还涉及**视觉惯例、音乐惯例、历史观众期待**——不是单靠困境结构就能定义的。
- 【我的理解】艺考不走学术路线,所以 Snyder 的 10 分法已经够用。但若是校考面试被问到"你理解的类型是什么",应当提及 Altman 的"语义 / 句法 / 语用"三层,以显示超出教材的眼界。
- 【你的推断】Snyder 的 10 类在"作者电影"(侯孝贤、小津、阿彼察邦)面前会崩溃——这些片的叙事不以"困境解决"为中心,而以"时间/情境的积累"为中心。

## Directing takeaways
1. **写剧本前先定类型**。类型一换,全剧的节拍都要重排。
2. **类型之间可混合但不能混乱**。混合要有一个主类型,其他做点缀,否则观众会失焦。
3. **两难必须能用一句话说清**。如果你说不清你故事的两难,类型没选对。
4. **主题不是类型的装饰品**,主题是从类型的两难里"浮上来"的东西。从类型出发做主题,比从主题出发选类型更容易写得结实。

## Common confusions

| 别混淆 | 正解 |
|---|---|
| 类型叙事 = 类型片 | 类型叙事是"故事困境结构",类型片是"市场标签"。一部"爱情类型片"可以属于"伙伴之情"或"变迁仪式"或"麻烦家伙"类型叙事。 |
| 伙伴之情 = 爱情片 | 爱情故事只是伙伴之情的一个**子类**(p.20 明确列出)。伙伴之情还包括宠物之爱、同事之情、灾难之情等。 |
| 麻烦家伙 = 动作片 | 麻烦家伙强调"无辜主角被迫卷入",可以是动作片,也可以是家庭恐怖片《房间》那种室内片。 |
| 被制度化 = 反乌托邦 | 被制度化的"集体"可以是监狱、家庭、公司、军队、师徒——不必然是反乌托邦。 |
| 变迁仪式 = 成长片 | 变迁仪式包含青春期,也包含中年危机、老年、死亡——它的核心是"面对无法控制的生活力量",不是"年轻人长大"。 |
