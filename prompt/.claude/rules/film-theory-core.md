# Film Theory Core Rules（长期规则）

这些规则用于约束“电影理论知识库”的质量与一致性。
若与 CLAUDE.md 冲突，以 CLAUDE.md 为准；若与具体 skill 冲突，以 skill 为准。

## 概念抽取的最低标准
每次写入概念 / 理论笔记，至少回答：
- 这是谁在什么语境下回答什么问题？
- 核心命题是什么？
- 关键术语如何定义？
- 通过什么证据说服你？
- 争议点在哪里？
- 对导演实践有什么可执行启发？

## 必须避免的常见错误
- 把译名差异当概念差异
- 把不同学派的问题域混在一起
- 用“风格标签”替代理论解释
- 只写剧情 / 梗概却没有形式分析入口

## 建议的理论组织方式
- 以“问题（question）”而不是“人物（person）”作为索引入口
- 以“对立 / 张力（tension）”组织比较笔记
- 把理论与可观察的电影语言层面连接：镜头 / 剪辑 / 声音 / 叙事 / 观众位置

## 冲突处理的默认策略
- 并列记录争议 + 标注语境差异 + 提出待核实问题
- 如必须做选择：说明选择依据与代价（trade-off）
